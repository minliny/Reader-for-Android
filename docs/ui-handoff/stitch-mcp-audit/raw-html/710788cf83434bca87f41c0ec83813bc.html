第一章：阿长与《山海经》

长妈妈，已经说过，是一个女工，没有正式的姓名。在我的家乡，这种女工统称为“长妈妈”，但我所认识的长妈妈，确乎有些不同。

她的姓名，已经记不真切了。但在我的记忆中，她是一个极其高大，又极其和蔼的人。她常常带我去逛那条充满了神秘感的小巷，给我讲那些听也听不完的故事。在那时，我还是一个不识字的孩子，对于她所描述的《山海经》，充满了无限的遐想。

“长妈妈，你见过那些人面马身的怪物吗？”我仰着头问她。

她笑了，眼角的皱纹里藏着智慧：“孩子，那些都在书里呢。等你识了字，我就送你一套，你自己去看个真切。”

后来，我的生命中确实出现了一套《山海经》，那是我一生中最初的，也是最珍贵的礼物。那些奇怪的插图，虽然简陋，却为我打开了一个广袤而奇幻的世界。而长妈妈的身影，也永远地和那些奇禽异兽叠印在了一起。
