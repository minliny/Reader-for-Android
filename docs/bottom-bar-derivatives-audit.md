## 10. P0 阻塞问题

1. 阅读页 - 目录底栏功能展开：快捷方式区第 4 项显示「夜间」，不是规范文案「夜间模式」。
2. 阅读页 - 朗读底栏功能展开：快捷方式区文案显示「自动 / 替换 / 夜间」，不是「自动翻页 / 内容替换 / 夜间模式」。
3. 阅读页 - 朗读底栏功能展开：底部三层控制面板缺少规范章节快速跳转区。
4. 阅读页 - 界面底栏功能展开：快捷方式区只有图标，缺少「搜索 / 自动翻页 / 内容替换 / 夜间模式」文字。
5. 阅读页 - 界面底栏功能展开：章节快速跳转区缺少拖拽圆点。
6. 阅读页 - 设置底栏功能展开：顶部第二行章节为「第九章：幽谷探秘」，不是「第一章：阿长与《山海经》」。
7. 阅读页 - 设置底栏功能展开：正文内容也回退为第九章/幽谷内容，不是母版阅读内容。
8. 阅读页 - 设置底栏功能展开：快捷方式区被拆成独立悬浮图标层，且无完整文案。
9. 阅读页 - 设置底栏功能展开：章节快速跳转区使用文字「上一章 / 下一章」，不是 skip_previous / 进度条 / skip_next。
10. 阅读页 - 设置底栏功能展开：「设置」底栏项出现 primary-container 胶囊/背景选中态。
11. 阅读页 - 设置底栏功能展开：UI 排列错层，快捷区悬浮在设置面板上方，底部三层结构被破坏。

## 11. P1 可后续修复问题

1. 阅读页 - 界面底栏功能展开：段落间距没有独立设置项。
2. 阅读页 - 界面底栏功能展开：未明确显示「跟随系统亮度」开关或文字。
3. 阅读页 - 设置底栏功能展开：阅读页手势设置不完整，仅覆盖点击屏幕两侧翻页。
4. 阅读页 - 朗读底栏功能展开：定时关闭按钮内部 chevron 嵌套略乱，视觉可再整理。

## 12. 是否允许进入下一阶段阅读页整体 handoff 审计

不允许。

原因：4 个底栏功能派生 screen 均存在 P0 或结构阻塞问题，尤其是设置 screen 出现章节回退、正文内容回退、章节跳转文字化、底部三层结构错层和底栏厚重选中态。必须先修复底栏功能派生 screen，再进入下一阶段阅读页整体 handoff 审计。

下一阶段阅读页整体 handoff 审计包括：

- 控制层母版
- 4 个快捷功能展开态
- 4 个底栏功能展开态
- 顶部结构
- 底部三层结构
- 边界标注
- UI 排列合理性
- Android handoff 可实现性
