## 2. 规则纠偏

- 母版显示亮度条：控制层参照模板「阅读页控制层参照模板 - 基于修正版含亮度条 (P1 Fixed) - 箭头修复版」继续保留左侧悬浮亮度条。
- 快捷功能展开态隐藏亮度条：搜索、自动翻页、内容替换、夜间模式 4 个快捷功能展开 screen 中，左侧悬浮亮度条不属于展开态控制层，必须隐藏。
- 旧审计标准已废弃：此前“快捷功能派生 screen 必须保留左侧亮度条，并检查 brightness_auto / slider / SVG 右向箭头”的要求不再适用。
- 新 P0 标准：4 个快捷功能展开 screen 中任意一个仍显示左侧悬浮亮度条，即判定为 P0。
