# 次级页面规划收敛表

审计日期：2026-06-13

## 1. 收敛口径

本文件只收敛次级页面，不重新定义主 TAB。

当前主 TAB 处理如下：

```text
书架：P0
设置：P1
发现：暂缓
RSS：暂缓
我的：废弃
书源独立主 TAB：废弃
```

次级页面判断标准：

1. 必须有明确上级入口。
2. 必须有返回目标。
3. 必须有中文页面文字稿。
4. 必须有唯一当前 source 图片；无图、旧图、参考图、FAIL 图均不能算完成。
5. 必须明确是否进入当前 P0、P1、P2、暂缓或废弃。
