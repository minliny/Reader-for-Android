## 5. 图片迁移

当前 source 图片必须从旧目录复制到页面包内：

```text
页面包/
  图片/
    01-默认态.png
    02-状态或模式.png
```

旧 `drafts/` 的处理：

- 当前 source：复制到页面包图片目录。
- 历史参考：迁移到 `91-历史归档/drafts/`。
- 废弃/失败：迁移到 `91-历史归档/rejected/` 并登记原因。
- 拼图/contact sheet：迁移到 `91-历史归档/contact-sheets/`。
