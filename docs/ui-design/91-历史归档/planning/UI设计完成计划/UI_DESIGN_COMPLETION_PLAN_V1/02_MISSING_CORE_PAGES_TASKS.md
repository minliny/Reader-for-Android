# 缺失核心页面任务

主页面定义：仅指底部主导航一级入口对应页面。当前底栏固定为 `书架 / 发现 / RSS / 设置`。

缺失主页面：无。书架、发现、RSS、设置均已有暂定主页面；本项目不规划账户功能，不保留 `我的` 底栏入口。书源管理首页、阅读内完整设置首页属于后续功能页，不计入主页面。

每个页面至少需要：

- 默认态
- 空状态
- 加载态
- 错误态
- 入口关系
- 底栏状态
- 标题栏规则
