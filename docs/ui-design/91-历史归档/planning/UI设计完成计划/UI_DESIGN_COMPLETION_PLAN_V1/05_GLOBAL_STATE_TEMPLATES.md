# 全局状态模板

| 状态模板 ID | 使用场景 | 必要组件 | 可复用模块 | 是否需要单独设计图 |
| ------- | ---- | ---- | ----- | --------- |
| `loading` | 页面首次加载、列表加载、详情加载 | 骨架屏或加载指示、标题栏保留 | 全页面、列表、卡片 | 是 |
| `empty` | 无书籍、无书源、无搜索结果、无书签 | 空状态图标、标题、说明、主操作 | 全页面、列表 | 是 |
| `error` | 普通加载失败、解析失败 | 错误图标、错误说明、重试按钮 | 全页面、底表 | 是 |
| `network_error` | 网络不可用、请求超时 | 网络图标、说明、重试、检查网络 | 全页面、弹层 | 是 |
| `permission_required` | 存储、通知、TTS 或系统权限缺失 | 权限说明、授权按钮、跳过/稍后 | 弹窗、页面提示 | 是 |
| `delete_confirm` | 删除书源、删除规则、移除书籍 | 标题、风险说明、取消、确认 | 对话框/底表 | 是 |
| `operation_success` | 保存成功、复制成功、刷新成功 | toast/snackbar、轻量反馈 | 全局反馈 | 否，可定义组件 |
| `operation_failed` | 保存失败、复制失败、刷新失败 | toast/snackbar、失败原因、重试入口 | 全局反馈 | 否，可定义组件 |

## 模板原则

- 中文文案短句化。
- 不改变主底栏结构。
- 页面级状态保留标题栏和必要返回。
- 列表级状态不破坏页面结构。
