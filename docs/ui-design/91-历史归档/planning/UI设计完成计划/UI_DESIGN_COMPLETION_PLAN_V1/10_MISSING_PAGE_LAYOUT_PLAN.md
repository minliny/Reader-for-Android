# 欠缺页面内容与布局规划

> 状态更新（2026-06-11）：第 1 节已有 `44` 与 `47` 系列，第 2 节已有 `46` 系列。
> 这些章节现作为布局依据和验收清单，不再表示图片尚未生成。

本文只规划尚未生成或尚未展开的页面/弹层内容与布局，不生成 UI 图。

## 优先级说明

- `P0`：下一批应优先补图，影响主要功能链路。
- `P1`：功能页或管理页，建议在 P0 后补。
- `P2`：低频菜单、状态或确认页，可后续补。

## 1. 书源管理首页

优先级：`P0`

入口：

- `设置首页 > 书源管理`
- `设置首页 > 书源与订阅`

页面类型：

- 完整管理页。
- 不属于底栏主页面。
- 不保留底部主导航也可以；如后续 App Shell 统一保留，则需单独确认。

主要内容：

- 顶部标题：`书源管理`
- 顶部按钮：返回、搜索、更多。
- 状态概览：`12 个书源`、`8 个启用`、`4 个异常`、`刚刚检测`。
- 筛选标签：`全部 / 已启用 / 异常 / 未检测 / 自定义`。
- 分组筛选：`全部分组 / 玄幻书源 / 起点导入 / 测试书源`。
- 书源列表：书源名称、域名或来源说明、分组、启用状态、最近检测结果、行内检测入口。
- 底部或右下浮动主操作：`新增书源`。

布局规划：

- 顶部使用普通 AppBar，不使用大 hero。
- 概览卡片放在标题下方，使用 2 x 2 小指标。
- 搜索可作为顶部图标进入，也可在列表上方提供紧凑搜索框。
- 筛选标签横向滚动，紧贴列表上方。
- 列表使用软卡片或分组列表，行高保持可扫读，不做大卡片。
- 每行右侧固定放启用开关和更多/检测入口，避免按钮漂移。

当前图状态：

- 已生成 `docs/ui-design/05-设置链路/书源管理/图片/01-书源管理首页.png` 作为默认态暂定版。
- loading / empty / error / 检测结果后续再补。

## 2. 阅读内完整设置首页

优先级：`P0`

入口：

- `阅读控制层 > 设置快捷模块 > 完整设置 >`

页面类型：

- 阅读内完整管理页。
- 不属于底栏主页面。
- 不保留阅读控制层底部四按钮。
- 不保留右侧亮度条。
- 不展示字体、主题、版式、翻页动画；这些属于 `界面`。

主要内容：

- 顶部标题：`阅读设置`
- 顶部按钮：返回、更多。
- 说明/状态：当前作用范围，例如 `仅当前书籍` 或 `默认阅读行为`。
- 分组入口：
  - `屏幕与显示行为`
  - `翻页与手势`
  - `进度与信息显示`
  - `阅读辅助`
  - `缓存与章节加载`
  - `预设与恢复默认`
- 底部操作：可选 `应用到当前书籍`、`恢复默认`，如空间不足放入右上菜单。

布局规划：

- 顶部使用完整页 AppBar。
- 标题下方放一个紧凑说明卡片，说明这是阅读行为设置，不是外观设置。
- 主体使用分组列表，每行左图标、标题、副说明、右箭头。
- 高风险操作如恢复默认放到列表末尾或右上菜单，不放在首屏大按钮。

当前图需求：

- 已生成 `46-reader-settings-home.png` 及 `46-1`～`46-5` 子页。
- 加载、错误、确认和反馈状态复用 `51` 系列，待 spec 整合。

## 3. 书架右上更多菜单

优先级：`P1`

入口：

- `书架首页 > 右上三点`

页面类型：

- 右上浮层菜单。
- 保留书架背景，可轻微暗化。

菜单内容：

- `刷新书架`
- `分组管理`
- `导入本地书`
- `通过网址添加`
- `书架导入导出`
- `查看日志`

布局规划：

- 从右上三点下方展开。
- 菜单宽度约占屏幕 45%-55%。
- 每行图标 + 文案，低频操作保持紧凑。
- 不遮挡底栏。

当前图需求：

- 可作为 `01` 或 `02` 的派生图生成一张。

## 4. 书架排序底表

优先级：`P1`

入口：

- `书架首页 > 排序图标`

页面类型：

- 底部弹层。

内容：

- 标题：`排序`
- 排序方式：`最近阅读`、`最近更新`、`书名`、`作者`、`阅读进度`、`加入时间`
- 顺序切换：`升序 / 降序`
- 操作：`取消`、`应用`

布局规划：

- 底表高度约 40%-50% 屏幕。
- 排序项使用单选列表。
- 当前排序项显示选中标记。

当前图需求：

- 后续生成一张底表图即可。

## 5. 书架布局设置底表

优先级：`P1`

入口：

- `书架首页 > 齿轮图标`

页面类型：

- 底部弹层或完整设置子页。
- 当前建议先用底部弹层。

内容：

- 标题：`书架显示`
- 展示模式：`封面 / 列表`
- 卡片密度：`舒适 / 紧凑`
- 可见字段：`作者`、`章节`、`进度`、`更新状态`
- 封面比例：`标准 / 高封面`
- 操作：`恢复默认`

布局规划：

- 使用分组控件，不做长表单。
- 二元项用开关，多选项用复选行。

当前图需求：

- 后续生成一张底表图即可。

## 6. 书籍操作底表

优先级：`P1`

入口：

- `书架首页 > 书籍条目更多`
- `长按书籍条目`

页面类型：

- 底部弹层。

内容：

- 顶部书籍摘要：封面、书名、作者、当前章节。
- 操作：
  - `查看信息`
  - `继续阅读`
  - `刷新目录`
  - `缓存书籍`
  - `移动分组`
  - `导出`
  - `移出书架`

布局规划：

- 顶部摘要固定。
- 常用操作使用两列小按钮或列表。
- 危险操作 `移出书架` 放底部，使用警示色但不夸张。

当前图需求：

- 后续生成一张底表图。
