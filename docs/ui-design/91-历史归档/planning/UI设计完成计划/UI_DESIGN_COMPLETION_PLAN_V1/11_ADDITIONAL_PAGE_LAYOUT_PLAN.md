# 后续页面内容与 UI 结构规划

> 状态更新（2026-06-11）：A 组书源管理、B 组阅读设置、App 设置和全局状态中的 4 个
> 推荐批次已由 46/47/48/51 系列覆盖。本文保留为结构验收依据；49/50 系列仍是待生成项。

本文继续规划当前尚未生成 UI 图、但已经能明确结构的后续页面。当前不生成图片。

## 规划边界

- 底栏主页面已齐：`书架 / 发现 / RSS / 设置`。
- 本文规划的是功能页、弹层、底表和状态模板，不再称为主页面。
- 本项目不规划账户功能，不出现 `我的`、头像、登录入口。
- 所有页面延续暖色纸张背景、柔和卡片、紧凑管理型布局。

## A. 书源管理链路

### A1. 书源详情页

入口：

- `书源管理首页 > 书源列表行`

页面类型：

- 完整功能页。
- 不属于底栏主页面。

内容：

- 顶部栏：返回、标题 `书源详情`、更多。
- 书源摘要卡：名称、域名、分组、启用状态、最近检测结果。
- 状态区：`可用 / 异常 / 未检测`，显示最后检测时间。
- 规则摘要：搜索规则、目录规则、正文规则、发现规则、订阅支持。
- 操作入口：`检测书源`、`编辑`、`查看日志`。
- 危险操作：`删除书源`，放页面底部。

布局：

- 摘要卡在首屏顶部。
- 规则摘要用分组列表，不展示大段规则源码。
- 操作按钮使用 2 列或列表行，危险操作单独分区。

### A2. 新增书源页

入口：

- `书源管理首页 > 新增书源`

页面类型：

- 完整表单页。

内容：

- 顶部栏：返回、标题 `新增书源`、保存。
- 导入方式切换：`手动填写 / 粘贴规则 / 从文件导入`。
- 表单字段：
  - `书源名称`
  - `书源地址`
  - `分组`
  - `启用`
  - `规则内容`
- 操作：`检测`、`保存`。
- 校验提示：字段为空、地址格式错误、规则解析失败。

布局：

- 首屏先展示名称、地址、分组和启用开关。
- 规则内容使用折叠编辑区或多行输入，不让首屏被长文本压满。
- 底部固定保存按钮可选；如果固定按钮影响手机安全区，则用顶部保存。

### A3. 编辑书源页

入口：

- `书源详情页 > 编辑`

页面类型：

- 完整表单页。

内容：

- 与新增书源页共用表单结构。
- 增加 `重置修改`。
- 显示当前检测状态和上次保存时间。
- 操作：`检测当前规则`、`保存修改`。

布局：

- 与新增页一致，减少学习成本。
- 修改后字段可显示轻微变更标记。

### A4. 书源检测页

入口：

- `书源管理首页 > 行内检测`
- `书源详情页 > 检测书源`
- `新增/编辑页 > 检测`

页面类型：

- 检测进度页或底部弹层。
- 全局管理建议用完整页；阅读内换源检测才用底表。

内容：

- 顶部栏：返回、标题 `书源检测`。
- 检测对象：书源名称、地址、分组。
- 步骤列表：
  - `连接书源`
  - `读取分类`
  - `搜索测试`
  - `目录解析`
  - `正文读取`
  - `发现页支持`
- 实时日志摘要。
- 操作：`取消检测`。

布局：

- 步骤列表用状态图标：等待、进行中、成功、失败。
- 日志摘要折叠在底部，避免压过主要进度。

### A5. 检测结果页

入口：

- `书源检测页 > 检测完成`

内容：

- 结果摘要：通过、部分失败、失败。
- 指标：耗时、成功项、失败项、警告项。
- 失败列表：失败步骤、原因、建议。
- 操作：`重新检测`、`查看日志`、`返回详情`、`启用/禁用书源`。

布局：

- 首屏给出结论和关键指标。
- 失败项用可展开列表。

### A6. 错误日志页

入口：

- `书源管理首页 > 更多 > 错误日志`
- `检测结果页 > 查看日志`

内容：

- 顶部栏：返回、标题 `错误日志`、清空。
- 筛选：`全部 / 网络 / 解析 / 规则 / 系统`。
- 日志列表：时间、来源、类型、摘要。
- 日志详情入口。

布局：

- 列表紧凑，支持快速定位。
- 清空操作需要二次确认。

## B. 阅读内完整设置链路

### B1. 阅读内完整设置首页

入口：

- `阅读控制层 > 设置快捷模块 > 完整设置 >`

内容：

- 顶部栏：返回、标题 `阅读设置`、更多。
- 说明卡：`管理阅读行为，不含字体、主题、版式。`
- 分组入口：
  - `屏幕与显示行为`
  - `翻页与手势`
  - `进度与信息显示`
  - `阅读辅助`
  - `缓存与章节加载`
  - `预设与恢复默认`

布局：

- 完整页，不保留阅读控制层。
- 分组列表为主，危险操作放右上菜单。

### B2. 屏幕与显示行为

内容：

- 屏幕方向：跟随系统、竖屏、横屏。
- 屏幕超时：跟随系统、5 分钟、常亮。
- 状态栏显示。
- 导航栏显示。
- 阅读页底部信息显示。

布局：

- 分段控件用于互斥选项。
- 开关用于显示/隐藏类选项。

### B3. 翻页与手势

内容：

- 音量键翻页。
- 点击区域设置。
- 左右滑动翻页。
- 单手翻页。
- 长按动作。
- 误触保护。

布局：

- 开关 + 当前值列表。
- 点击区域设置可以后续单独做交互图。

### B4. 进度与信息显示

内容：

- 章节进度显示。
- 书籍进度显示。
- 当前章标题显示。
- 剩余页数/剩余时间。
- 顶部弱信息显示。

布局：

- 以复选/开关行展示。
- 底部可放小预览条，但不使用阅读正文大预览。

### B5. 阅读辅助

内容：

- 章节加载失败重试。
- 缓存优先读取。
- 断点恢复。
- 阅读统计记录。
- 异常章节提示。

布局：

- 开关列表。
- 涉及权限的项显示状态标签。

### B6. 预设与恢复默认

内容：

- 当前预设。
- 保存当前设置为预设。
- 预设列表。
- 应用预设。
- 删除预设。
- 恢复默认。

布局：

- 预设列表使用卡片。
- `恢复默认` 放危险区，必须二次确认。

## C. 设置次级功能页

### C1. 通用设置

入口：

- `设置首页 > 通用`

内容：

- 语言。
- 启动页。
- 主题跟随系统。
- 时间格式。
- 更新检查。

布局：

- 完整页 AppBar + 分组列表。
- 启动页使用单选底表或内联单选。

### C2. 书架与搜索设置

入口：

- `设置首页 > 书架与搜索`

内容：

- 默认书架视图：封面、列表。
- 书架排序默认值。
- 搜索范围默认值。
- 搜索历史保存。
- 搜索结果合并同名书。

布局：

- 分组：书架、搜索、历史。
- 开关和选择器混排。

### C3. 书源与订阅设置

入口：

- `设置首页 > 书源与订阅`

内容：

- 书源管理入口。
- RSS/订阅管理入口。
- 来源检测入口。
- 自动检测频率。
- 隐藏不可用来源。

布局：

- 顶部提供两个主入口卡：书源管理、订阅管理。
- 其余为设置列表。

### C4. 同步与备份

入口：

- `设置首页 > 同步与备份`

内容：

- 最近备份时间。
- 本地备份。
- 从备份恢复。
- WebDAV 配置。
- 导入导出配置。

布局：

- 顶部状态卡显示最近备份。
- 操作列表按备份、恢复、远程同步分组。

### C5. 隐私与权限

入口：

- `设置首页 > 隐私与权限`

内容：

- 存储权限。
- 通知权限。
- TTS 权限或系统能力。
- 清理搜索历史。
- 清理阅读记录。
- 清理缓存。

布局：

- 权限状态用标签：已授权、未授权。
- 清理类操作放底部危险区。

### C6. 关于与反馈

入口：

- `设置首页 > 关于与反馈`

内容：

- App 名称与版本。
- 更新日志。
- 检查更新。
- 反馈入口。
- 开源协议。
- 日志导出。

布局：

- 顶部小信息卡。
- 列表入口为主，不做个人中心样式。

## D. RSS 后续页面

### D1. RSS 搜索页

入口：

- `RSS 首页 > 搜索`

内容：

- 搜索输入。
- 范围：全部、未读、收藏、书单。
- 结果列表：标题、来源、摘要、时间。

布局：

- 参考书籍搜索，但搜索对象是订阅条目。

### D2. 订阅源管理页

入口：

- `设置首页 > RSS/订阅管理`

内容：

- 概览：订阅源数量、异常数量。
- 搜索订阅源。
- 分组筛选。
- 订阅源列表：名称、地址、分组、未读数、启用状态。
- 新增订阅源。

布局：

- 与书源管理保持同类管理风格。
- 列表行右侧放启用开关和更多。

### D3. 添加订阅源页

入口：

- `订阅源管理页 > 新增`

内容：

- 订阅名称。
- 订阅地址。
- 分组。
- 更新频率。
- 测试连接。
- 保存。

布局：

- 表单页，顶部保存。
- 测试连接结果在表单内展示。

### D4. 订阅条目详情页

入口：

- `RSS 首页 > 条目行`

内容：

- 标题。
- 来源。
- 发布时间。
- 正文摘要或正文。
- 关联书籍/链接。
- 操作：收藏、稍后读、标记已读。

布局：

- 阅读类详情页，但不使用阅读控制层。
- 文本阅读密度可以比阅读页更紧凑。

## E. 发现后续页面与弹层

### E1. 信息源选择底部弹层

入口：

- `发现页 > 当前来源`
- `发现页 > 切换来源`

内容：

- 标题：`选择信息源`
- 分段：全部、书源组、单个书源、订阅组、单个订阅。
- 搜索来源。
- 来源列表：名称、类型、可用状态、更新时间。
- 操作：取消、应用。

布局：

- 底表高度 70%-80%。
- 列表支持滚动。
- 当前选中项显示勾选。

### E2. 更多分类底部弹层

入口：

- `发现页 > 更多分类`

内容：

- 当前来源名称。
- 分类搜索。
- 分类网格或列表。
- 支持二级分类展开。

布局：

- 分类数量多时用两列或三列紧凑 chip。

### E3. 来源检测底部弹层

入口：

- `发现页 > 检测`

内容：

- 当前来源组。
- 可用数量。
- 异常来源列表。
- 操作：重新检测、查看详情、关闭。

布局：

- 底表高度 50%-70%。
- 只展示摘要，不做完整日志。

### E4. 发现顶部更多菜单

入口：

- `发现页 > 顶部更多`

内容：

- 刷新当前来源。
- 管理信息源。
- 隐藏不可用来源。
- 清除发现缓存。

布局：

- 右上浮层菜单。
- 管理信息源跳转到设置/来源管理。

## F. 全局状态模板细化

### F1. 页面 loading

适用：

- 书源管理首页。
- RSS 管理页。
- 设置子页。
- 搜索结果。

布局：

- 保留顶部栏。
- 主体使用骨架行，不使用全屏大 spinner。

### F2. 列表 empty

适用：

- 无书源。
- 无订阅源。
- 无搜索结果。
- 无日志。

布局：

- 居中偏上空状态。
- 标题、说明、主操作按钮。
- 保留页面原有筛选区时，空状态放在列表区域。

### F3. 页面 error / network_error

适用：

- 来源加载失败。
- 网络不可用。
- 解析失败。

布局：

- 错误图标。
- 短标题。
- 一行原因。
- 主按钮 `重试`。
- 次入口 `查看日志` 可选。

### F4. permission_required

适用：

- 存储权限。
- 通知权限。
- 系统 TTS。

布局：

- 权限说明卡。
- 主按钮 `去授权`。
- 次按钮 `稍后`。

### F5. delete_confirm

适用：

- 删除书源。
- 删除订阅源。
- 移出书架。
- 删除预设。

布局：

- 对话框或底表。
- 明确对象名称。
- 风险说明。
- `取消`、`确认删除`。
