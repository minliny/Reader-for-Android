# 04-2 阅读控制层完整相对参数规范

> 本文件不再作为当前阅读控制层参数定义源。最新几何与视觉参数以
> `../04-阅读链路/阅读控制层/03-几何规格.md` 为准。

本文保留 `04-2-reader-controls-opaque-icon-aligned.png` 的历史测量过程和手机默认态
比例记录，用于追溯，不得单独覆盖当前响应式、侧置模式、模块态或专项浮层规格。

本文不使用固定坐标作为核心约束，而使用：

1. 相对画布比例；
2. 相对父容器比例；
3. 锚点和布局路径；
4. 相对字体与图标大小；
5. 统一的 SVG 内部坐标系。

## 一、基准坐标系

### 1. 画布

| 参数 | 定义 |
| --- | --- |
| 基准画布 | `390 × 884` |
| 原始参考图 | `834 × 1886` |
| 宽高比 | `0.44118` |
| 坐标原点 | 左上角 |
| 横向归一化 | `rx = x / 390` |
| 纵向归一化 | `ry = y / 884` |
| 宽度归一化 | `rw = width / 390` |
| 高度归一化 | `rh = height / 884` |

### 2. 实现单位

```text
screenWidth = 当前可用画布宽度
screenHeight = 当前可用画布高度
scaleX = screenWidth / 390
scaleY = screenHeight / 884
uniformScale = min(scaleX, scaleY)
```

使用规则：

- 横向位置和宽度优先依据父容器宽度计算。
- 纵向位置和高度优先依据基准画布的 `uniformScale` 计算。
- 字体、图标、圆角、边框和阴影使用 `uniformScale`，禁止分别使用 `scaleX`、`scaleY` 造成形变。
- 屏幕比例变化时，阅读正文区域吸收高度差；顶部栏和底部控制层不做纵向拉伸。

## 十五、图标路径系统


### 2. 安全区

| 类型 | 安全区 |
| --- | --- |
| 普通线性图标 | `x: 2–22, y: 2–22` |
| 圆形图标 | `x: 2.5–21.5, y: 2.5–21.5` |

- 所有路径必须使用相对统一的视觉中心 `(12,12)`。

- 填充图形只允许播放三角、目录圆点和更多圆点。

| 系统 A 按钮 | 直径 `50%` |

### 3. 阴影

顶部栏：

```text
x = 0
y = 高度 × 0.12

spread = 0

```

```text

```

内部卡片不使用明显悬浮阴影，只允许：

```text

```

## 十七、交互状态参数

| 状态 | 变化 |
| --- | --- |
| 默认 | 使用本文基础参数 |
| 按下 | 图标槽亮度降低约 `3%`；不缩放布局 |
| 选中 | 图标保持深色，按钮底可使用强调蓝 `8%–12%` 混合浅底 |
| 禁用 | 文字和图标使用 `--text-muted`，整体不透明度不得低于 `0.45` |
| 滑块拖动 | 滑块直径放大 `10%`，不改变轨道位置 |
| 控制层进入 | 从自身高度 `100%` 的下方移动到 `0`，不改变正文 |

建议动效：

```text
控制层进入：220ms，cubic-bezier(0.22, 1, 0.36, 1)
控制层退出：180ms，cubic-bezier(0.4, 0, 1, 1)
按钮反馈：90ms，ease-out
滑块反馈：120ms，ease-out
```

## 十八、响应式适配

| 屏幕高度 | 规则 |
| --- | --- |
| `<760px` | 控制层保持组件比例，整体高度最多缩小 `10%` |
| `760–960px` | 底部控制层使用基准高度比例 |
| `>960px` | 控制层不继续按比例增高，额外高度归入正文区 |

### 3. 安全区

```text
topInset = 系统状态栏安全区
bottomInset = 系统手势安全区
```

- 设计图按完整屏幕处理。
- 实际 App 中顶部弱状态可以位于 App 自绘状态区域。
- 底部控制层在正文页上悬浮，底部必须保留可见页面背景。
- 存在 `bottomInset` 时，悬浮底距取 `max(1.13% 画布高, bottomInset)`；三个内部组件的相对比例不变。

## 十九、验收误差

| 项目 | 最大允许误差 |
| --- | --- |
| 主要区域位置 | 画布宽或高的 `0.5%` |
| 主要区域尺寸 | `1%` |
| 内部间距 | 父容器尺寸的 `2%` |
| 字号 | `1px` |
| 图标显示尺寸 | `1px` |
| 图标描边 | `0.15px` |
| 颜色 RGB | 每通道 `±4` |
| 圆角 | `1px` |

## 二十、禁止项

- 禁止把所有元素直接按整个屏幕百分比独立定位。
- 禁止使用绝对坐标覆盖父子布局关系。
- 禁止非等比缩放图标。
- 禁止所有 SVG 使用同一描边。
- 禁止字体随视口无限放大。
- 禁止控制层出现透明背景。
- 禁止底部增加额外进度条或状态栏。
- 禁止控制层与屏幕底边直接相连；必须保留四角和底部悬浮间距。
- 禁止控制层出现后重新计算正文高度。
