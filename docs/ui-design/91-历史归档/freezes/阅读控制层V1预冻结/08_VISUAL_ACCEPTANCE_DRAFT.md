# 视觉验收草案

## P0 必须完全一致

| 项目 | 验收标准 |
| --- | --- |
| 控制层位置 | 各控制层快捷模块的底部控制层相对位置一致，不得上下漂移。 |
| 顶部控制栏位置 | 顶部控制栏在默认态和快捷模块态保持一致。 |
| 底部章节进度条位置 | 控制层内部章节进度条位于底部工具栏上方。 |
| 底部工具栏位置 | 底部四主按钮固定在控制层底部区域，不得下移或被功能面板重画。 |
| 四主按钮数量和顺序 | 必须为 `目录 / 朗读 / 界面 / 设置`，数量、顺序、语义不变。 |
| 正文不重排 | 控制层、快捷区、浮层出现时，阅读正文不移动、不挤压、不重排。 |
| 快捷区不挤压正文 | 快捷区是 overlay / bottom sheet / floating panel，不是正文内容流。 |
| 亮度条为浮层 | 亮度条不参与正文布局，四主按钮切换时保持位置。 |
| 章节进度提示为浮层 | 拖动进度条时的提示不参与正文布局。 |
| 全页弹层层级 | 目录、搜索、替换、自动翻页等弹层不得和底部控制层混乱叠加。 |
| 底部阅读细进度条 | 沉浸页和控制层页不得出现贴屏幕底部的阅读细进度条。 |

## P1 高度一致

| 项目 | 验收标准 |
| --- | --- |
| 字号 | 正文、标题、按钮、标签大小接近冻结图。 |
| 图标尺寸 | 顶部图标、四主按钮图标、功能按钮图标保持一致。 |
| 间距 | 面板内分区、按钮、卡片间距稳定。 |
| 圆角 | 顶部栏、底部面板、卡片、按钮圆角风格一致。 |
| 遮罩透明度 | 控制层遮罩强度接近冻结图。 |
| 面板高度 | 快捷模块面板高度和默认控制层高度一致。 |
| 正文排版密度 | 字距、行高、页边距与冻结图接近。 |

## P2 允许微调

| 项目 | 验收标准 |
| --- | --- |
| 阴影 | 可因平台渲染略有差异。 |
| 抗锯齿 | 允许平台字体和图标渲染差异。 |
| 小范围色差 | 允许轻微色差，但不能改变主题风格。 |
| 设备安全区微调 | 不同设备安全区可微调，但 P0 位置关系不得破坏。 |

## 截图回归建议

- 基准图来自正式冻结包 `images/`。
- 运行截图必须与基准图进行人工或工具对比。
- P0 项不允许明显偏移。
- P1 项要求高度一致。
- P2 项允许小范围平台差异。
