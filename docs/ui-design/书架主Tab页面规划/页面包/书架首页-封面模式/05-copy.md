# 书架首页封面模式 文案稿

## 固定文案

- 页面标题：`书架`
- 分组：`全部`
- 顶部分区：`继续阅读`
- 顶部分区：`最近更新`
- 书架分区：`我的书架`
- 展示模式：`封面`
- 展示模式：`列表`
- 继续阅读按钮：`阅读`
- 错误标题：`书架加载失败`
- 错误说明：`请检查网络或稍后重试`
- 主按钮：`重试`
- 离线提示：`当前网络不可用，显示本地缓存`
- 部分失败提示：`部分书籍更新失败`

## 可变文案

- 用户自定义分组名称。
- 书名。
- 作者。
- 当前章节。
- 最新章节。
- 阅读进度百分比。

## 文案约束

- 候选图里的 `默认`、`本地书`、`追更` 只是示例，不是固定文案。
- 不在书架首页展示书源名称。
