# 书架首页封面模式 状态稿

## Default

显示顶部栏、分组、顶部信息卡片、封面网格和底部导航。

## Loading

- 保留顶部栏、分组和底部导航。
- 顶部信息卡片和封面网格显示骨架。
- 不使用全屏转圈。

## Empty

进入书架空状态页面稿。

## Current Group Empty

- 保留顶部栏、分组和底部导航。
- 内容区显示当前分组为空文案。
- 不自动切回 `全部`。

## Error

- 保留页面框架。
- 内容区显示 `书架加载失败`、说明和 `重试`。
- 不跳全局错误页。

## Offline

- 有本地缓存时继续显示书架内容。
- 用行内提示说明 `当前网络不可用，显示本地缓存`。

## Partial Success

- 保留已有书籍。
- 轻提示 `部分书籍更新失败`。
- 不清空列表。
