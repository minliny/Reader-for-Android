# 书架首页封面模式 交互稿

## 入口

- App 启动默认进入书架。
- 底部导航点击 `书架`。
- 从阅读、搜索、详情返回。

## 返回

- 位于书架根页面时，系统返回按 App 根页面规则处理。
- 从次级页返回时，恢复上次分组、排序、展示模式和滚动位置。

## 操作

- 搜索按钮：进入书籍搜索首页。
- 更多按钮：打开书架级菜单。
- 分组 chip：切换当前分组并刷新书架内容。
- `阅读`：进入阅读沉浸页。
- 书籍封面或卡片主体：进入阅读沉浸页。
- 书名/元信息区域：进入书籍详情页。
- 长按书籍或行级更多：打开书籍操作底表。
- `封面` / `列表`：切换展示模式。
- 排序按钮：打开排序与筛选底表。
- 布局按钮：打开布局设置入口，P0 不展开完整设置页。

## 动效

- 封面/列表切换：180-220ms 交叉淡化或轻量重排。
- chip 切换：指示状态 150-180ms。
- 卡片按压：轻微缩放，不改变布局尺寸。
- 返回书架：恢复滚动位置，不重新跳顶，除非数据源被重置。
