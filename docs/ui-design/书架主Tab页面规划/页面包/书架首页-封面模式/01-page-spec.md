# 书架首页封面模式 页面结构稿

## 页面目的

书架首页封面模式用于展示用户已加入书架的书籍，并提供继续阅读、最近更新、分组筛选、搜索和书架级管理入口。

## 权威图片

- `images/default.png`

该图片只作为结构和视觉方向参考；最终组件 token 以本页面包的组件规格为准。图片中的固定分组名称和三列布局不能被视为冻结规则。

## 页面层级

1. 系统状态栏。
2. 顶部栏：标题 `书架`、搜索按钮、更多按钮。
3. 横向分组筛选区。
4. 顶部信息区：`继续阅读` 卡片、`最近更新` 卡片。
5. 书架标题行：`我的书架`、当前模式 `封面`、排序按钮、布局按钮。
6. 封面网格：书籍封面卡片。
7. 底部主导航：书架、发现、RSS、设置。

## 固定区和滚动区

- 固定区：系统状态栏、顶部栏、底部主导航。
- 可横向滚动区：分组筛选。
- 可纵向滚动区：顶部信息区和封面网格整体。
- 书架标题行随内容滚动，不做吸顶。

## 数据字段

每本书需要字段：

- 书名。
- 作者。
- 当前章节或最新章节。
- 阅读进度。
- 封面图。
- 是否已读完。
- 是否有更新。

书架首页不常态展示书源信息。

## 自适应目标

页面必须优先保证：

1. 顶部栏和底部导航不被裁切。
2. 书名、作者、章节不与封面或进度条重叠。
3. 封面比例稳定。
4. 网格列数根据可用宽度自适应，而不是固定三列。
5. 短屏设备减少顶部信息区高度，保留封面网格首行可见。
