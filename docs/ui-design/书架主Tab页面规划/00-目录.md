# 书架主 Tab 页面规划目录

审计日期：2026-06-13

## 当前定位

书架是当前唯一 P0 主 Tab。发现和 RSS 暂缓，设置属于 P1。

书架主 Tab 收敛范围包括：

1. 书架首页。
2. 书架空状态。
3. 书架分组管理。
4. 书籍操作底表。
5. 书架排序与筛选。
6. 书籍搜索首页。
7. 书籍搜索结果。
8. 搜索 loading / empty / error。
9. 书籍详情。
10. 书籍详情 loading / 来源失效。
11. 阅读入口。

不纳入书架 P0：

- 发现页面与发现次级页。
- RSS 页面与 RSS 次级页。
- 设置页面与设置次级页。
- 本地书导入，归入 P1。
- 书源管理完整链路。
- 我的、账户、头像、登录。

## 完成标准

每个页面必须满足：

1. 有明确入口。
2. 有明确返回。
3. 有中文文字稿。
4. 有唯一当前 source 图片。
5. 无历史稿、参考稿、多版本冲突。
6. 通过页面级审核。
7. 符合 `19-页面设计输入约束示例.md` 的文字稿和图片示例要求。
