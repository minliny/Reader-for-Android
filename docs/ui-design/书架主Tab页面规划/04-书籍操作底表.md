# 书籍操作底表

文字稿状态：待产品确认

## 页面定位

书籍操作底表用于对单本已加入书架的书籍执行低频操作。它只处理书架内书籍管理，不处理复杂书源切换。

## 当前 source

缺图：建议编号 `49-1`。

## 入口

- 书架条目更多按钮。
- 长按书籍条目。

## 返回

- 点击遮罩、下滑底表、点击系统返回：关闭底表，回到原书架状态。
- 操作成功后：关闭底表或停留在底表显示反馈，按操作类型决定。
- 操作取消后：关闭确认弹窗，回到底表。
- 关闭后保留当前分组、排序、展示模式和滚动位置。

## 按钮与操作

- `继续阅读`：进入阅读沉浸页。
- `查看信息`：进入书籍详情。
- `刷新目录`：刷新当前书籍目录。
- `缓存书籍`：进入缓存任务确认或开始缓存。
- `移动分组`：打开分组选择底表。
- `导出`：进入导出位置或系统分享面板。
- `移出书架`：打开二次确认。
- `同时清理本书缓存`：复选框，默认不选中。
- `移出`：危险按钮，确认移出。
- `取消`：关闭确认弹窗。

按钮状态：

- 刷新目录中：`刷新目录` 显示加载态。
- 缓存中：`缓存书籍` 显示当前任务状态。
- 不支持导出时：`导出` 禁用，并显示原因。
- 移出提交中：`移出` 显示加载态，防止重复点击。

## 状态文案

loading：

- 底表打开时如果书籍元信息未加载，摘要区域显示骨架。

empty：

- 不适用。

error：

- 刷新目录失败：`目录刷新失败，请重试`
- 移动分组失败：`移动分组失败，请重试`
- 导出失败：`导出失败，请重试`

permission：

- 导出需要系统授权时提示：`需要选择保存位置`

offline：

- 网络不可用时，`刷新目录` 和网络缓存操作提示：`当前网络不可用`

partial success：

- 缓存部分章节失败：`部分章节缓存失败`

invalid source：

- 来源失效不在底表内展开，跳转到书籍详情或阅读异常换源流程。

## 错误与空状态

- 操作失败时不关闭底表，保留用户上下文。
- 移出失败时书籍仍保留在书架。
- 移动分组失败时书籍仍保留在原分组。

## 不允许

- 不在底表中切换书源。
- 不在底表中展示复杂书源规则。
- 不出现账号、分享社区、推荐算法入口。
- 不把 `移出书架` 写成 `删除书籍`。
