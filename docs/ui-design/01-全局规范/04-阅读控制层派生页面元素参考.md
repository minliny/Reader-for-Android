# 基于 04 号图的页面元素样式定义

> 本文件不再作为阅读控制层结构定义源，最新规格以
> `READER_CONTROL_LAYER_SPEC.md`、`READER_CONTROL_ACTION_FLOWS.md`、
> `READER_CONTROL_RESPONSIVE_RULES.md`、`READER_CONTROL_GEOMETRY_SPEC.md`、
> `READER_CONTROL_IMAGE_USAGE.md` 为准。
> 本文件仅保留阅读控制层及阅读内覆盖层的历史视觉语言。设置、完整管理页及相关状态页
> 的当前视觉参数以 `SETTINGS_PAGE_DESIGN_SPEC.md` 为准。

本文从 `docs/ui-design/04-阅读链路/阅读控制层/图片/01-阅读控制层-默认总控态.png` 抽取功能结构，并以
`docs/ui-design/04-阅读链路/阅读控制层/图片/03-当前视觉基线.png` 作为唯一视觉和
几何母版，用于约束后续页面、Stitch 提示词和 UI 示例图生成。`04` 不代表所有页面都
要出现阅读控制层，也不得用于覆盖 `04-2` 的精确颜色和几何参数。

## 一、继承原则

| 原则 | 说明 |
| --- | --- |
| 视觉母版 | 阅读控制层及阅读内覆盖层优先继承 `04-2`；完整设置页和管理页不得继承其 pill 顶栏、粗图标或大圆角。 |
| 结构不混用 | 阅读控制层、App 主页面、完整管理页是三类结构，不能把底部亮度条、章节进度条、App 主底栏互相混用。 |
| 面板不透明 | 所有可交互面板、卡片、按钮底、底表主体必须使用不透明色。阴影可透明，但面板本身不可半透明。 |
| 图标统一 | 图标应使用 04 号图的粗黑圆角线性风格，不使用 Material 默认图标、细线 lucide 风格或彩色插画图标。 |
| 正文不重排 | 阅读场景中，控制层、快捷区、浮层出现时覆盖正文，不改变正文排版。 |

## 三、图标语言

04 号图的图标不是默认 Material 风格，也不是细线 lucide 风格。后续页面应按下面规则定义。

| 类型 | 规则 |
| --- | --- |
| 线条 | 黑色、较粗、端点圆角、连接圆角。 |
| 复杂度 | 只保留必要轮廓，不加阴影、不加彩色、不加装饰性细节。 |
| 尺寸 | 同一组按钮必须放入相同大小图标槽，视觉尺寸一致。 |
| 填充 | 原则上不填充；播放三角、目录圆点等可使用黑色小填充。 |
| 文本图标 | `Aa`、`A` 等文字图标必须居中，字号与图形图标视觉重量一致。 |

典型图标定义：

| 功能 | 图标形态 |
| --- | --- |
| 返回 | 粗线左箭头。 |
| 更多 | 竖向三点，点径统一。 |
| 换源 | 两条交叉/切换箭头，箭头端简洁。 |
| 搜索 | 放大镜，圆环粗线，手柄 45 度。 |
| 自动翻页 | 圆环 + 播放三角，圆环可带短缺口。 |
| 替换 | 上下相对回转箭头。 |
| 目录 | 三条列表线 + 左侧圆点。 |
| 朗读 | 竖向声波条，中间最高，两侧递减。 |
| 设置 | 六边形外轮廓 + 中心圆。 |
| 亮度 | 太阳圆环 + 八向短线。 |

## 四、通用组件定义

### 1. 顶部栏

| 项 | 定义 |
| --- | --- |
| 使用页面 | 阅读页顶部控制栏和阅读内覆盖层。完整管理页、设置页改用 `SETTINGS_PAGE_DESIGN_SPEC.md`。 |
| 高度 | 阅读控制层顶部栏参考 `59px`；完整页顶栏可保持 `56px` - `64px`。 |
| 背景 | `#FFFAF1` 或页面底色上不透明浅色面板。 |
| 边框 | 低对比暖灰边框。 |
| 图标 | 左侧返回、右侧更多/搜索/新增等均使用 04 图标语言。 |
| 禁止 | 不使用高饱和彩色 AppBar，不使用 Material 默认大标题栏。 |

### 2. 主底栏

| 项 | 定义 |
| --- | --- |
| 使用页面 | 书架、发现、RSS、设置。 |
| 固定入口 | `书架 / 发现 / RSS / 设置`。 |
| 样式 | 暖色不透明底，轻边框，图标粗黑圆角线性，选中态使用 `#087EAD` 或暖色浅底。 |
| 禁止 | 不出现 `我的`；不出现阅读亮度条和章节进度条。 |

### 3. 卡片

| 类型 | 使用场景 | 样式 |
| --- | --- | --- |
| 书籍卡片 | 书架、发现、搜索结果 | 不透明浅底，圆角 12px 左右，封面/标题/作者/来源信息分区清楚。 |
| 信息卡片 | 阅读内信息面板 | 暖浅底，低对比边框，标题较重，说明文字弱化。 |
| 操作卡片 | 快捷区、底表 | 内层底色 `#FFFDF8`，按钮底 `#EEF0ED`。 |

### 4. 列表行

| 项 | 定义 |
| --- | --- |
| 高度 | 阅读控制层紧凑行建议 `52px` - `64px`；完整设置页不得引用本参数。 |
| 内容 | 左侧图标或标题，中间说明，右侧状态/开关/箭头。 |
| 图标 | 34px 图标槽或 28px 轻量图标槽；同页保持一致。 |
| 分隔 | 使用暖灰细分割线或卡片内间距，不使用深色硬线。 |

### 5. 搜索框

| 项 | 定义 |
| --- | --- |
| 背景 | `#FFFDF8` 或 `#F7F2EB`，不透明。 |
| 图标 | 粗线放大镜，左侧固定。 |
| 圆角 | 12px 或胶囊圆角，跟所在页面密度一致。 |
| 文案 | 中文占位，如 `搜索书名、作者或关键词`。 |

### 6. 设置项

| 项 | 定义 |
| --- | --- |
| 分组 | 使用不透明分组卡片，不嵌套卡片。 |
| 行结构 | 标题 + 可选说明 + 右侧控件。 |
| 控件 | 开关、滑块、分段控件、选择项均继承 04 的圆角和暖色。 |
| 强调 | 开关开启、滑块进度使用 `#087EAD`。 |

### 7. 底表 / 覆盖层

| 项 | 定义 |
| --- | --- |
| 使用场景 | 阅读内搜索、替换、换源、更多菜单、确认操作。 |
| 位置 | 从底部上浮或覆盖阅读内容，不改变正文排版。 |
| 背景 | 主体不透明 `#FFFAF1`，内层不透明 `#FFFDF8`。 |
| 圆角 | 顶部圆角 18px 左右，带短拖拽条。 |
| 禁止 | 不出现半透明底表主体；不把 App 主底栏放进阅读控制层。 |

### 8. 状态模板

| 状态 | 元素定义 |
| --- | --- |
| loading | 暖色页面底，中心轻量加载符号，短文案，不使用强彩色加载器。 |
| empty | 简化线性图标 + 主文案 + 次文案 + 一个主按钮。 |
| error | 简化线性警示图标 + 错误文案 + 重试按钮。 |
| delete_confirm | 不透明确认弹窗或底表，标题明确，危险按钮用克制红色，不大面积红底。 |

## 五、按页面类型继承

| 页面类型 | 应继承 04 的元素 | 不应继承 |
| --- | --- | --- |
| 阅读沉浸页 | 纸张底、正文排版、顶部弱信息 | 底部控制层、主底栏 |
| 阅读控制层 | 顶部控制栏、底部控制层、亮度栏、章节进度、四主按钮 | App 主底栏 |
| 阅读快捷区 | 底部控制层结构、右侧亮度栏、底部四主按钮 | 完整页面布局 |
| 阅读完整管理页 | 顶栏、卡片、列表行、设置项、粗线图标 | 常驻亮度条、章节进度条 |
| 书架 | 主底栏、书籍卡片、搜索/筛选控件 | 阅读控制层、亮度条 |
| 发现 | 主底栏、来源选择、分类切换、书籍卡片 | 阅读章节进度、阅读亮度条 |
| RSS | 主底栏、订阅源列表、文章卡片、状态模板 | 阅读控制层 |
| 设置 | 使用 `SETTINGS_PAGE_DESIGN_SPEC.md` | 账户中心、我的页、阅读控制层 |
| 书源管理 | 使用 `SETTINGS_PAGE_DESIGN_SPEC.md` | 二级页不出现 App 主底栏或阅读控制层 |

## 七、给图片生成的硬性提示词片段

后续生成阅读控制层或阅读内覆盖层时，若需要继承 04 号图风格，应加入以下约束。该片段
不得用于完整设置页、书源管理页或 App 设置二级页：

```text
功能结构参考 docs/ui-design/04-阅读链路/阅读控制层/图片/01-阅读控制层-默认总控态.png，视觉和几何必须继承
docs/ui-design/04-阅读链路/阅读控制层/图片/03-当前视觉基线.png：
- 暖纸质中文阅读 App，不使用默认 Material Design。
- 所有可交互面板和卡片必须是不透明浅暖色，不允许半透明主体。
- 图标必须是粗黑、圆角端点、简化线性的 04 号图风格，不使用细线 lucide、Material 默认图标或彩色插画图标。
- 同一组按钮的图标槽尺寸一致，图标视觉大小一致。
- 大面板圆角约 15-18px，内层卡片圆角约 12px，图标按钮底约 34×34px。
- 页面纸张底色使用 #F8F1E7，主文字使用 #11100E。
- 强调色使用低饱和蓝 #087EAD，外层边框使用暖灰 #D5C8B8。
- 主底栏只用于 App 主页面：书架 / 发现 / RSS / 设置。
- 阅读控制层页面不得出现 App 主底栏；App 主页面不得出现阅读亮度条和章节进度条。
```

## 八、已决议与实现约束

| 项 | 原因 |
| --- | --- |
| 精确 dp 值 | 阅读控制层使用 `READER_CONTROL_GEOMETRY_SPEC.md` 已定义的 `390×884dp` 几何；Android 按 dp/sp 实现，不再从 PNG 二次猜测。 |
| 主底栏最终高度 | App Shell 使用 `80dp`（含内容区与底部安全区适配）；图标/标签内容区 `56dp`，额外安全区由系统 inset 吸收。 |
| 非阅读页背景 | 书架、发现、RSS、设置统一使用暖纸张底 `#F8F1E7`；卡片使用 `#FFFCF7/#FFFCF8`。 |
| 图标资产来源 | 使用项目自有 SVG path 集；可参考开源图标的语义，但必须按 04-2 圆角端点、线宽和视觉边界重绘，不直接依赖第三方运行时图标库。 |
