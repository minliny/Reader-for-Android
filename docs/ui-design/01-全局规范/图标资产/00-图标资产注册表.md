# 图标资产注册表

状态：P0 源资产包
适用范围：主标签页应用外壳、阅读控制层
维护原则：页面规格稿引用本注册表中的语义名和 SVG 路径，生成稿直接复用源文件。

## 0. 当前采用策略

当前 UI 设计图中的图标与手画 SVG 不一致。已开始从设计图直接抠出 PNG 参考资产，但当前抠图仍在修正中，不能作为最终图标资产直接使用：

```text
docs/ui-design/01-全局规范/图标资产/从设计图抠图/
  原始局部/
  完整组件局部/
  透明图标/
  透明图标-紧裁/
  标准化96/
  01-抠图资产总览.png
  02-标准化96总览.png
  03-完整组件局部总览.png
```

SVG 文件保留为后续矢量重绘目标，不作为当前视觉基准。当前可信基准优先看 `完整组件局部/`，透明图标和标准化 96 只作为中间产物。

## 4. 页面引用格式

页面规格稿中的图标写法：

```text
图标：reader.catalog
资产：docs/ui-design/01-全局规范/图标资产/reader-control/catalog.svg
槽位：64dp
视觉尺寸：26dp
颜色：继承当前组件 active/inactive color
```

同一页面内的图标颜色由组件 token 决定，SVG 文件只提供形状和线条结构。
