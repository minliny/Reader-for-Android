# App Shell 组件视觉规格

状态：主标签页 根页面共享 token 草案
适用页面：书架首页、发现首页、RSS 首页、设置首页

## 1. 页面与安全区

| Token | 建议值 | 说明 |
| --- | --- | --- |
| `AppShell.Page.Background` | `#F8F5EF` | 主标签页 统一暖白背景 |
| `AppShell.Page.HorizontalPadding` | `16dp` | 常规手机左右边距 |
| `AppShell.Page.CompactHorizontalPadding` | `14dp` | 宽度小于 360dp |
| `AppShell.Page.MaxContentWidth` | `720dp` | 大屏/折叠屏居中内容最大宽 |
| `AppShell.SafeArea.Top` | follow system inset | 顶部内容避让状态栏 |
| `AppShell.SafeArea.Bottom` | bottom nav + system inset | 滚动内容避让底部主导航 |

主标签页 根页面不得各自使用不同背景色。页面根背景、底部导航背景和卡片背景必须形成稳定层级。

## 2. 顶部栏

| Token | 建议值 | 说明 |
| --- | --- | --- |
| `AppShell.TopBar.MinHeight` | `64dp` | 不含系统状态栏 |
| `AppShell.TopBar.TitleSize` | `28sp` | 主标签页 标题 |
| `AppShell.TopBar.TitleWeight` | `600` | 标题字重 |
| `AppShell.TopBar.IconSize` | `28dp` | 搜索/更多视觉尺寸 |
| `AppShell.TopBar.IconStroke` | `2dp` | 线性图标线宽 |
| `AppShell.TopBar.TouchTarget` | `44dp` | 图标按钮最小触控区 |
| `AppShell.TopBar.IconColor` | `#24201A` | 图标颜色 |

书架、发现、RSS、设置的标题位置、右侧图标尺寸和触控区必须一致。

## 3. 通用卡片

| --- | --- | --- |
| `AppShell.Card.Background` | `#FFFDF8` | 通用卡片背景 |

| `AppShell.Card.BorderWidth` | `1dp` | 统一边框 |

## 8. 响应式规则

| 条件 | 规则 |
| --- | --- |
| 宽度 `<360dp` | 使用紧凑左右边距，优先保留标题、主要操作和首个内容卡 |
| 宽度 `360-430dp` | 标准手机布局，内容按页面包规则排列 |
| 宽度 `>430dp` | 内容最大宽度居中，列表和卡片不无限拉宽 |
| 高度 `<760dp` | 压缩顶部概览卡高度，优先让列表首项可见 |
| 横屏/折叠屏 | 本阶段不冻结主标签页 横屏图，但实现必须按 MaxContentWidth 居中 |

## 9. Stitch 使用约束

Stitch 生成四个主标签页 根页面时：

- 必须复用本文件 token。
- 不得根据某张图单独创造底部导航、顶部栏或卡片样式。
- 图片与本文件冲突时，以本文件为准。
- 暂缓页面只生成根页面，不补全次级页。
