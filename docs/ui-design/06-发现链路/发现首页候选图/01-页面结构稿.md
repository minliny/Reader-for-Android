# 发现首页候选图页面结构稿

## 页面目标

保存发现首页当前候选图及其复原约束。

## 入口

发现主 Tab 页面包或 Stitch 输入包。

## 页面区域

1. 顶部安全区和标题区：提供页面标题、返回或关闭入口。
2. 主内容区：承载本页面核心内容，滚动时不得遮挡底部操作或系统手势区。
3. 操作区：主操作优先，危险操作必须二次确认。
4. 反馈区：loading、empty、error、offline 使用统一状态组件。

## 自适应规则

- 手机竖屏以 `390x884dp` 为逻辑基准。
- 宽度小于 `360dp` 时使用紧凑边距，长文案换行，不压缩按钮触控区。
- 高度不足时优先让主内容区滚动，顶部标题和关键操作保持可达。
- 页面不得依赖固定像素截图；组件尺寸以全局 token 和页面包文字稿为准。
