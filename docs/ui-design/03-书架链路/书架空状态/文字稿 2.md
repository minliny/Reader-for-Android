# 书架空状态

## 页面内容
### 页面目标

当书架没有可展示书籍时，引导用户搜索书籍、导入本地书或进入书源能力。

### 入口

书架首页无书籍、当前分组无书籍、筛选后无结果。

### 页面区域

1. 顶部安全区和标题区：提供页面标题、返回或关闭入口。
2. 主内容区：承载本页面核心内容，滚动时不得遮挡底部操作或系统手势区。
3. 操作区：主操作优先，危险操作必须二次确认。
4. 反馈区：loading、empty、error、offline 使用统一状态组件。

### 专项约束补充

#### 结构执行规则

- 顶部、滚动区、底表/浮层和状态层必须分层实现，不能合并成单个自由排版容器。

### 组件清单

空状态插画、标题、副说明、主操作按钮、次操作按钮、底部主导航占位避让。

### 组件约束

- 图标只表达明确动作；无法确认语义时使用文字按钮。
- 危险操作使用危险色和二次确认，不与普通主按钮同层级混淆。

### 专项约束补充

#### 页面专属组件

- `EmptyStateCard`
- `PrimaryActionButton`
- `SecondaryTextButton`
- `BottomNav`

#### 组件执行规则

- 未在本清单出现的组件只能作为状态反馈或系统控件出现，不能成为新的主功能入口。

### 核心文案

- `暂无书籍`
- `去搜索`
- `导入本地书`
- `换个分组看看`

### 通用反馈文案

- `正在加载`
- `加载失败，请重试`
- `网络不可用，请稍后重试`
- `保存成功`
- `操作失败，请重试`
- `取消`
- `确认`

### 文案规则

- 中文文案短句优先，不使用技术字段名。
- 危险操作必须说明影响范围。
- 不显示账号、社区、推荐算法等当前范围外文案。

### 专项约束补充

#### 必须覆盖的页面文案

- `书架暂无书籍`
- `添加书籍`
- `导入本地书`

#### 文案执行规则

- 生成稿不得替换这些核心中文文案。
- 状态文案必须说明原因和下一步，不能只写通用失败。
- 按钮文案必须与交互结果一致，不能使用含糊的 `确定` 替代关键动作。

## 交互逻辑
### 主要操作

搜索书籍、导入本地书、切换分组、返回书架首页。

### 返回规则

- 从入口页进入时，系统返回回到入口页并保留入口页滚动位置和筛选条件。
- 弹层、底表、确认框优先关闭当前浮层，再返回上级页面。
- 操作提交中禁止重复点击主按钮。

### 反馈规则

- 成功反馈使用轻量 toast 或行内状态，不改变页面结构。
- 失败反馈保留用户输入和当前上下文，提供重试入口。
- 网络不可用时只阻断依赖网络的动作，不阻断本地查看和关闭页面。

### 手势规则

- 底表支持点击遮罩、下滑、系统返回关闭。
- 列表页支持纵向滚动，不额外定义横向手势。
- 长按只用于已有明确业务入口的书籍或列表项，不新增隐藏功能。

### 专项约束补充

#### 页面关键操作

- 点击 添加书籍 进入书籍搜索或添加入口
- 点击 导入本地书 进入本地书导入 P1 页面
- 底部导航只允许切换四个主 Tab

#### 交互执行规则

- 所有操作必须保留来源页上下文，包括滚动位置、筛选条件、当前章节或当前设置值。
- 提交中禁用重复点击；失败时保留用户已输入或已选择内容。
- 浮层关闭顺序为：先关闭当前浮层，再返回上级页面。

### default

展示当前页面的完整默认内容，所有主操作可见且可达。

### loading

- 首次进入可使用骨架屏或居中加载。
- 局部刷新只替换对应局部内容，不清空整页。

### empty

- 空状态必须说明原因，并给出一个主行动作。
- 空状态不得显示错误语气。

### error

- 错误状态保留返回入口。
- 可恢复错误提供 `重试`；不可恢复错误给出原因和下一步。

### offline

- 网络不可用时保留本地数据展示。
- 需要联网的按钮进入禁用或点击后提示 `网络不可用，请稍后重试`。

### permission

- 需要系统权限时先说明用途，再提供去授权入口。
- 未授权不得进入空白页。

### 专项约束补充

#### 页面状态定义

- `default`：无书时展示空态卡和两个操作
- `loading`：仅用于书架数据首次检查
- `error`：书架数据读取失败时提供重试
- `permission`：本地导入前不得提前请求全盘权限

#### 状态执行规则

- 状态层不得改变页面主结构，只替换对应内容区或显示可恢复反馈。

### 功能禁止

- 不新增当前页面包未声明的入口、按钮或跳转。
- 不引入账号、社区、会员、推荐流、广告或云端商业化入口。

### 交互禁止

- 不在提交中允许重复点击。
- 不在关闭弹层后丢失来源页筛选、分组、排序和滚动位置。
- 不用隐藏手势承载唯一关键操作。

### 专项约束补充

#### 页面专属禁止项

- 不得把空态做成营销页或推荐书城
- 不得出现 我的/账号/会员 入口
- 不得把本地导入作为 P0 强制流程
