# 书架空状态

状态：正式 UI 设计稿已建立
主入口：`10-正式UI设计稿.md`

## 页面定位

书架空状态用于首次进入、清空书架后，或当前自定义分组没有书籍时。它属于书架首页内部状态，不是全局空状态页。

## 当前 source

候选图：`docs/ui-design/03-书架链路/书架空状态/图片/01-书架空状态.png`

图片状态：组件参考；如与正式稿冲突，以 `10-正式UI设计稿.md` 和 `08-书架空状态视觉规格.md` 为准。

## 入口

- `书架` route 首次加载后无书籍。
- 用户清空书架后返回书架首页。
- 用户切换到没有书籍的自定义分组。

## 返回

- 保留在书架首页。
- 点击系统返回按书架首页根页面规则处理。
- 从搜索或 P1 本地导入返回后，仍回到书架首页。

## 页面结构

必须保留：

- 顶部栏：`书架`
- 图标按钮：`搜索`
- 图标按钮：`更多`
- 分组筛选。
- 底部主导航。

主体内容：

- 轻量书本图标。
- 标题。
- 说明。
- 主按钮。
- 次按钮。

## 主要文案

全书架为空：

- 标题：`书架还是空的`
- 说明：`可以搜索网络书籍，或导入本地文件`
- 主按钮：`搜索书籍`
- 次按钮：`导入本地书`

当前分组为空：

- 标题：`当前分组没有书籍`
- 说明：`可以切换分组，或把书籍移动到这个分组`
- 主按钮：`搜索书籍`
- 次按钮：`管理分组`

## 按钮与操作

- `搜索书籍`：主按钮，进入书籍搜索首页。
- `导入本地书`：次按钮，进入 P1 本地书导入流程；不阻塞书架 P0。
- `管理分组`：次按钮，进入书架分组管理。
- `搜索`：图标按钮，进入书籍搜索首页。
- `更多`：图标按钮，打开书架级菜单。

按钮状态：

- 搜索入口始终可用。
- 本地导入入口在 P0 可展示，但完整导入流程归 P1。
- 管理分组仅在当前分组为空时展示。

## 状态文案

loading：

- 不适用，loading 属于书架首页。

empty：

- 使用本页定义的全书架为空和当前分组为空两类文案。

error：

- 不适用，错误属于书架首页加载失败。

permission：

- 不适用。

offline：

- 网络不可用时仍可展示空状态。

partial success：

- 不适用。

invalid source：

- 不适用。

## 错误与空状态

- 全书架为空和当前分组为空必须使用不同标题。
- 空状态不隐藏分组筛选和底部导航。
- 空状态不自动跳转到搜索页。

## 不允许

- 不使用营销插画或大面积宣传语。
- 不出现账号、登录、云同步引导。
- 不把 P1 本地书导入流程当作 P0 必须闭合项。

## 完成条件

- 默认主图覆盖全书架为空；当前分组为空作为文字变体。
- `导入本地书` 在 P0 仅作为入口保留，完整流程归 P1。
