## 4. 页面生成约束

书架空状态是书架首页内部内容状态，不是独立页面。页面必须保留当前书架 App Shell：顶部栏、分组 chip、底部四项导航和书架激活态。

空状态内容区包含图形、标题、说明、主按钮和次按钮。全书架为空时主按钮为 `搜索书籍`，次按钮为 `导入本地书`；当前分组为空时主按钮为 `搜索书籍`，次按钮为 `管理分组`。

当前图片中的旧底部导航只用于识别历史状态，不参与生成。正式 HTML 必须使用 `书架 / 发现 / RSS / 设置` 四项底部导航，当前激活项为书架。

响应式以 `833 x 1888 px` 为基准。小屏优先保留顶部栏、分组 chip、空状态主体、主按钮和底部导航；短屏允许内容区纵向滚动。

## 5. 验收清单

| 项目 | 通过标准 |

| 输出状态 | 初次生成只能标记 `HTML_DRAFT` |
