# 书架空状态视觉规格

状态：正式组件 token
适用范围：全书架为空、当前分组为空

## 1. 继承关系

- App Shell、页面背景、顶部栏、分组 chip、底部主导航继承 `../../02-主标签页/书架/08-书架首页组件视觉规格.md`。
- 本规格只定义书架内容区为空时的空状态模块，不定义新的主页面结构。

## 2. 空状态模块

| Token | 值 |
| --- | --- |
| `ShelfEmpty.Container.MinHeight` | `360dp` |
| `ShelfEmpty.Container.PaddingX` | `24dp` |
| `ShelfEmpty.Container.Align` | `center` |
| `ShelfEmpty.Icon.Size` | `72dp` |
| `ShelfEmpty.Icon.Color` | `#7A7166` |
| `ShelfEmpty.Title.Size` | `20sp` |
| `ShelfEmpty.Title.Weight` | `600` |
| `ShelfEmpty.Body.Size` | `14sp` |
| `ShelfEmpty.Body.Color` | `#5F574F` |
| `ShelfEmpty.Body.MaxWidth` | `280dp` |
| `ShelfEmpty.ButtonGap` | `10dp` |

空状态不允许使用营销插画或大横幅。图标保持轻量、低干扰。

## 3. 操作按钮

| Token | 值 |
| --- | --- |
| `ShelfEmpty.PrimaryButton.Height` | `44dp` |
| `ShelfEmpty.PrimaryButton.Radius` | `14dp` |
| `ShelfEmpty.SecondaryButton.Height` | `44dp` |
| `ShelfEmpty.ButtonText.Size` | `15sp` |

主按钮固定为 `搜索书籍`。次按钮根据状态显示 `导入本地书` 或 `管理分组`。

## 4. 文案变体

| 场景 | 标题 | 说明 | 次按钮 |
| --- | --- | --- | --- |
| 全书架为空 | `书架还是空的` | `可以搜索网络书籍，或导入本地文件` | `导入本地书` |
| 当前分组为空 | `当前分组没有书籍` | `可以切换分组，或把书籍移动到这个分组` | `管理分组` |

## 5. Stitch 要求

- 空状态必须保留顶部栏、分组 chip 和底部主导航。
- 空状态只替换书籍内容区，不重画 App Shell。
- `导入本地书` 是 P1 入口，不要求本页闭合导入流程。
