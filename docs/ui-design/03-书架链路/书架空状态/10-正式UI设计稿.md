# 书架空状态正式 UI 设计稿

状态：正式稿
等级：P0 / COMPONENT_MASTER
适用页面：书架首页空内容区
视觉参考：`图片/01-书架空状态.png`

## 1. 设计目标

书架空状态是书架首页内部状态，不是独立页面。它在保持 App Shell 和分组筛选的前提下，引导用户搜索书籍、导入本地书或管理分组。

## 2. 页面结构

```text
BookshelfPage
  AppTopBar
  ShelfGroupChipRail
  ShelfEmptyState
    Icon
    Title
    Body
    PrimaryButton("搜索书籍")
    SecondaryButton
  BottomNav(active="书架")
```

## 3. 状态变体

| 场景 | 主按钮 | 次按钮 |
| --- | --- | --- |
| 全书架为空 | `搜索书籍` | `导入本地书` |
| 当前分组为空 | `搜索书籍` | `管理分组` |

## 4. 交互

| 操作 | 结果 |
| --- | --- |
| 搜索书籍 | 进入 `../书籍搜索/00-说明.md` |
| 导入本地书 | 进入 P1 `../本地书导入/00-说明.md` |
| 管理分组 | 进入 `../分组管理/00-说明.md` |

## 5. 明确排除

- 不隐藏分组 chip。
- 不隐藏底部主导航。
- 不展示账号、登录、云同步、会员或广告。
- 不把本地书导入作为 P0 闭合流程。
