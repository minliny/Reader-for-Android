# 书架主 Tab 图片草稿目录

本目录只收纳书架主 Tab 收敛范围内的图片草稿和引用副本。全局 `docs/ui-design/91-历史归档/drafts/旧全局草稿目录/` 不再登记书架、书籍搜索、书籍详情的 Tab 专属候选图。

## 命名规则

- 使用中文页面名，前缀数字只表示本目录内的阅读顺序。
- 默认态、状态态、弹层态分开命名。
- `引用副本` 表示该图服务于书架主 Tab 文档串联，但权威源仍属于其他模块。

## 使用规则

- 书架主 Tab 的页面规划、图片审计、Stitch 输入包优先引用本目录。
- 阅读入口引用副本只能说明书架到阅读的跳转边界，不得反向覆盖阅读模块规格。
- 候选图进入冻结前，必须先通过页面文字稿、图片内容、组件规格、交互规格和产品确认。
