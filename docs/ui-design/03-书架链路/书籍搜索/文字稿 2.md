# 书籍搜索

## 页面内容
### 页面目标

搜索书籍并展示搜索结果，支持从结果进入详情或加入书架。

### 页面区域

1. 顶部标题或控制区。
2. 主内容区，承载当前任务的核心列表、表单或阅读覆盖层。
3. 操作反馈区，展示 loading、empty、error、offline 和 permission。
4. 返回区，必须保留来源页上下文。

### 组件清单

搜索输入框、范围选择、分组选择器、历史记录、结果列表、加入书架按钮、阅读按钮。

### 核心文案

- `搜索书籍`
- `输入书名或作者`
- `全部`
- `书源`
- `本地`
- `加入书架`
- `阅读`

### 通用反馈

- `正在加载`
- `加载失败，请重试`
- `网络不可用，请稍后重试`
- `取消`
- `确认`

## 交互逻辑
### 主要操作

输入关键词、提交搜索、切换范围、点击结果进入详情、加入书架。

### 返回与反馈

- 系统返回回到来源页。
- 弹层优先关闭当前层。
- 操作失败保留用户上下文并提供重试。
- 提交中禁止重复点击。

### default

展示完整可操作内容。

### loading

使用骨架或局部加载，不清空已存在内容。

### empty

说明空原因，并给出下一步。

### error

展示失败原因和重试入口。

### offline

保留本地可用内容，阻断联网动作。

### permission

需要权限时说明用途并提供授权入口。

- 不新增未声明入口。
- 不引入账号、社区、广告、会员或推荐流。
- 不压缩触控区来适配小屏。
- 不在提交中允许重复点击。
