# 书籍搜索视觉规格

状态：正式视觉 token
适用范围：搜索首页、搜索结果、loading、empty、error

## 1. 继承关系

- 页面背景、顶部栏、卡片、封面、底部安全区继承 `../../02-主标签页/书架/08-书架首页组件视觉规格.md`。
- 搜索结果卡片继承 `Shelf.BookListItem.*`，但按钮和来源摘要由本规格补充。

## 6. 状态

- Loading：保留搜索框和筛选区，结果区骨架。
- Empty：显示查询词和建议操作。
- Error：显示重试，保留搜索框查询。
- Offline：提示网络不可用，但保留历史搜索。

## 7. Stitch 要求

Stitch 生成时必须区分搜索首页和搜索结果页。结果页是本页视觉基准；状态图只替换结果区。
