## 概念

书籍搜索按书名、作者或关键词查找书籍。搜索范围可以是本地书架、网络书源、全局范围或自定义分组。

它与内容搜索不同；内容搜索只在当前书籍内部搜索。

## 书籍搜索页

## 布局

- 顶部应用栏：返回、标题 `书籍搜索`、页面菜单
- 搜索输入框：`搜索书名、作者、关键词`
- 范围选择：`本地 / 网络 / 全局 / 分组`
- 仅选择 `分组` 时显示分组选择器
- 搜索历史卡片
- 单一主按钮：`搜索`

## 范围选择

| 范围 | 含义 |
| --- | --- |
| 本地 | 搜索书架或本地书籍。 |
| 网络 | 搜索已启用的网络书源。 |
| 全局 | 同时搜索本地和网络结果。 |
| 分组 | 搜索选中的自定义分组或导入来源分组。 |

## 分组选择器

仅在选择 `分组` 时显示。

示例：

- `玄幻书源`
- `本地导入`
- `起点导入`
- `晋江导入`
- `自定义规则`
- `RSS 书单`
- `测试书源`

按钮：

| 界面元素 | 功能 |
| --- | --- |
| `管理分组` | 打开分组管理。 |
| 搜索历史标签 | 将该历史查询填入搜索框。 |
| `清空` | 清空搜索历史。 |
| `搜索` | 使用当前范围执行搜索。 |
| 右上更多菜单 | 搜索设置、清空历史、管理分组、书源管理。 |

## 书籍搜索结果页

结果会将相同书名和作者的记录合并为一个书籍实体。不同来源通过 `换源` 选择，不展示为重复卡片。

## 布局

- 顶部应用栏：返回、标题 `搜索结果`、页面菜单
- 带当前查询词的搜索输入框
- 范围选择
- 摘要/筛选行
- 合并后的结果卡片

## 结果卡片内容

每张卡片包含：

- 封面
- 书名
- 作者
- 当前来源行和 `换源`
- 最新章节
- 可选阅读进度
- 三个固定操作按钮

## 固定按钮顺序

| 位置 | 按钮 | 可用状态 |
| --- | --- | --- |
| 1 | `查看信息` | 始终可用，打开书籍信息页。 |
| 2 | `加入书架 / 已加入` | 未加入时可用；已加入时禁用或弱化。 |
| 3 | `开始阅读` | 始终可用，打开阅读页。 |

状态通过按钮状态表达，不额外显示 `状态：` 文本。

## 交互矩阵

### 书籍搜索页

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开书籍搜索。 | 上一页，通常为 `书架`、`发现` 或阅读上下文 |
| 右上更多菜单 | 打开搜索级操作。 | 菜单：搜索设置、清空历史、管理分组、书源管理 |
| 搜索输入框 | 输入书名、作者或关键词。 | 当前页，键盘聚焦 |
| 清除图标 | 清空当前查询。 | 当前页，输入框为空 |
| 范围标签 `本地/网络/全局/分组` | 切换搜索范围。 | 当前页；选择 `分组` 时显示分组选择器 |
| `管理分组` | 管理自定义或导入来源分组。 | 分组管理页或底表 |
| 分组标签 | 选择自定义搜索分组。 | 当前页，选中分组更新 |
| 搜索历史标签 | 复用历史查询。 | 当前页，查询词填入输入框 |
| `清空` | 清空历史。 | 当前页，历史为空状态 |
| `搜索` | 使用当前范围执行搜索。 | `书籍搜索结果页` |

### 书籍搜索结果页

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 返回书籍搜索。 | `书籍搜索页`，保留查询词 |
| 右上更多菜单 | 打开结果级操作。 | 菜单：排序、筛选、刷新搜索、搜索日志 |
| 搜索输入框 | 编辑查询词。 | 当前页；提交后重新搜索 |
| 清除图标 | 清空查询。 | `书籍搜索页` 空输入状态 |
| 范围标签 | 切换结果范围。 | 当前页，结果刷新 |
| `筛选` | 细化结果列表。 | 筛选底表 |
| `换源` | 为合并后的书籍实体切换来源。 | 换源底表或页面 |
| 结果卡片主体 | 查看选中书籍。 | `书籍信息页` |
| `查看信息` | 查看选中书籍。 | `书籍信息页` |
| `加入书架` | 加入书架。 | 当前结果卡片更新为 `已加入`；书籍出现在 `书架` |
| `已加入` | 显示已加入书架状态。 | 不跳转 |
| `开始阅读` | 打开选中书籍。 | `阅读页 - 沉浸模式` |
