## 4. 页面生成约束

书籍搜索不是主 Tab 根页面，不显示底部主导航。页面结构从上到下为顶部栏、搜索框、范围筛选、结果摘要/筛选入口、搜索结果列表。

结果卡片包含封面、书名、作者、来源、最新章节、阅读进度或匹配信息，以及查看信息、加入书架/已加入、开始阅读等操作。点击结果卡进入书籍详情；配置修改不在搜索页完成。

首页、loading、empty、error 状态保留顶部搜索结构，只替换结果内容区。长查询保持搜索框单行处理，不压缩字号破坏可读性。

响应式以 `880 x 1788 px` 为结果页基准。小屏优先保留顶部栏、搜索框、范围筛选、结果摘要和至少一个结果卡片；短屏允许结果列表滚动。
