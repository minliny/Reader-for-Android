# 书籍搜索结果

文字稿状态：待产品确认

## 页面定位

书籍搜索结果页展示合并后的书籍实体。相同书名和作者视为同一本书，来源通过换源或默认书源配置处理。

本页允许展示书源信息，因为用户需要判断添加哪一个来源。

## 当前 source

- `docs/ui-design/03-书架链路/书籍搜索/图片/02-书籍搜索-结果页.png`

当前状态：候选可用，但页面级映射未闭合。

## 入口

- 书籍搜索首页提交查询。
- 搜索历史项提交查询后进入。

## 返回

- 顶部返回：返回书籍搜索首页，并保留查询词。
- 系统返回：返回书籍搜索首页，并保留查询词、范围和分组。
- 加入书架后返回：停留当前结果页，卡片状态更新为 `已加入`。
- 开始阅读后返回：回到阅读来源页面。

## 页面结构

顶部应用栏：

- 图标按钮：`返回`
- 标题：`搜索结果`
- 图标按钮：`更多`

主体：

- 带当前查询词的搜索输入框。
- 搜索范围选择。
- 摘要/筛选行。
- 合并后的结果卡片。

结果卡片：

- 封面。
- 书名。
- 作者。
- 当前来源行。
- `换源` 入口。
- 当前来源章节数量。
- 当前来源目录状态。
- 最新章节。
- 可选阅读进度。
- 固定操作按钮。

## 按钮与操作

- `返回`：返回书籍搜索首页。
- `更多`：打开搜索结果页菜单。
- `筛选`：打开结果筛选底表。
- `换源`：打开来源选择底表或页面。
- `查看信息`：进入书籍详情。
- `加入书架`：如果存在多个可用书源，先确认默认书源，再加入书架。
- `已加入`：禁用或作为状态按钮，不重复添加。
- `开始阅读`：进入阅读沉浸页。

按钮状态：

- 加入中：`加入书架` 显示加载态。
- 已加入：按钮显示 `已加入`。
- 没有可用来源：`加入书架` 和 `开始阅读` 禁用。
- 来源检测中：`换源` 显示加载态或不可用。

## 状态文案

loading：

- 使用 `09-搜索状态.md` 的搜索 loading。

empty：

- 使用 `09-搜索状态.md` 的搜索 empty。

error：

- 使用 `09-搜索状态.md` 的搜索 error。

permission：

- 不适用。

offline：

- 网络不可用但本地有结果时，提示：`当前网络不可用，仅显示本地结果`

partial success：

- `部分书源搜索失败，已显示可用结果`

invalid source：

- `当前来源不可用，请更换来源`

## 错误与空状态

- 不得清空已经成功返回的结果。
- 部分书源失败时保留可用结果。
- 加入失败时卡片不变，提示失败原因。
- 默认书源不可用时要求重新选择。

## 书源信息边界

必须展示或可展开查看：

- 书源名称。
- 章节数量。
- 目录是否可用。
- 最新章节。
- 更新时间或检测时间。
- 当前默认书源标记。

添加至书架时必须确定默认书源。

## 不允许

- 不把不同书源拆成重复书籍卡片。
- 不在书架首页继承展示书源信息。
- 不把搜索失败和搜索无结果混为一类。
