# 书籍搜索首页

文字稿状态：待产品确认

## 页面定位

书籍搜索首页用于按书名、作者或关键词搜索书籍。它与书内内容搜索不同。

## 当前 source

- `docs/ui-design/03-书架链路/书籍搜索/图片/01-书籍搜索-首页.png`

当前状态：候选可用，但页面级映射未闭合。

## 入口

- 书架顶部 `搜索` 图标。
- 书架空状态主按钮 `搜索书籍`。
- 其他页面的全局书籍搜索入口。

## 返回

- 顶部返回：返回来源页面，通常为书架首页。
- 系统返回：返回来源页面。
- 从搜索结果返回：保留查询词、范围选择、分组选择和搜索历史。

## 页面结构

顶部应用栏：

- 图标按钮：`返回`
- 标题：`书籍搜索`
- 图标按钮：`更多`

主体：

- 搜索输入框。
- 搜索范围选择。
- 分组选择器。
- 搜索历史卡片。
- 底部主按钮。

## 主要文案

- 页面标题：`书籍搜索`
- 输入框占位：`搜索书名、作者、关键词`
- 范围项：`本地`
- 范围项：`网络`
- 范围项：`全局`
- 范围项：`分组`
- 分区标题：`选择分组`
- 入口：`管理分组`
- 分区标题：`搜索历史`
- 文本按钮：`清空`
- 主按钮：`搜索`

范围说明：

- `本地`：搜索书架或本地书籍。
- `网络`：搜索已启用网络书源。
- `全局`：同时搜索本地和网络。
- `分组`：搜索选中的自定义分组或导入来源分组。

## 按钮与操作

- `返回`：返回来源页面。
- `更多`：打开搜索页菜单。
- `清除输入`：清空搜索框。
- `本地` / `网络` / `全局` / `分组`：切换搜索范围。
- `管理分组`：进入分组管理。
- 搜索历史项：填入搜索框。
- `清空`：清空搜索历史。
- `搜索`：提交查询并进入书籍搜索结果页。

按钮状态：

- 查询词为空时，`搜索` 禁用。
- 搜索提交中，`搜索` 显示加载态。
- 搜索历史为空时，不展示历史卡片或展示空提示。

## 状态文案

loading：

- 提交搜索时按钮显示：`搜索中`

empty：

- 搜索历史为空：`暂无搜索历史`

error：

- 历史加载失败：`搜索历史加载失败`

permission：

- 不适用。

offline：

- 选择网络或全局时提示：`当前网络不可用`

partial success：

- 不适用。

invalid source：

- 网络书源不可用时，在搜索结果页处理。

## 错误与空状态

- 查询词为空不进入结果页。
- 清空历史需要二次确认或即时撤销，需产品确认。
- 网络不可用时，本地搜索仍可继续。

## 不允许

- 不与书内内容搜索混用。
- 不在搜索首页直接展示网络结果列表。
- 不在书架 P0 中展开完整书源管理。
