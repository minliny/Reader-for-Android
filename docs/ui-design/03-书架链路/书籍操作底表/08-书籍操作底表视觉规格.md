# 书籍操作底表视觉规格

状态：正式组件 token
适用范围：书架页单本书操作底表、删除确认弹窗

## 1. 继承关系

- 页面遮罩、底表和弹窗继承 `../../01-全局规范/02-应用外壳组件视觉规格.md` 的基础色彩与触控规则。
- 书籍摘要中的封面、标题、作者、章节层级继承 `../../02-主标签页/书架/08-书架首页组件视觉规格.md` 的 `Shelf.Cover.*`、`Shelf.BookListItem.*`。
- 本规格只定义底表组件，不定义 App Shell、不定义书架列表、不定义书籍详情页。

## 2. 底表容器

| Token | 值 | 说明 |
| --- | --- | --- |
| `BookActionSheet.ScrimColor` | `rgba(36,32,26,0.32)` | 页面遮罩 |
| `BookActionSheet.Background` | `#FFFDF8` | 底表背景 |
| `BookActionSheet.BorderColor` | `#E7DED0` | 顶部低对比边线 |
| `BookActionSheet.BorderWidth` | `1dp` | 顶部边线 |
| `BookActionSheet.RadiusTop` | `20dp` | 顶部圆角 |
| `BookActionSheet.PaddingX` | `20dp` | 横向内边距 |
| `BookActionSheet.PaddingTop` | `12dp` | 顶部内边距 |
| `BookActionSheet.PaddingBottom` | `system inset + 16dp` | 避让手势区 |
| `BookActionSheet.MaxHeightRatio` | `0.56` | 不超过屏幕高度 56% |

底表不得超过半屏太多；默认态只展示摘要、两项操作和安全区。

## 3. 书籍摘要

| Token | 值 | 说明 |
| --- | --- | --- |
| `BookActionSheet.Summary.MinHeight` | `88dp` | 摘要区最小高度 |
| `BookActionSheet.Cover.Width` | `56dp` | 摘要封面宽 |
| `BookActionSheet.Cover.Height` | `84dp` | 摘要封面高，保持 2:3 |
| `BookActionSheet.Cover.Radius` | `8dp` | 封面圆角 |
| `BookActionSheet.Summary.Gap` | `12dp` | 封面与文本间距 |
| `BookActionSheet.Title.Size` | `17sp` | 书名 |
| `BookActionSheet.Title.Weight` | `600` | 书名字重 |
| `BookActionSheet.Meta.Size` | `13sp` | 作者/章节 |
| `BookActionSheet.Meta.Color` | `#5F574F` | 次级文字 |

书名最多两行；作者和章节最多一行，尾部省略。

## 4. 操作行

| Token | 值 | 说明 |
| --- | --- | --- |
| `BookActionSheet.ActionRow.Height` | `52dp` | 操作行高度 |
| `BookActionSheet.ActionRow.Radius` | `12dp` | 按压态圆角 |
| `BookActionSheet.ActionRow.IconSize` | `22dp` | 图标尺寸 |
| `BookActionSheet.ActionRow.IconStroke` | `2dp` | 图标线宽 |
| `BookActionSheet.ActionRow.TextSize` | `16sp` | 操作文案 |
| `BookActionSheet.DividerColor` | `#E7DED0` | 分割线 |

操作项固定为 `修改` 和 `删除`。不得加入换源、缓存、分组、导出、刷新目录、继续阅读等操作。

## 5. 删除确认弹窗

| Token | 值 |
| --- | --- |
| `BookActionDialog.WidthRule` | `min(screen - 48dp, 328dp)` |
| `BookActionDialog.Background` | `#FFFDF8` |
| `BookActionDialog.Radius` | `18dp` |
| `BookActionDialog.Padding` | `20dp` |
| `BookActionDialog.TitleSize` | `18sp` |
| `BookActionDialog.BodySize` | `14sp` |
| `BookActionDialog.ButtonHeight` | `44dp` |

删除确认说明必须明确：只从书架移除，不删除本地文件或网络来源。

## 6. Stitch 要求

- 这是组件母版，不是独立全屏页面。
- 只允许 `修改` 和 `删除` 两个操作。
- `修改` 不在底表内展开配置，必须进入书籍详情页。
- 图片如果出现多余操作，以本规格为准删除。
