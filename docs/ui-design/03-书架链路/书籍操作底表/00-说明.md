# 书籍操作底表

状态：正式 UI 设计稿已建立
主入口：`10-正式UI设计稿.md`

## 页面定位

书籍操作底表用于对单本已加入书架的书籍执行书架级入口操作。它只提供 `修改` 和 `删除` 两项；所有配置、来源、缓存、分组和目录相关修改都进入书籍详情页处理。

## 入口

- 书架条目更多按钮。
- 长按书籍条目。

## 返回

- 点击遮罩、下滑底表、点击系统返回：关闭底表，回到原书架状态。
- 操作成功后：关闭底表或停留在底表显示反馈，按操作类型决定。
- 操作取消后：关闭确认弹窗，回到底表。
- 关闭后保留当前分组、排序、展示模式和滚动位置。

## 页面结构

顶部摘要：

- 封面。
- 书名。
- 作者。
- 当前章节或阅读进度。

操作列表：

- `修改`
- `删除`

确认弹窗：

- 删除确认。

## 主要文案

- 底表标题：书名。
- 副标题：作者和当前章节。
- 操作项：`修改`
- 危险操作：`删除`
- 删除说明：`从书架移除，不会删除本地文件或网络来源`
- 确认标题：`删除书架记录？`
- 确认说明：`确认后书籍将从书架移除，不会删除本地文件或网络来源。`
- 主按钮：`删除`
- 次按钮：`取消`
- 成功提示：`已从书架移除`
- 失败提示：`删除失败，请重试`

## 按钮与操作

- `修改`：进入书籍详情页。分组、来源、缓存、目录刷新、导出等配置和管理操作都在详情页内处理。
- `删除`：打开二次确认；确认后执行移除书架。
- `删除` 确认按钮：危险按钮，确认从书架移除。
- `取消`：关闭确认弹窗。

按钮状态：

- 书籍详情不可打开时：`修改` 禁用，并显示原因。
- 删除提交中：确认弹窗内 `删除` 显示加载态，防止重复点击。

## 状态文案

loading：

- 底表打开时如果书籍元信息未加载，摘要区域显示骨架。

empty：

- 不适用。

error：

- 删除失败：`删除失败，请重试`
- 详情打开失败：`打开书籍详情失败，请重试`

permission：

- 不适用。需要权限的配置项在书籍详情页内处理。

offline：

- 网络不可用不影响打开底表；需要网络的配置项在书籍详情页内提示。

partial success：

- 不适用。

invalid source：

- 来源失效不在底表内展开，进入书籍详情页处理。

## 错误与空状态

- 操作失败时不关闭底表，保留用户上下文。
- 删除失败时书籍仍保留在书架。

## 不允许

- 不在底表中切换书源。
- 不在底表中展示复杂书源规则。
- 不在底表中展示继续阅读、刷新目录、缓存书籍、移动分组、导出等配置项。
- 不出现账号、分享社区、推荐算法入口。
- 不把 `删除` 解释为删除本地文件或网络来源。

## 完成条件

- 默认底表只展示 `修改` 和 `删除`。
- `修改` 进入书籍详情页。
- `删除` 二次确认后执行移除书架。
- 图片不混入删除确认态；确认态如需冻结，另补独立状态图。
