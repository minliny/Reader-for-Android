# 本地书导入禁止项

状态：`FORMAL_DRAFT_READY`

## 1. 功能边界

- 不扫描整个存储。
- 不默认申请 `MANAGE_EXTERNAL_STORAGE`。
- 不要求登录、会员、同步或云端账号。
- 不把本地导入放入书架 P0 闭环。
- 不在取消文件选择后展示错误页。

## 2. 数据边界

- 不静默丢弃失败文件。
- 不把重复文件当作失败。
- 不在全部失败时自动返回书架。
- 不隐藏失败原因。

## 3. 视觉边界

- 无独立 UI 图前不得标记为 `STITCH_READY`。
- 不使用旧图、候选图或其他页面截图替代本页母版。
- 不生成与书架链路不一致的顶部栏、卡片、按钮和列表行样式。

## 4. 交互边界

- 解析中不得允许重复提交。
- 重新选择不得丢失上一轮失败原因，除非新一轮选择完成。
- 关键操作不得只依赖隐藏手势。
