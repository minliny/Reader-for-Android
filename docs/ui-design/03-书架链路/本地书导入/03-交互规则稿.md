# 本地书导入交互规则稿

状态：`FORMAL_DRAFT_READY`

## 1. 操作流

```text
进入本地书导入
  -> 点击选择文件
  -> Android 系统文件选择器
    -> 取消：返回进入前书架状态
    -> 选择文件：进入解析中
      -> 全部成功：导入完成
      -> 部分失败：展示部分失败结果
      -> 全部失败：展示失败结果
```

## 2. 交互定义

| 操作 | 结果 |
| --- | --- |
| 返回 | 返回进入前页面并保留上下文 |
| 选择文件 | 调起 Android 系统文件选择器 |
| 取消系统选择器 | 返回进入前书架状态，不显示错误 |
| 重新选择 | 再次调起系统文件选择器 |
| 完成 | 返回书架首页 |
| 返回书架 | 放弃当前失败结果并返回书架首页 |
| 点击失败行 | 展开或查看失败原因 |

## 3. 上下文保留

从书架进入时，返回后必须保留：

- 当前分组。
- 当前排序和筛选。
- 书架滚动位置。
- 已有书籍列表状态。

## 4. 提交中规则

- 解析中禁止重复点击 `选择文件` 或 `重新选择`。
- 解析中允许系统返回，但需要保留已经完成的解析结果或给出中断确认。
- 失败后重新选择不得清空上一轮结果，除非用户完成新一轮选择。

## 5. 权限边界

本页不请求全盘扫描权限。文件访问只来自 Android 系统文件选择器返回的用户授权 URI。
