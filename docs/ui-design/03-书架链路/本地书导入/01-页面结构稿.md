# 本地书导入页面结构稿

状态：`FORMAL_DRAFT_READY`

## 1. 页面目标

让用户明确本地导入的授权方式，选择文件后查看解析进度和逐项导入结果。

## 2. 页面树

```text
LocalBookImportPage
  StatusBar
  TopBar
    BackButton
    Title("导入本地书")
  ContentScroll
    ImportIntroCard
      Title("选择本地书籍")
      Body
      SupportedFormats
      PrimaryAction("选择文件")
    ImportProgressCard(optional)
      ProgressIndicator
      CurrentFile
      ParsedCount
    ImportResultSummary(optional)
      SuccessCount
      SkippedCount
      FailedCount
    ImportResultList(optional)
      ImportResultRow*
    ErrorGuidance(optional)
  BottomActionBar(optional)
    SecondaryAction
    PrimaryAction
  SystemFilePicker(external)
```

## 3. 默认结构

默认页包含：

1. 顶部栏：返回、标题 `导入本地书`。
2. 导入说明卡：说明系统文件选择器授权方式。
3. 支持格式说明：`TXT / EPUB` 等可支持格式。
4. 主按钮：`选择文件`。
5. 辅助说明：不会扫描整个存储。

## 4. 结果结构

导入结果页包含：

1. 结果摘要：成功、重复跳过、失败数量。
2. 文件结果列表：文件名、状态、失败原因。
3. 操作区：`重新选择`、`完成` 或 `返回书架`。

## 5. 自适应规则

- 手机竖屏以 `390x884dp` 为逻辑基准。
- 宽度 `<360dp` 时压缩外边距，不压缩按钮触控区。
- 结果列表过长时内容区滚动，顶部栏和底部操作保持可达。
- 长文件名单行省略，结果原因允许换行。

## 6. 图片状态

当前无独立 UI 图。页面结构可用于补图和生成提示，但不得作为最终视觉复刻依据。
