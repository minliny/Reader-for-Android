# 本地书导入

文字稿状态：`FORMAL_DRAFT_READY`
视觉状态：`PAGE_VISUAL_CANDIDATE`
页面等级：`P1 / BOOKSHELF_FLOW`

## 1. 页面定位

本地书导入用于把用户通过 Android 系统文件选择器授权选择的本地书籍文件加入书架。

本页是书架链路 P1 页面，不阻塞书架 P0 主流程。当前阶段先冻结产品结构、状态、文案和交互边界；当前已有本页视觉候选图，确认前不得作为正式页面视觉基准或其他页面视觉来源。

## 2. 入口

| 入口 | 说明 |
| --- | --- |
| 书架空状态次按钮 `导入本地书` | 全书架为空时引导用户导入 |
| 书架右上更多菜单 | 非空书架中补充导入入口 |
| 设置/书架与搜索相关入口 | P1 后续可接入，不作为当前主入口 |

## 3. 页面范围

本页覆盖以下状态：

1. 导入说明页。
2. 调起 Android 系统文件选择器。
3. 文件解析中。
4. 导入完成。
5. 部分文件导入失败。
6. 全部文件导入失败。
7. 取消选择。

## 4. 返回结果

| 场景 | 返回规则 |
| --- | --- |
| 导入完成 | 返回书架首页，并定位到导入结果所在分组或 `未分组` |
| 取消文件选择 | 返回进入前的书架状态，不展示错误 |
| 部分失败 | 停留在导入结果页，用户点击 `完成` 后返回书架 |
| 全部失败 | 停留在失败结果页，用户可 `重新选择` 或 `返回书架` |

## 5. 关键原则

- 只通过系统文件选择器获取用户授权文件。
- 不扫描整个存储。
- 不默认申请 `MANAGE_EXTERNAL_STORAGE`。
- 不要求登录、同步或云端账号。
- 不静默丢弃失败文件。
- 导入结果必须逐项说明成功、重复跳过、格式不支持和解析失败。
