# 本地书导入

文字稿状态：草稿

## 页面定位

本地书导入用于把 Android 文件选择器授权的本地文件加入书架。

当前优先级：P1，不阻塞书架 P0 冻结。

## 当前 source

无专用 source。已从书架 P0 移出，归入 P1。

## 入口

- 书架空状态次按钮 `导入本地书`。
- 书架右上 `更多` 菜单。

## 返回

- 导入完成：返回书架首页，并进入本地导入结果对应的分组或 `未分组`。
- 取消文件选择：返回进入前的书架状态。
- 部分失败：停留在导入结果页，用户点击 `完成` 后返回书架。
- 全部失败：停留在失败结果页，用户可 `重新选择` 或 `返回书架`。

## 页面结构

P1 阶段需要覆盖：

- 权限说明。
- 系统文件选择器。
- 解析中。
- 导入成功。
- 部分失败。
- 全部失败。
- 取消。

## 主要文案

权限说明：

- 标题：`选择本地书籍`
- 说明：`通过系统文件选择器导入文件，不会扫描整个存储。`
- 主按钮：`选择文件`

解析中：

- 标题：`正在导入`
- 说明：`正在解析文件，请稍候`

成功：

- 标题：`导入完成`
- 按钮：`完成`

部分失败：

- 标题：`部分文件导入失败`
- 说明：`可以查看失败原因后重新选择`
- 主按钮：`重新选择`
- 次按钮：`完成`

全部失败：

- 标题：`导入失败`
- 主按钮：`重新选择`
- 次按钮：`返回书架`

结果字段：

- `成功`
- `重复跳过`
- `格式不支持`
- `解析失败`

## 按钮与操作

- `选择文件`：打开 Android 系统文件选择器。
- `重新选择`：重新打开文件选择器。
- `完成`：返回书架首页。
- `返回书架`：放弃当前结果并返回书架首页。
- 文件结果行：展示文件名、状态和失败原因。

按钮状态：

- 解析中禁用返回以外的重复导入操作。
- 文件选择取消后不展示错误。

## 状态文案

loading：

- `正在导入`

empty：

- 文件选择器未选中文件：返回原书架状态，不显示空状态。

error：

- `导入失败`
- `文件格式不支持`
- `文件解析失败`

permission：

- `需要通过系统文件选择器选择文件`

offline：

- 不适用。

partial success：

- `部分文件导入失败`

invalid source：

- 不适用。

## 错误与空状态

- 不得静默丢弃失败文件。
- 结果页必须逐项说明成功、重复跳过、格式不支持和解析失败。
- 全部失败时不得自动返回书架。

## 不允许

- 不默认扫描整个存储。
- 不在 P0 书架冻结中要求完成本流程。
- 不要求用户登录或开启同步。
