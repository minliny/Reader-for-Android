## 4. 页面生成约束

书籍详情是单本书的配置和阅读入口页，不显示主 Tab 底部导航。书架页底表中的 `修改` 进入本页，所有书籍配置在本页完成。

页面结构从上到下为顶部栏、书籍头部信息、章节预览、简介、书源与更新。主操作优先 `开始阅读`，加入书架/已加入为次级状态，配置入口不得回到底表扩展。

loading 和来源失效状态只替换对应内容区或状态卡片，保留顶部栏和页面主结构。来源失效状态提供修复或换源入口，不把错误页重画成独立全屏营销页。

响应式以 `834 x 1886 px` 为基准。小屏优先保留顶部栏、封面信息、阅读主操作、章节预览首行；短屏允许内容区纵向滚动。
