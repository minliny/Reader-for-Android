# 书籍详情

## 页面内容
### 页面目标

展示书籍信息、目录预览、简介和书架状态，并承接所有书籍配置修改。

### 页面区域

1. 顶部标题栏。
2. 书籍头部信息：封面、书名、作者、当前来源、`换源`、最新章节和阅读主操作。
3. 章节预览：位于简介上方，展示当前阅读章节和不少于 20 条章节预览数据。
4. 简介：位于完整章节预览之后，长内容折叠。
5. 操作反馈区，展示 loading、empty、error、offline 和 permission。
6. 返回区，必须保留来源页上下文。

### 层级顺序

章节预览与简介是同一纵向滚动内容的一部分。默认首屏可只露出简介顶部，但简介必须排在完整章节预览之后。

### 组件清单

封面、标题作者、来源状态、阅读按钮、加入/已在书架、目录预览、章节预览长列表、简介、配置入口、换源底表。

### 组件约束

- `BookDetail.ChapterPreviewCard` 位于 `IntroCard` 之前。
- `BookDetail.ChapterPreviewList` 的 `查看目录` 仍是完整目录页入口，不能替代长列表预览本身。
- `BookDetail.SourceChangeSheet` 是当前详情页上的底表覆盖层，不进入阅读控制层流程页。

- `确认`

## 交互逻辑
### 主要操作

开始阅读、加入书架、修改配置、查看目录、滚动章节预览、展开简介、换源、返回来源页。

### 返回与反馈

- 系统返回回到来源页。
- 弹层优先关闭当前层。
- 操作失败保留用户上下文并提供重试。
- 提交中禁止重复点击。

### 详情页内交互

- 点击头部来源行的 `换源` 后，在书籍详情当前页面打开 `BookDetail.SourceChangeSheet` 底表。
- 详情页 `换源` 不跳转到阅读控制层，不展示阅读页换源流程顺序图。
- 换源底表关闭后回到书籍详情当前滚动位置。
- 章节预览支持纵向滚动查看不少于 20 条章节；点击章节行进入对应章节阅读。

### default

展示完整可操作内容。章节预览在简介上方，预览数据不少于 20 条，页面可纵向滚动查看完整预览与简介。

### loading

使用骨架或局部加载，不清空已存在内容。

### empty

说明空原因，并给出下一步。

### error

展示失败原因和重试入口。

### source_sheet

点击头部 `换源` 后展示详情页内底表。底表复用换源来源列表 UI 语言，背景是书籍详情页遮罩，不切换到阅读控制层。

### offline

保留本地可用内容，阻断联网动作。

### permission

需要权限时说明用途并提供授权入口。

- 不新增未声明入口。
- 不引入账号、社区、广告、会员或推荐流。
- 不压缩触控区来适配小屏。
- 不在提交中允许重复点击。
- 不把简介放到章节预览上方。
- 不把章节预览缩成仅 3 到 4 条固定列表。
- 不让书籍详情的 `换源` 直接跳转到阅读控制层或阅读页换源流程顺序图。
