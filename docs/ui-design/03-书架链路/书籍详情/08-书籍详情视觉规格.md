# 书籍详情视觉规格

状态：正式视觉 token
适用范围：书籍详情默认页、loading、来源失效 error

## 1. 继承关系

- 页面背景、顶部栏、卡片、封面、进度条继承 `../../02-主标签页/书架/08-书架首页组件视觉规格.md`。
- 本页定义书籍详情头部、阅读主操作、信息组、目录预览和简介区域。

## 5. 状态

- Loading：保持顶部栏和页面骨架。
- 来源失效：在来源状态卡中提示，不遮挡主操作。
- 无简介：显示低对比空说明，不删除简介区域。
- 已加入书架：主操作仍是阅读，书架状态作为次级信息。

## 6. Stitch 要求

Stitch 生成时不得把书籍详情做成营销详情页。它是工具型阅读详情页，重点是阅读、目录、简介和配置入口。
