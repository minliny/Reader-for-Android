## 页面定位

书籍信息页描述一个合并后的书籍实体。相同书名和作者视为同一本书，书源是可切换属性。

## 布局

- 顶部应用栏：返回、标题 `书籍信息`、页面菜单
- 头部卡片：封面与书籍元信息
- 两个主操作
- 目录预览
- 简介
- 更新时间

## 头部内容

- 封面
- 书名
- 作者
- 当前来源：`当前来源：起点中文网  换源`
- 最新章节
- 阅读进度

换源只出现在顶部来源行，不单独做来源卡片。

## 按钮

| 界面元素 | 功能 |
| --- | --- |
| `换源` | 打开当前书籍的换源入口。 |
| `已加入 / 加入书架` | 显示书架状态；已加入时为禁用或弱化状态。 |
| `开始阅读` | 打开阅读页。 |
| 右上更多菜单 | 刷新书籍信息、刷新目录、复制链接、打开来源页、从书架移除。 |

不需要 `查看信息` 按钮，因为当前页本身就是书籍信息页。

## 目录预览

`目录预览` 卡片展示：

- 当前阅读章节
- 最新章节
- 上下文操作 `全部目录`

操作：

| 界面元素 | 功能 |
| --- | --- |
| `全部目录` | 打开完整目录页。 |
| 章节行 | 跳转到该章节。 |

## 简介

简介卡片展示折叠状态的书籍简介，并提供 `展开` 查看全文。

## 交互矩阵

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开书籍信息页。 | 上一页，通常为 `书籍搜索结果页` 或 `书架` |
| 右上更多菜单 | 打开书籍级低频操作。 | 菜单：刷新书籍信息、刷新目录、复制书籍链接、打开来源页、从书架移除 |
| 来源行中的 `换源` | 切换当前来源。 | 换源底表或换源页 |
| `已加入` | 显示已加入书架状态。 | 不跳转 |
| `加入书架` | 将书籍加入书架。 | 当前页，按钮变为 `已加入` |
| `开始阅读` | 打开书籍。 | `阅读页 - 沉浸模式` |
| `全部目录` | 打开完整章节列表。 | `目录全览页` |
| 目录预览章节行 | 跳转到选中章节。 | `阅读页 - 沉浸模式`，定位到选中章节 |
| 简介中的 `展开` | 展开简介全文。 | 当前页，简介区域原地展开 |
