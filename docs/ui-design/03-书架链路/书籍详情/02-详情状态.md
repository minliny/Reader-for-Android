# 书籍详情状态

文字稿状态：待产品确认

## 页面定位

书籍详情状态属于书籍详情页，不是全局状态页。状态页必须保留书籍详情上下文。

## 入口

- 进入书籍详情时加载中。
- 刷新书籍信息时加载中。
- 当前来源失效时进入来源失效状态。
- 网络失败、书籍不存在或目录失败时进入对应状态。

## 返回

- 顶部返回：返回来源页面。
- 系统返回：返回来源页面。
- 重试成功：回到书籍详情正常态。
- 更换来源成功：回到书籍详情正常态并更新默认书源。

## 页面结构

详情 loading：

- 顶部应用栏。
- 页面布局骨架。
- 封面骨架。
- 标题和作者骨架。
- 目录预览骨架。
- 简介骨架。

来源失效：

- 保留已有书籍信息。
- 来源行显示失效状态。
- 展示更换来源操作。

网络失败：

- 有缓存时保留缓存内容。
- 无缓存时使用详情页内错误区域。

书籍不存在：

- 保留顶部栏。
- 展示明确错误区域。

## 主要文案

loading：

- 标题：`正在加载书籍信息`

来源失效：

- 标题：`当前来源不可用`
- 说明：`可以更换来源后重新加载内容`
- 主按钮：`更换来源`
- 次按钮：`重试`

网络失败：

- 标题：`网络不可用`
- 说明：`请检查网络后重试`
- 主按钮：`重试`
- 次按钮：`返回`

书籍不存在：

- 标题：`书籍信息不存在`
- 说明：`该书籍可能已被来源删除或暂时不可访问`
- 主按钮：`更换来源`
- 次按钮：`返回`

## 按钮与操作

- `更换来源`：进入来源选择。
- `重试`：重新加载当前来源。
- `返回`：返回来源页面。
- 来源列表项：选择新来源。
- `设为默认并重载`：确认新来源并重新加载详情或阅读内容。

按钮状态：

- 重试中：`重试` 显示加载态。
- 来源检测中：来源列表显示检测状态。
- 无可用来源时：`设为默认并重载` 禁用。

## 状态文案

loading：

- `正在加载书籍信息`

empty：

- 目录为空：`暂无目录`

error：

- `网络不可用`
- `书籍信息不存在`

permission：

- 不适用。

offline：

- `当前网络不可用，显示缓存信息`

partial success：

- `书籍信息已加载，目录更新失败`

invalid source：

- `当前来源不可用`

## 错误与空状态

- 来源失效不等于书籍不存在。
- 网络失败不等于来源失效。
- 有缓存时优先保留缓存内容。
- 无缓存时才展示详情页内错误区域。
- 不使用全屏转圈，不隐藏顶部栏。

## 书源信息边界

来源失效时允许展示书源信息，用于让用户重新选择可用来源。

换源列表需要展示：

- 书源名称。
- 章节数量。
- 目录状态。
- 最新章节。
- 更新时间或检测时间。
- 当前默认书源标记。

## 不允许

- 不跳到全局错误页。
- 不把来源失效写成书籍不存在。
- 不在 loading 状态隐藏页面结构。
