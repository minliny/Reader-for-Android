# 书籍详情

文字稿状态：待产品确认

## 页面定位

书籍详情页描述一个合并后的书籍实体。书源是可切换属性，不是重复书籍卡片。

书籍详情页可以展示书源信息，但仅用于添加、编辑或修复。

## 入口

- 书架书名或元信息区域。
- 搜索结果卡片。
- 搜索结果 `查看信息`。
- 最近更新条目。

## 返回

- 顶部返回：返回来源页面，通常为书架或搜索结果。
- 系统返回：返回来源页面。
- 加入书架成功后：停留详情页，按钮更新为 `已加入`。
- 开始阅读后返回：回到阅读来源页面。

- 图标按钮：`返回`

## 按钮与操作

- `返回`：返回来源页面。
- `更多`：打开刷新书籍信息、刷新目录、复制链接、打开来源页、从书架移除。
- `换源`：打开来源选择。
- `加入书架`：如果还没有默认书源，先配置默认书源。
- `已加入`：状态按钮，避免重复添加。
- `开始阅读`：进入阅读沉浸页。
- `查看目录`：进入完整目录页。
- 章节行：进入阅读页。
- `展开`：原地展开简介。

按钮状态：

- 加入中：`加入书架` 显示加载态。
- 目录加载中：`查看目录` 可用但目录区域显示骨架。
- 来源失效：`开始阅读` 可禁用，并显示 `更换来源`。

## 状态文案

loading：

- 使用 `11-书籍详情状态.md` 的详情 loading。

empty：

- 目录为空：`暂无目录`

error：

- 使用 `11-书籍详情状态.md` 的详情 error。

permission：

- 不适用。

offline：

- `当前网络不可用，显示缓存信息`

partial success：

- `书籍信息已更新，目录更新失败`

invalid source：

- `当前来源不可用，请更换来源`

## 错误与空状态

- 来源失效不等于书籍不存在。
- 目录加载失败时保留头部书籍信息。
- 简介缺失时展示 `暂无简介`。

## 书源信息边界

允许展示：

- 添加前查看不同书源的章节数量、目录和最新章节。
- 添加至书架时配置默认书源。
- 已加入后编辑默认书源。
- 阅读内容异常后从详情页或阅读页触发换源。

必须包含：

- 书源名称。
- 章节数量。
- 目录状态。
- 最新章节。
- 更新时间或检测时间。
- 当前默认书源标记。

## 不允许

- 不把不同书源显示成重复书籍详情页。
- 不在常态书架首页复制本页的书源信息。
- 不把来源失效写成书籍不存在。
