# 分组管理视觉规格

状态：正式视觉 token
适用范围：分组管理列表、新建/重命名输入、删除确认

## 1. 继承关系

- 页面背景、顶部栏、卡片、列表行继承 `../../02-主标签页/书架/08-书架首页组件视觉规格.md`。
- 输入弹窗和删除确认继承全局弹窗与底表按钮规则。

| Token | 值 |

## 3. 分组行

| Token | 值 |
| --- | --- |
| `ShelfGroup.Row.Height` | `56dp` |
| `ShelfGroup.Row.Background` | `#FFFDF8` |
| `ShelfGroup.Row.BorderColor` | `#E7DED0` |

| `ShelfGroup.Count.Size` | `13sp` |
| `ShelfGroup.Icon.Size` | `22dp` |

`全部` 不作为可管理行。`未分组` 可显示数量，但不可重命名和删除。

## 4. 输入与确认

- 分组名输入框高度 `44dp`。
- 保存按钮高度 `44dp`。
- 分组名为空或重复时禁用保存。
- 删除确认必须说明：删除分组不会删除书籍。
