## 4. 页面生成约束

分组管理用于新增、重命名、排序和删除用户自定义分组，不删除书籍本身。页面不显示主 Tab 底部导航。

`全部` 是系统聚合入口，不参与重命名、删除或排序。`未分组` 可显示数量，但不可删除。自定义分组支持重命名、删除和拖动排序；删除分组时，组内书籍移动到 `未分组`。

新建分组弹窗是当前页面状态，不是独立页面。弹窗保留遮罩、标题、输入框、说明文字、取消和保存按钮。

响应式以 `833 x 1888 px` 为基准。小屏优先保留顶部栏、至少三条分组行和弹窗主要操作；短屏允许列表滚动，弹窗内容不遮挡操作按钮。
