# 分组管理正式 UI 设计稿

状态：正式稿
等级：P0 / PAGE_VISUAL_BASELINE
适用页面：书架分组管理
视觉参考：`图片/01-书架分组管理.png`

## 1. 设计目标

分组管理用于新增、重命名、排序和删除用户自定义分组，不删除书籍本身。

## 2. 结构

```text
ShelfGroupManagementPage
  TopBar
    BackButton
    Title("分组管理")
    TextButton("新建")
  GroupList
    ReadonlyRow("未分组")
    CustomGroupRow*
  CreateOrRenameDialog(optional)
  DeleteConfirmDialog(optional)
```

## 3. 规则

- `全部` 是系统聚合入口，不参与重命名、删除或排序。
- `未分组` 可显示数量，但不可删除。
- 自定义分组支持重命名、删除、拖动排序。
- 删除分组时，组内书籍移动到 `未分组`。

## 4. 状态

| 状态 | 要求 |
| --- | --- |
| 默认 | 展示分组列表和新建入口 |
| 空自定义分组 | 显示空说明和新建分组 |
| Loading | 列表骨架 |
| Error | 保留顶部栏，提供重试 |
| 删除失败 | 列表保持原样 |

## 5. 明确排除

- 不删除书籍本身。
- 不允许删除 `全部`。
- 不把 `全部` 当作普通分组。
- 不展示书源规则。
