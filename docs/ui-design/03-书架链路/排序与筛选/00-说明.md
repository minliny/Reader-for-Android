# 书架排序与筛选

状态：正式 UI 设计稿已建立
主入口：`10-正式UI设计稿.md`

## 页面定位

书架排序与筛选用于改变当前书架列表的展示顺序和可见范围。它作用于当前书架上下文，不改变书籍本身。

## 当前 source

候选图：`docs/ui-design/03-书架链路/排序与筛选/图片/01-书架排序与筛选.png`

图片状态：组件参考；如与正式稿冲突，以 `10-正式UI设计稿.md` 和 `08-排序与筛选视觉规格.md` 为准。

## 入口

- 书架 `我的书架` 标题右侧 `排序` 图标。
- 书架右上 `更多` 菜单中的排序入口。

## 返回

- 点击 `应用`：关闭底表，返回书架首页并刷新列表。
- 点击 `重置`：恢复默认选项，但不自动关闭。
- 点击遮罩、系统返回或下滑：放弃未应用改动，返回书架首页。
- 返回后保留当前分组、展示模式和滚动位置；应用排序后滚动位置回到列表顶部。

## 按钮与操作

- 排序项：单选。
- 顺序项：单选。
- 筛选项：可多选。
- `重置`：次按钮，恢复默认排序和筛选。
- `应用`：主按钮，应用当前选择并关闭底表。
- 遮罩和系统返回：取消未应用改动。

按钮状态：

- 未改变任何选项时，`应用` 可以保持可用，也可以禁用；需要产品确认。
- 应用中显示加载态。
- 当前分组无书籍时，排序筛选入口禁用或不展示。

## 状态文案

loading：

- 应用中：`正在应用`

empty：

- 当前筛选无结果：`没有符合条件的书籍`
- 主按钮：`重置筛选`

error：

- 应用失败：`排序与筛选失败，请重试`

permission：

- 不适用。

offline：

- 不适用，排序筛选使用本地数据。

partial success：

- 不适用。

invalid source：

- 不适用。

## 错误与空状态

- 筛选后无结果时，仍停留在书架首页并展示空状态，不跳转独立页。
- 应用失败时保留旧列表和旧排序。
- 重置后如果仍无书籍，进入书架空状态。

## 不允许

- 不在排序筛选里展示书源规则。
- 不把筛选项写死为不可扩展结构。
- 不在未点击 `应用` 前刷新书架列表。

## 完成条件

- `应用` 默认可用；应用当前选择后关闭底表。
- 筛选项保留 `网络书` 和 `本地书`。
