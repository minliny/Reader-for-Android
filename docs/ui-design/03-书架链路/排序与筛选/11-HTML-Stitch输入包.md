## 4. 页面生成约束

排序与筛选是书架页面上的底表浮层，不是独立全屏页面。背景保留书架页面并叠加遮罩；底表从底部锚定，包含拖拽柄、标题、排序方式、排序顺序、筛选条件和底部操作栏。

排序方式为单选：`最近阅读`、`最近更新`、`加入时间`、`书名`。排序顺序为单选：`降序`、`升序`。筛选条件可多选：`全部`、`本地书`、`网络书`、`有更新`、`已缓存`。

点击 `应用` 后关闭底表并刷新列表；点击遮罩、系统返回或下滑放弃未应用改动。`重置` 只重置当前底表选项，不立即刷新列表。

响应式以 `833 x 1888 px` 为基准。小屏优先保留底表标题、排序方式、排序顺序、筛选条件和底部操作栏；短屏时底表内容区滚动，操作栏保持可触达。
