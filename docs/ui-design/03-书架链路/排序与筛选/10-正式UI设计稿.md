# 排序与筛选正式 UI 设计稿

状态：正式稿
等级：P0 / COMPONENT_MASTER
适用场景：书架首页排序筛选底表
视觉参考：`图片/01-书架排序与筛选.png`

## 1. 设计目标

排序与筛选只改变当前书架列表的展示顺序和可见范围，不改变书籍数据。

## 2. 结构

```text
ShelfSortSheet
  Scrim
  SheetContainer
    Handle
    Title("排序与筛选")
    SortModeRadioGroup
    SortOrderRadioGroup
    FilterChipGroup
    ActionBar
      ResetButton("重置")
      ApplyButton("应用")
```

## 3. 规则

- 排序方式：`最近阅读`、`最近更新`、`加入时间`、`书名`，单选。
- 排序顺序：`升序`、`降序`，单选。
- 筛选条件：`全部`、`本地书`、`网络书`、`有更新`、`已缓存`，可多选。
- 点击 `应用` 后关闭底表并刷新列表。
- 点击遮罩、系统返回或下滑，放弃未应用改动。

## 4. 明确排除

- 不展示书源规则。
- 不在未点击 `应用` 前刷新列表。
- 不把筛选项写成不可扩展结构。
