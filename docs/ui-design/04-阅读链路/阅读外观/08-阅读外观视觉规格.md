## 2. QuickModule 约束

- 只替换阅读控制层左侧主内容区：`x=10 y=34 width=288 height=297`。
- 字号步进器高度 `38dp`，减号/加号按钮 `32dp`，数值文字 `20sp / 600`。
- 主题色块尺寸 `36dp`，圆角 `10dp`，选中态外描边 `#4B817A / 2dp`。
- 选项 chip 高度 `32dp`，圆角 `16dp`，激活背景 `#DDEBE5`，未激活背景 `#F4EFE6`。
- 模块内最多展示 3 组高频设置，更多项进入 FullPage。

## 3. FullPage 约束

- 页面背景 `#F8F5EF`。
- 预览卡片使用真实阅读纸色，不使用纯白或高饱和渐变。
- 字体列表行高 `56dp`，字体名 `16sp`，样例文字 `13sp`。
- 主题色块最小 `72dp x 96dp`，圆角 `14dp`，边框 `#E7DED0 / 1dp`。
- 设置分组卡片圆角 `16dp`，分组间距 `12dp`。
