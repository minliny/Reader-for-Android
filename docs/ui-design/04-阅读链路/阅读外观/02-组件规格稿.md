# 阅读外观组件规格稿

## 组件清单

快捷面板、字体选择、主题选择、主题编辑、版式高级、翻页动画。

## 统一 token

- 页面背景、标题栏、卡片、列表行、底表、弹窗和按钮使用 `01-全局规范/` 中的全局 token。
- 当前页面如属于主 Tab 延展，底部导航和顶部栏必须继承主 Tab 页面包 token。
- 当前页面如属于阅读覆盖层，几何、遮罩、正文避让和快捷面板必须继承 `04-阅读链路/阅读控制层/` 规格。
- 当前页面如属于设置链路，列表项高度、分割线、开关、说明文字继承 `05-设置链路/00-设置页面视觉规范.md`。

## 组件约束

- 组件顺序必须服务主要任务，不为补齐视觉复杂度添加无业务意义模块。
- 图标只表达明确动作；无法确认语义时使用文字按钮。
- 危险操作使用危险色和二次确认，不与普通主按钮同层级混淆。
- 所有列表项点击区域不小于 `44dp` 高度。

## 专项约束补充

### 页面专属组件

- `ThemeSwatch`
- `FontOption`
- `Stepper`
- `Slider`
- `PreviewCard`
- `SegmentControl`

### 组件执行规则

- 上述组件必须按页面树出现，不得拆成无关卡片集合。
- 同类组件必须继承本链路视觉规格中的边框、圆角、内边距、字号和触控区。
- 未在本清单出现的组件只能作为状态反馈或系统控件出现，不能成为新的主功能入口。
