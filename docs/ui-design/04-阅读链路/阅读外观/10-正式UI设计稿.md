# 阅读外观正式 UI 设计稿

状态：正式稿

## 定义范围

本页面包定义界面快捷模块，以及字体选择、主题选择、主题编辑、版式高级、翻页动画等完整管理页。

## 结构

- `QuickModule`：字体、字号、字距、行距、页边距、主题、翻页动画和正文预览。
- `FullPage`：字体选择、主题选择、主题编辑、版式高级、翻页动画。

## 继承规则

- QuickModule 只替换阅读控制层左侧主内容区。
- 右侧亮度栏常驻。
- 完整管理页不保留阅读控制层底部四主按钮。
