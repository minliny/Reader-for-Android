## 2. QuickModule 约束

- 只替换阅读控制层左侧主内容区：`x=10 y=34 width=288 height=297`。
- 主播放按钮尺寸 `52dp`，图标 `24dp`，背景 `#4B817A`，圆角 `26dp`。
- 语速、定时、声音选择使用紧凑控制行，行高 `42dp`。
- 滑杆轨道高 `4dp`，填充 `#4B817A`，轨道 `#E7DED0`，圆角 `2dp`。
- 次要按钮高度 `34dp`，圆角 `17dp`，背景 `#F4EFE6`，文字 `13sp / #5E554B`。

## 3. RunningCapsule 约束

- 胶囊尺寸遵循共享规格：高度 `44dp`，圆角 `18dp`，背景 `#FFFDF8`。
- 胶囊内容只保留朗读状态、暂停/继续和关闭。
- 不扩展为底表，不显示章节列表，不出现底部模块导航。

## 4. FullPage 约束

- 页面背景 `#F8F5EF`。
- 设置卡片背景 `#FFFDF8`，边框 `#E7DED0 / 1dp`，圆角 `16dp`。
- 设置行最小高度 `52dp`，标题 `15sp / #3D352C`，说明 `12sp / #8C8173`。
- 强操作按钮使用 `#4B817A`，高度 `44dp`，圆角 `22dp`。
