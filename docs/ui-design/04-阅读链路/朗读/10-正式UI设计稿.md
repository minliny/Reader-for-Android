# 朗读正式 UI 设计稿

状态：正式稿
母版：`../08-阅读派生模块继承视觉规格.md`

## 定义范围

本页面包定义朗读快捷模块、朗读运行胶囊和朗读设置页。P0 冻结快捷模块和运行胶囊，完整设置页为 P1。

## 结构

- `QuickModule`：当前句、上一段/下一段、开始朗读、语速、音色、范围、定时、试听。
- `RunningCapsule`：朗读中、暂停/继续、关闭。
- `FullPage`：朗读设置页，管理 TTS 引擎、音色、语速、音调、范围和行为。

## 继承规则

- 快捷模块只替换阅读控制层左侧主内容区。
- 右侧亮度栏、底部模块导航和顶部控制条不移动。
- 运行胶囊不得扩展成底表或音频播放器。

## 禁止项

- 不显示段落数量或复杂播放器时间轴。
- 不把自动翻页和朗读混成同一运行状态。
- 不把朗读设置页塞入 QuickModule。
