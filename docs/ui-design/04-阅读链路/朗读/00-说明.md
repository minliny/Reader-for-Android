## 概念

朗读用于控制当前书籍的 TTS 朗读。它不是自动翻页，也不是音频文件播放器。

朗读分为三层：

- `朗读快捷模块`：属于阅读控制层左侧主内容区，用于快速开始、暂停和调整当前会话。
- `朗读运行胶囊`：朗读开始后，控制收起为小型胶囊。
- `朗读设置页`：完整管理页，用于持久化 TTS 设置。

后续朗读快捷模块必须遵守 `READER_CONTROL_LAYER_SPEC.md`：只替换左侧主内容区，
底部模块导航和右侧亮度栏保持 `04-2` 的基本位置关系。

## 朗读快捷模块

从阅读控制层底栏 `朗读` 打开。

布局：

- 阅读页保留在底层。
- 只替换阅读控制层左侧主内容区。
- 底部四主按钮保持原位置，并高亮 `朗读`。
- 右侧亮度条常驻。
- 内容包含当前句、段落导航、开始操作、快捷设置和预览。
- 快捷模块不显示段落数量或段落进度。
- `语速 / 音色 / 范围 / 定时` 使用大尺寸两列模块，不使用过小按钮。

## 运行胶囊

朗读开始后，控制收起为小型胶囊：

- `朗读中`
- `暂停`
- `关闭`

## 设置页

用于持久化 TTS 设置：

- 朗读引擎
- 音色
- 语速和音调
- 朗读范围
- 定时关闭
- 行为

## 交互矩阵

### 朗读快捷模块

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 底栏 `朗读` | 切换控制层左侧主内容区。 | `朗读快捷模块`，亮度条与四主按钮保持原位 |
| 右上三点菜单 | 打开朗读会话操作。 | 菜单：重置会话、打开设置、TTS 引擎状态 |
| `上一段` | 移动到上一段或上一句。 | 当前覆盖层，当前句更新 |
| `开始/开始朗读` | 从当前位置开始 TTS。 | 阅读页上的 `朗读 - 运行中小胶囊` |
| `下一段` | 移动到下一段或下一句。 | 当前覆盖层，当前句更新 |
| 进度滑条 | 调整朗读位置。 | 当前覆盖层，当前句和进度更新 |
| `语速 1.0x` | 调整朗读语速。 | 语速选择器/底表或内联滑条 |
| `音色 系统女声` | 选择音色。 | 音色选择器/底表 |
| `范围 当前章节` | 选择朗读范围。 | 范围选择器/底表 |
| `定时 不开启` | 选择定时关闭。 | 定时选择器/底表 |
| `试听` | 播放短样本。 | 当前覆盖层，预览播放状态 |

### 运行胶囊

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| `暂停` | 暂停朗读并保留位置。 | 当前阅读页，胶囊可切换为继续状态 |
| `关闭` | 停止朗读并关闭胶囊。 | `阅读页 - 沉浸模式`，移除胶囊 |
| `朗读中` | 仅显示状态。 | 不跳转 |

### 设置页

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开设置页。 | 上一覆盖层或上一页 |
| 重置/菜单 | 打开低频设置操作。 | 菜单：恢复默认、TTS 诊断 |
| `更换` | 更换 TTS 引擎。 | 引擎选择器或系统 TTS 设置 |
| `试听` | 测试当前音色设置。 | 当前页，预览播放状态 |
| 音色选择器 | 修改音色。 | 音色选择器/底表 |
| 语速滑条 | 调整默认语速。 | 当前页，数值更新 |
| 音调滑条 | 调整默认音调。 | 当前页，数值更新 |
| 范围标签 | 选择默认朗读范围。 | 当前页，选中项更新 |
| 定时选项 | 选择默认停止条件。 | 当前页，选中项更新 |
| `朗读时保持屏幕常亮` | 朗读时阻止屏幕休眠。 | 当前页，开关更新 |
| `章节结束自动继续` | 章节结束后自动继续下一章。 | 当前页，开关更新 |
| `高亮当前句` | 朗读时高亮当前句。 | 当前页，开关更新 |
| `取消` | 放弃修改。 | 上一覆盖层或上一页 |
| `保存设置` | 保存设置。 | 上一覆盖层或上一页，设置更新 |
