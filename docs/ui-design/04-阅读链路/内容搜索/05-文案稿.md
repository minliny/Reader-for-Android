# 内容搜索文案稿

## 核心文案

- `搜索正文`
- `输入关键词`
- `未找到结果`
- `上一条`
- `下一条`

## 通用反馈文案

- `正在加载`
- `加载失败，请重试`
- `网络不可用，请稍后重试`
- `保存成功`
- `操作失败，请重试`
- `取消`
- `确认`

## 文案规则

- 中文文案短句优先，不使用技术字段名。
- 危险操作必须说明影响范围。
- 不显示账号、社区、推荐算法等当前范围外文案。

## 专项约束补充

### 必须覆盖的页面文案

- `搜索本书内容`
- `无匹配结果`
- `搜索失败，请重试`

### 文案执行规则

- 生成稿不得替换这些核心中文文案。
- 状态文案必须说明原因和下一步，不能只写通用失败。
- 按钮文案必须与交互结果一致，不能使用含糊的 `确定` 替代关键动作。
