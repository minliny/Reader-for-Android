# 内容搜索视觉规格

状态：正式视觉规格
继承：`../08-阅读派生模块继承视觉规格.md`

## 1. 页面类型

| 状态 | 类型 | 图片基准 | 等级 |
| --- | --- | --- | --- |
| 内容搜索覆盖层 | `Overlay` | `图片/01-内容搜索覆盖层.png` | P0 |
| 完整内容搜索页 | `FullPage` | `图片/02-完整内容搜索页.png` | P1 |

## 2. Overlay 约束

- 搜索层覆盖阅读页时，正文不重排。
- 顶部搜索框高度 `44dp`，圆角 `14dp`，背景 `#FFFDF8`，边框 `#E7DED0 / 1dp`。
- 命中结果卡片背景 `#FFFDF8`，圆角 `16dp`，边框 `#E7DED0 / 1dp`。
- 命中关键词高亮背景 `#F4DFA8`，文字保持正文色 `#3D352C`。
- 列表行最小高度 `58dp`，章节名 `13sp / #8C8173`，摘录 `15sp / #3D352C`。

## 3. FullPage 约束

- 页面背景 `#F8F5EF`。
- 顶栏高度 `56dp`，搜索框固定在顶栏下方。
- 结果列表左右边距 `16dp`，卡片间距 `10dp`。
- 只搜索当前书籍内容，不出现全局书城或网络搜索入口。
