# 内容搜索正式 UI 设计稿

状态：正式稿
母版：`../08-阅读派生模块继承视觉规格.md`

## 定义范围

本页面包定义当前书籍内搜索覆盖层和完整内容搜索页。它只搜索当前书籍，不搜索书源、网络书籍或 RSS。

## 结构

- `Overlay`：搜索输入、范围标签、结果计数、命中结果列表。
- `FullPage`：书籍上下文、搜索输入、范围标签、搜索选项、结果列表。

## 继承规则

- 覆盖层不改变阅读正文排版。
- 搜索结果跳转后返回沉浸阅读并高亮命中内容。
- 完整页不显示主 Tab 底部导航。

## 禁止项

- 不展示书籍封面、书源信息、加入书架或开始阅读按钮。
- 不搜索网络书库。
- 不改变正文分页。
