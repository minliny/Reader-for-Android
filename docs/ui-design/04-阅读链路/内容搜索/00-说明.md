## 概念

内容搜索只在当前书籍内部搜索：

- 章节标题
- 正文片段
- 书签或笔记

它不搜索书源或网络书籍。

## 阅读中覆盖层

用于阅读时快速搜索。

布局：

- 阅读页保留在底层。
- 紧凑搜索栏覆盖层。
- 底部或中部结果面板。
- 范围标签：`全部 / 章节名 / 正文 / 书签`
- 结果行展示片段和命中词高亮。

按钮：

| 界面元素 | 功能 |
| --- | --- |
| 关闭/返回 | 关闭覆盖层并返回阅读页。 |
| 清除 | 清空查询。 |
| 范围标签 | 过滤结果类型。 |
| 结果行 | 跳转到对应位置并高亮命中内容。 |

## 完整内容搜索页

用于完整浏览搜索结果。

布局：

- 顶部应用栏：返回、标题 `内容搜索`、页面菜单
- 书籍上下文行
- 搜索输入框
- 范围标签
- 搜索选项
- 摘要行
- 结果列表

## 搜索选项

| 选项 | 功能 |
| --- | --- |
| `区分大小写` | 在适用时启用大小写敏感匹配。 |
| `整词匹配` | 只匹配完整词。 |
| `仅当前章节` | 将搜索限制在当前章节。 |

## 结果行内容

- 章节标题
- 命中类型标签
- 带高亮查询词的片段
- 位置，例如段落位置或书签时间
- 跳转箭头或跳转提示

这里不展示书籍封面、书源信息、加入书架或开始阅读按钮。

## 交互矩阵

### 阅读中覆盖层

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 关闭/返回 | 关闭内容搜索覆盖层。 | `阅读页 - 沉浸模式` |
| 搜索输入框 | 编辑当前书籍内查询词。 | 当前覆盖层，结果更新 |
| 清除图标 | 清空查询。 | 当前覆盖层，空状态 |
| 范围标签 `全部/章节名/正文/书签` | 过滤命中类型。 | 当前覆盖层，结果列表更新 |
| 结果行 | 跳转到命中位置并高亮。 | `阅读页 - 沉浸模式`，定位到命中位置 |

### 完整内容搜索页

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开内容搜索。 | 上一阅读或搜索上下文 |
| 右上三点菜单 | 打开内容搜索操作。 | 菜单：清空查询、搜索设置、导出命中结果 |
| 搜索输入框 | 编辑查询。 | 当前页，结果更新 |
| 清除图标 | 清空查询。 | 当前页，空状态 |
| 范围标签 | 按全部/章节/正文/书签过滤。 | 当前页，结果更新 |
| `区分大小写` | 切换大小写敏感匹配。 | 当前页，结果重新计算 |
| `整词匹配` | 切换完整词匹配。 | 当前页，结果重新计算 |
| `仅当前章节` | 限制在当前章节。 | 当前页，结果重新计算 |
| `筛选` | 打开高级筛选。 | 筛选底表 |
| 结果行 | 跳转到命中位置并高亮。 | `阅读页 - 沉浸模式`，定位到命中位置 |
