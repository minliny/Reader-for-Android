沉浸阅读只定义阅读背景与正文排版。四角弱信息属于阅读信息层，透明点击热区属于交互层，阅读控制条和底部控制面板属于阅读控制层。

## 沉浸阅读页

阅读页是所有阅读覆盖层的不可变基础层。

## 基础布局

内容：

- 柔和纸张背景
- 左上弱文本：书名和章节，例如 `长夜余火 · 第 32 章`
- 右上弱文本：时间，例如 `10:30`
- 左下弱文本：阅读百分比
- 右下弱文本：当前章节信息；只显示当前章节名或当前章节序号，不显示总章节数
- 正文使用宽松页边距和适合长时间阅读的行高

规则：

- 不显示应用主底部导航。
- 不显示书源。
- 沉浸模式不显示按钮。
- 沉浸模式不显示贴屏幕底部的额外细进度条。
- 四角信息层中的左下阅读百分比和右下当前章节信息属于信息层弱信息，不属于正文母版。右下不得显示总章节数。
- 阅读正文区域内不放卡片。
- 覆盖层不得重排或移动阅读正文。

## 透明点击热区

透明点击热区属于交互规格，不是图片中的显性 UI：

- 左侧约 `30%`：翻上一页；
- 中间约 `40%`：显示或隐藏阅读控制层；
- 右侧约 `30%`：翻下一页。

文字选择、链接、批注等正文交互优先于翻页热区。

## 控制层

控制层叠加在阅读基础页之上。

## 顶部控制栏

内容：

- 返回箭头
- 两行标题块：
  - 书名
  - 来源与章节，例如 `起点中文网 · 第 32 章`
- 换源图标
- 三点菜单

按钮：

| 界面元素 | 功能 |
| --- | --- |
| 返回 | 离开阅读页。 |
| 换源图标 | 打开当前书籍的换源入口。 |
| 三点菜单 | 打开阅读页低频操作：刷新章节、刷新目录、打开来源页、复制链接、调试信息。 |

## 底部控制面板

分区：

- 左侧主内容区：默认显示快捷操作和章节进度；点击四主按钮后切换为对应快捷模块。
- 主工具：`目录 / 朗读 / 界面 / 设置`，保持原位置，不被功能模块移动或重画。
- 右侧亮度条：亮度滑条、系统亮度选项，四主按钮切换时常驻。

默认左侧主内容区：

- 快捷操作：`搜索 / 自动翻页 / 替换`
- 章节进度：章节标题、`上一章`、进度滑条、`本章 38%`、`下一章`

四主按钮快捷模块：

- `目录`：左侧主内容区切换为目录/书签快捷模块。
- `朗读`：左侧主内容区切换为朗读快捷模块。
- `界面`：左侧主内容区切换为阅读外观快捷模块。
- `设置`：左侧主内容区切换为阅读设置快捷模块。

按钮：

| 界面元素 | 功能 |
| --- | --- |
| `搜索` | 打开阅读中内容搜索覆盖层。 |
| `自动翻页` | 打开自动翻页设置覆盖层。 |
| `替换` | 打开内容替换快捷覆盖层。 |
| `上一章` | 跳转上一章。 |
| `下一章` | 跳转下一章。 |
| 进度滑条 | 修改当前章节阅读位置。 |
| `目录` | 在左侧主内容区显示目录快捷模块。 |
| `朗读` | 在左侧主内容区显示朗读快捷模块。 |
| `界面` | 在左侧主内容区显示阅读外观快捷模块。 |
| `设置` | 在左侧主内容区显示阅读设置快捷模块。 |
| 亮度滑条 | 调整阅读亮度。 |
| `系统` | 使用系统亮度模式。 |

## 交互矩阵

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 退出阅读页。 | 上一页，通常为 `书架` 或 `书籍信息页` |
| 换源图标 / `换源` | 切换当前书籍来源。 | 换源底表或换源页 |
| 右上三点菜单 | 打开阅读维护操作。 | 菜单：刷新章节、刷新目录、打开来源页、复制链接、调试信息 |
| `搜索` | 在当前书籍内搜索。 | `内容搜索 - 阅读中覆盖层` |
| `自动翻页` | 配置自动翻页。 | `自动翻页 - 阅读中覆盖层` |
| `替换` | 配置阅读显示替换规则。 | `替换 - 阅读中快捷页` |
| `上一章` | 加载上一章。 | 当前阅读页，上一章 |
| `下一章` | 加载下一章。 | 当前阅读页，下一章 |
| 章节进度滑条 | 跳转当前章节内位置。 | 当前阅读页，阅读位置更新 |
| `目录` | 切换控制层左侧主内容区。 | `目录快捷模块`，亮度条与四主按钮保持原位 |
| `朗读` | 切换控制层左侧主内容区。 | `朗读快捷模块`，亮度条与四主按钮保持原位 |
| `界面` | 切换控制层左侧主内容区。 | `界面快捷模块`，亮度条与四主按钮保持原位 |
| `设置` | 切换控制层左侧主内容区。 | `设置快捷模块`，亮度条与四主按钮保持原位 |
| 亮度滑条 | 调整亮度。 | 当前控制层，亮度立即生效 |
| `系统` 亮度 | 使用系统亮度。 | 当前控制层，系统亮度模式启用 |
