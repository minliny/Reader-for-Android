## 4. 页面生成约束

沉浸阅读是阅读正文的基础状态，不显示主 Tab 底部导航，也不显示阅读控制层。页面由阅读背景、正文层、信息层组成。

正文层定义阅读母版：背景色、正文宽度、字号、行高、段间距、首行缩进、段落间留白。信息层独立于正文母版：顶部/底部弱信息不参与正文排版母版定义，只作为覆盖在阅读页边缘的状态信息。

右下角章节信息只显示当前章节信息，不显示总章节数量。底部信息层保留阅读进度和当前章节关系，具体位置、透明度和文字颜色以组件局部为准。

响应式以 `835 x 1884 px` 为默认竖屏基准。小屏优先保持正文可读性，先收紧页面边距，再微调字号；短屏保持正文滚动，不把控制层或主 Tab 导航引入沉浸态。横屏/夜间以 `图片/02-夜间模式状态.png` 的宽屏排版和色值作为变体参考。
