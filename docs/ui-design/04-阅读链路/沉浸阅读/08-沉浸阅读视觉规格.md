# 沉浸阅读视觉规格

状态：正式视觉 token
适用范围：阅读正文母版、日间/夜间沉浸态
正式页面入口：`10-正式UI设计稿.md`

## 1. 母版边界

沉浸阅读母版只定义：

- 阅读背景。
- 正文排版。
- 章节标题。
- 日间和夜间正文可读性。

以下内容不属于沉浸阅读母版：

- 四角弱信息：归属阅读信息层。
- 透明点击热区：归属阅读交互层。
- 顶部控制条、底部控制面板、亮度栏：归属阅读控制层。
- 自动翻页/朗读运行胶囊：归属对应运行态。

图片 `图片/01-沉浸阅读-默认态.png` 中的四角文字只作为信息层示例，不反向定义正文母版。

## 2. 页面背景

| Token | 日间值 | 夜间值 | 说明 |
| --- | --- | --- | --- |
| `Reader.Page.Background` | `#F7F0E4` | `#1F2220` | 阅读画布背景 |
| `Reader.Page.Texture` | subtle paper grain | subtle dark paper grain | 纸张纹理可弱化，不得干扰正文 |
| `Reader.Page.EdgeVignette` | `4%-6%` warm shadow | `6%-8%` dark shadow | 边缘轻微压暗 |
| `Reader.Page.CardUsage` | none | none | 正文层禁止卡片 |

背景覆盖完整屏幕，不显示 App 主底部导航、主 Tab 顶栏、书源列表或广告。

## 3. 手机正文排版

以 `390×884dp` 逻辑画布为基准：

| Token | 值 | 说明 |
| --- | --- | --- |
| `Reader.Text.ContainerX` | `34dp` | 正文左边距 |
| `Reader.Text.ContainerY` | `126dp` | 正文起始 y |
| `Reader.Text.ContainerWidth` | `322dp` | 正文宽度 |
| `Reader.Text.Bottom` | `884dp` | 正文可延伸到底部 |
| `Reader.Text.FontSize` | `18sp` | 正文字号 |
| `Reader.Text.LineHeight` | `36dp` | 正文行高 |
| `Reader.Text.Weight` | `400` | 正文字重 |
| `Reader.Text.LetterSpacing` | `0` | 禁止负字距 |
| `Reader.Text.ParagraphSpacing` | `17dp` | 段后距 |
| `Reader.Text.FirstLineIndent` | `2em` | 首行缩进 |
| `Reader.Text.Align` | `start` | 中文正文左起排版 |

控制层出现时只覆盖正文，不改变正文宽度、行高、分页结果或滚动位置。

## 4. 章节标题

| Token | 值 | 说明 |
| --- | --- | --- |
| `Reader.ChapterTitle.FontSize` | `22sp` | 章节标题 |
| `Reader.ChapterTitle.LineHeight` | `32dp` | 标题行高 |
| `Reader.ChapterTitle.Weight` | `600` | 标题字重 |
| `Reader.ChapterTitle.MarginBottom` | `32dp` | 标题到正文 |
| `Reader.ChapterTitle.MaxLines` | `2` | 超长章节名最多两行 |

章节标题属于正文排版层，不属于顶部信息层。

## 5. 阅读信息层

四角弱信息使用独立 token，不参与正文边距计算：

| Token | 手机值 | 说明 |
| --- | --- | --- |
| `Reader.Info.CornerMarginX` | `22dp` | 左右边距 |
| `Reader.Info.TopMarginY` | `18dp` | 顶部边距 |
| `Reader.Info.BottomMarginY` | `18dp` | 底部边距 |
| `Reader.Info.FontSize` | `14sp` | 弱信息字号 |
| `Reader.Info.LineHeight` | `20dp` | 弱信息行高 |
| `Reader.Info.DayColor` | `rgba(36, 32, 26, 0.44)` | 日间弱信息 |
| `Reader.Info.NightColor` | `rgba(245, 239, 230, 0.48)` | 夜间弱信息 |

右下弱信息内容规则：

- 只显示当前章节信息，例如 `第 12 章` 或 `第一章 愚者`。
- 不显示总章节数。
- 不使用 `第 N / M 章`、`12 / 186`、`共 186 章` 等全书目录总量表达。

控制层显示时，四角弱信息隐藏，避免与顶部控制条和底部控制面板重复。

## 6. 夜间模式

| Token | 值 | 说明 |
| --- | --- | --- |
| `Reader.Night.Background` | `#1F2220` | 夜间阅读背景 |
| `Reader.Night.Text` | `#D8D0C4` | 夜间正文 |
| `Reader.Night.SecondaryText` | `rgba(216, 208, 196, 0.56)` | 弱信息 |
| `Reader.Night.Selection` | `rgba(47, 111, 147, 0.38)` | 选中文本 |

夜间模式不得使用纯黑背景或高饱和蓝紫背景。

## 7. 自适应规则

| 条件 | 规则 |
| --- | --- |
| 宽度 `<360dp` | 正文左右边距可降到 `28dp`，字号不低于 `17sp` |
| 宽度 `360-430dp` | 使用标准手机参数 |
| 宽度 `>600dp` 竖屏 | 正文列居中，宽度 `560dp` 左右，字号 `22sp` |
| 横屏/大屏 | 使用双栏连续文本流，不重复章节标题 |
| 高度不足 | 保持正文可读性，信息层可弱化或隐藏 |

不通过压缩行高、负字距或缩小到不可读字号适配。

## 8. 禁止项

- 正文层不放卡片、按钮、底部导航、推荐、广告。
- 控制层和专项浮层不得改变正文分页。
- 四角弱信息不得写入正文段落或占用正文边距。
- 不使用历史废弃图反向覆盖本文 token。
