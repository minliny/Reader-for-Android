# 沉浸阅读页面结构稿

## 页面目标

展示正文阅读默认态，定义阅读背景和正文排版。点击热区和四角弱信息属于独立交互层/信息层，不归入正文排版母版。

## 入口

书架继续阅读、书籍详情阅读按钮、搜索结果阅读按钮。

## 页面区域

1. 阅读背景层：覆盖完整画布，定义纸张背景和夜间背景。
2. 正文排版层：承载章节标题、正文段落、行高、段距和正文边距。
3. 阅读信息层：四角弱信息，独立于正文排版层，不参与正文边距计算。
4. 点击热区层：透明交互层，不作为可见视觉组件。

## 自适应规则

- 手机竖屏以 `390x884dp` 为逻辑基准。
- 宽度小于 `360dp` 时使用紧凑边距，长文案换行，不压缩按钮触控区。
- 高度不足时优先保持正文可读性，四角弱信息按信息层规则避让或隐藏。
- 页面不得依赖固定像素截图；组件尺寸以全局 token 和页面包文字稿为准。

### 结构执行规则

- 当前页面结构以本页面树和 `10-正式UI设计稿.md` 为准。
- `图片/` 目录中的当前 UI 图只定义视觉比例和密度，不允许反向新增页面树外模块。
- 顶部、滚动区、底表/浮层和状态层必须分层实现，不能合并成单个自由排版容器。
