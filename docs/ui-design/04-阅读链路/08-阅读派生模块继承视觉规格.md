# 阅读派生模块继承视觉规格

状态：正式继承规格
适用范围：目录与书签、朗读、阅读外观、阅读设置、内容搜索、内容替换、自动翻页
母版来源：`沉浸阅读/08-沉浸阅读视觉规格.md`、`阅读控制层/09-阅读控制层视觉规格.md`

## 1. 母版继承

所有阅读派生模块必须继承两个候选母版：

- `沉浸阅读`：阅读背景、正文排版、章节标题。
- `阅读控制层`：顶部控制条、底部控制面板、右侧亮度栏、底部模块导航。

派生模块不得重新定义阅读正文宽度、行高、分页、顶部控制条位置、底部面板尺寸、右侧亮度栏位置或底部模块导航位置。

## 2. 派生类型

| 类型 | 定义 | 适用页面 |
| --- | --- | --- |
| `QuickModule` | 只替换阅读控制层左侧主内容区 | 目录快捷模块、朗读快捷模块、界面快捷模块、设置快捷模块 |
| `Overlay` | 在阅读页上方出现临时搜索/替换/自动翻页设置层 | 内容搜索覆盖层、内容替换快捷覆盖层、自动翻页设置覆盖层 |
| `RunningCapsule` | 运行态极简浮层 | 朗读运行胶囊、自动翻页运行胶囊 |
| `FullPage` | 离开控制层结构的完整管理页 | 完整目录页、朗读设置页、字体选择、阅读设置页、替换规则管理页 |

## 3. QuickModule 约束

QuickModule 只能替换：

```text
ReaderControl.LeftContent x=10 y=34 width=288 height=297
```

必须保持不变：

- 阅读背景和正文。
- 顶部控制条。
- 底部面板外壳。
- 右侧亮度栏。
- 底部模块导航。

QuickModule 内容超出时必须在左侧主内容区内部滚动或精简，不得扩大底部面板。

## 4. Overlay 约束

Overlay 出现时：

- 阅读正文不重排。
- 控制层如果仍可见，几何保持不变。
- 遮罩使用暖黑低透明度。
- 覆盖层不得引入主 Tab 底部导航。

内容搜索、内容替换和自动翻页设置覆盖层可以覆盖控制层局部区域，但不得改变沉浸阅读正文参数。

## 5. RunningCapsule 约束

运行胶囊必须是极简浮层：

| Token | 值 |
| --- | --- |
| `ReaderRunCapsule.Background` | `#FFFDF8` |
| `ReaderRunCapsule.BorderColor` | `#E7DED0` |
| `ReaderRunCapsule.Radius` | `18dp` |
| `ReaderRunCapsule.Height` | `44dp` |
| `ReaderRunCapsule.PaddingX` | `12dp` |
| `ReaderRunCapsule.IconSize` | `20dp` |
| `ReaderRunCapsule.TextSize` | `14sp` |

运行胶囊只提供当前运行状态、暂停/继续、关闭。不得扩展成底表、停靠栏或工具条。

## 6. FullPage 约束

FullPage 是完整管理页：

- 不保留阅读控制层底部四主按钮。
- 不复制右侧亮度栏。
- 可以使用完整顶栏、列表、搜索、底部固定操作区。
- 页面背景使用阅读管理页暖白背景 `#F8F5EF`。
- 卡片使用 `#FFFDF8 / #E7DED0 / 16dp radius`。

## 7. Stitch 输入顺序

派生页面进入 Stitch 前，输入顺序必须为：

1. `沉浸阅读/10-正式UI设计稿.md`
2. `沉浸阅读/08-沉浸阅读视觉规格.md`
3. `阅读控制层/10-正式UI设计稿.md`
4. `阅读控制层/09-阅读控制层视觉规格.md`
5. `04-阅读链路/08-阅读派生模块继承视觉规格.md`
6. 当前页面 `10-正式UI设计稿.md`
7. 当前页面图片示例

不得只给派生页面图片让 Stitch 自行推断控制层。

## 8. 禁止项

- 不移动右侧亮度栏。
- 不重排底部四个模块入口。
- 不把完整设置页塞进 QuickModule。
- 不把运行胶囊做成底表。
- 不用历史旧版图覆盖当前页面视觉基准。
- 不在阅读链路中出现主 Tab 底部导航。
