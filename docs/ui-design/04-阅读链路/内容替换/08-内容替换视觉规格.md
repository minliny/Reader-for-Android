## 2. Overlay 约束

- 覆盖层只管理显示替换，不修改原始正文。
- 容器背景 `#FFFDF8`，边框 `#E7DED0 / 1dp`，圆角 `18dp`。
- 输入框高度 `40dp`，背景 `#F4EFE6`，圆角 `12dp`。
- 预览区最小高度 `72dp`，原文和替换后文字使用上下对照。
- 主按钮高度 `40dp`，圆角 `20dp`，背景 `#4B817A`。

## 3. FullPage 约束

- 页面背景 `#F8F5EF`。
- 规则卡片最小高度 `72dp`，圆角 `16dp`，边框 `#E7DED0 / 1dp`。
- 规则标题 `15sp / #3D352C`，匹配说明 `12sp / #8C8173`。
- 编辑页表单控件行距 `12dp`，输入区圆角 `14dp`。
- 不出现文本校对、翻译或原文保存类入口。
