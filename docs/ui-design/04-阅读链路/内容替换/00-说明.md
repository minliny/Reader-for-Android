## 概念

内容替换管理阅读显示时的文本清理和替换规则。它只影响阅读展示，不修改原始文本。

它不是书源管理。

## 快捷覆盖层

从阅读控制层打开。

内容：

- 当前书籍规则
- 总开关
- 快捷规则列表
- 替换预览
- 操作

按钮：

| 界面元素 | 功能 |
| --- | --- |
| 总开关 | 启用或停用替换显示。 |
| `全部规则` | 打开替换规则管理页。 |
| 规则开关 | 启用或停用单条规则。 |
| `临时关闭` | 临时停用替换。 |
| `新增规则` | 打开规则编辑器。 |
| 齿轮 | 打开替换设置或规则管理。 |

## 规则管理页

内容：

- 搜索输入框
- 筛选：`全部 / 当前书 / 全局 / 已停用`
- 摘要卡片
- 规则列表
- 导入和新增操作

按钮：

| 界面元素 | 功能 |
| --- | --- |
| 搜索 | 按规则名称或内容查找。 |
| 筛选标签 | 过滤规则列表。 |
| 规则行 | 打开编辑器。 |
| 开关 | 启用或停用规则。 |
| `导入规则` | 导入替换规则。 |
| `新增规则` | 打开规则编辑器。 |

## 规则编辑器

分区：

- 基本信息
- 匹配内容
- 替换结果
- 预览
- 应用范围

预览包含 `输入测试文本`，用户可以粘贴或输入样例内容，验证规则命中位置。

按钮：

| 界面元素 | 功能 |
| --- | --- |
| `当前书 / 全局` | 选择规则作用域。 |
| `启用规则` | 启用或停用规则。 |
| `区分大小写` | 启用大小写敏感匹配。 |
| `正则表达式` | 将匹配内容作为正则表达式处理。 |
| `整词匹配` | 只匹配完整词。 |
| `输入测试文本` | 打开自定义预览文本输入。 |
| `重新预览` | 重新计算预览。 |
| `当前章节 / 整本书 / 后续章节` | 设置应用范围。 |
| `取消` | 放弃编辑。 |
| `保存规则` | 保存规则。 |
| 三点菜单 | 重置修改、复制规则、删除规则、查看命中记录、导出规则。 |

## 交互矩阵

### 快捷覆盖层

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 关闭/返回 | 关闭快捷覆盖层。 | `阅读页 - 沉浸模式` |
| 总开关 | 启用或停用替换显示。 | 当前覆盖层，预览和列表更新 |
| `全部规则` | 打开完整规则列表。 | `替换规则页` |
| 规则行 | 编辑选中规则。 | `替换规则编辑页` |
| 规则开关 | 启用或停用单条规则。 | 当前覆盖层 |
| `临时关闭` | 本次会话临时停用替换。 | `阅读页 - 沉浸模式` |
| `新增规则` | 新建规则。 | `替换规则编辑页` |
| 齿轮 | 打开替换管理或设置。 | `替换规则页` 或设置底表 |

### 规则管理页

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开规则管理。 | 上一页或覆盖层 |
| 右上三点菜单 | 打开规则管理操作。 | 菜单：批量导出、批量删除、停用规则、日志 |
| 搜索输入框 | 按名称或内容查找规则。 | 当前页，列表过滤 |
| 筛选标签 | 过滤规则列表。 | 当前页 |
| 规则行 | 编辑规则。 | `替换规则编辑页` |
| 开关 | 启用或停用规则。 | 当前页 |
| `导入规则` | 导入替换规则。 | 规则导入流程或底表 |
| `新增规则` | 新建规则。 | `替换规则编辑页` |

### 规则编辑器

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开编辑器。 | 上一页；如有未保存修改，需要二次确认 |
| `当前书/全局` | 选择规则作用域。 | 当前页 |
| `启用规则` | 启用或停用规则。 | 当前页 |
| `区分大小写` | 切换大小写敏感匹配。 | 当前页，预览可重新计算 |
| `正则表达式` | 将匹配内容作为正则表达式。 | 当前页，校验和预览更新 |
| `整词匹配` | 切换完整词匹配。 | 当前页，预览可重新计算 |
| `输入测试文本` | 输入自定义预览文本。 | 测试文本输入弹窗或底表 |
| `重新预览` | 重新计算预览。 | 当前页，预览更新 |
| `当前章节/整本书/后续章节` | 设置应用范围。 | 当前页 |
| `取消` | 放弃编辑。 | 上一页 |
| `保存规则` | 保存规则。 | 上一页或快捷覆盖层，规则更新 |
| 三点菜单 | 打开高级规则操作。 | 菜单：重置修改、复制规则、删除规则、命中记录、导出规则 |
