# 阅读设置正式 UI 设计稿

状态：正式稿
母版：`../08-阅读派生模块继承视觉规格.md`

## 定义范围

本页面包定义设置快捷模块和完整阅读设置页。设置内容面向阅读行为，不负责书源、RSS、同步或账号。

## 结构

- `QuickModule`：完整设置入口、屏幕方向、屏幕超时、隐藏状态栏、音量键翻页、单手翻页、进度信息。
- `FullPage`：屏幕与显示、翻页与手势、阅读辅助、进度与信息、预设管理。

## 继承规则

- QuickModule 只替换阅读控制层左侧主内容区。
- 完整设置页不显示右侧亮度栏和底部模块导航。
- 设置项使用设置链路列表 token，但背景和管理页卡片遵守阅读派生 FullPage 约束。

## 禁止项

- 不重新设计底部四主按钮。
- 不覆盖右侧亮度栏。
- 不把完整设置页伪装成控制层局部替换。
