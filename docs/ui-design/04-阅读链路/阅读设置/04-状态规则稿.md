# 阅读设置状态规则稿

## default

展示当前页面的完整默认内容，所有主操作可见且可达。

## loading

- 首次进入可使用骨架屏或居中加载。
- 局部刷新只替换对应局部内容，不清空整页。

## empty

- 空状态必须说明原因，并给出一个主行动作。
- 空状态不得显示错误语气。

## error

- 错误状态保留返回入口。
- 可恢复错误提供 `重试`；不可恢复错误给出原因和下一步。

## offline

- 网络不可用时保留本地数据展示。
- 需要联网的按钮进入禁用或点击后提示 `网络不可用，请稍后重试`。

## permission

- 需要系统权限时先说明用途，再提供去授权入口。
- 未授权不得进入空白页。

## 专项约束补充

### 页面状态定义

- `default`：设置首页
- `subpage`：分组二级页
- `loading`：读取设置
- `error`：保存失败可重试

### 状态执行规则

- 状态层不得改变页面主结构，只替换对应内容区或显示可恢复反馈。
- 有当前 UI 图的状态以图中比例和密度为准；缺图状态只能标为待补图，不得冻结。
