## 2. QuickModule 约束

- 只替换阅读控制层左侧主内容区：`x=10 y=34 width=288 height=297`。
- 快捷项使用两列栅格，每项最小高度 `52dp`，圆角 `14dp`。
- 图标尺寸 `20dp`，标题 `13sp / #3D352C`，状态值 `12sp / #8C8173`。
- 开关使用系统形态，轨道宽 `42dp`，高 `24dp`。
- 不把完整设置页塞进快捷模块。

## 3. FullPage 约束

- 页面背景 `#F8F5EF`。
- 设置分组卡片背景 `#FFFDF8`，边框 `#E7DED0 / 1dp`，圆角 `16dp`。
- 设置行最小高度 `52dp`，左右 padding `16dp`。
- 分组标题 `13sp / #8C8173 / 600`，设置标题 `15sp / #3D352C`。
- 危险或重置操作使用文字按钮，不使用高饱和红色大按钮。
