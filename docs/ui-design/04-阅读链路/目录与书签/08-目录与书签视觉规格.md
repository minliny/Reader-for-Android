## 2. QuickModule 约束

- 只替换阅读控制层左侧主内容区：`x=10 y=34 width=288 height=297`。
- 不移动顶部控制条、底部控制面板外壳、右侧亮度栏和底部模块导航。
- 模块内使用白色暖纸卡片 `#FFFDF8`，边框 `#E7DED0 / 1dp`，圆角 `14dp`。
- 标签行高度 `32dp`，激活标签文字 `#4B817A / 14sp / 600`，未激活文字 `#786F63 / 14sp / 500`。
- 章节列表行高 `38dp`，标题 `14sp / #3D352C`，次要信息 `12sp / #8C8173`。

## 3. Overlay 约束

- 覆盖层不改变阅读正文排版。
- 容器背景 `#FFFDF8`，边框 `#E7DED0 / 1dp`，圆角 `20dp`。
- 搜索框高度 `40dp`，圆角 `12dp`，背景 `#F4EFE6`。
- 章节列表使用单列纵向列表，当前章节只使用浅绿底色提示，不使用强投影。

## 4. FullPage 约束

- 页面背景 `#F8F5EF`。
- 顶栏高度 `56dp`，正文列表左右边距 `16dp`。
- 章节行最小高度 `48dp`，分割线 `#E7DED0 / 1dp`。
- 不显示主 Tab 底部导航，不显示阅读控制层右侧亮度栏。
