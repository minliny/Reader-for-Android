# 目录与书签正式 UI 设计稿

状态：正式稿
母版：`../08-阅读派生模块继承视觉规格.md`

## 定义范围

本页面包定义阅读中的目录快捷模块、目录/书签覆盖层和完整目录页。当前 P0 优先冻结 `目录快捷模块`，完整目录页作为 P1 完整管理页。

## 结构

- `QuickModule`：替换阅读控制层左侧主内容区，显示当前章节、目录标签、最近章节、书签入口和 `完整目录 >`。
- `Overlay`：目录/书签覆盖层，保留阅读背景，提供搜索、标签切换、章节列表、回到当前。
- `FullPage`：完整目录页，从书籍详情或 `完整目录 >` 进入。

## 继承规则

- 必须继承阅读控制层顶部条、底部面板、右侧亮度栏和底部模块导航。
- QuickModule 不扩大底部面板。
- 完整目录页不显示右侧亮度栏和底部模块导航。

## 禁止项

- 不在章节行添加无意义竖线、圆点或三点菜单。
- 不从覆盖层直接强制跳完整目录页。
- 不把目录层级重复写入每一行章节标题。
