## 概念

目录用于章节和书签导航。它不处理书源、书籍搜索或阅读外观设置。

## 共享规则

- 目录支持多级目录上下文。
- 从任意滚动位置都支持回到当前阅读章节。
- 章节行不显示未读标记。
- 章节行不显示左侧竖线、圆点、行级箭头或行级三点菜单。
- 状态图标仅包含缓存和书签。

当前规划校正：

- 阅读控制层底栏 `目录` 首先进入 `目录快捷模块`，该模块属于阅读控制层左侧主内容区。
- `目录快捷模块` 必须遵守 `READER_CONTROL_LAYER_SPEC.md`：底部模块导航和右侧亮度栏保持 `04-2` 的基本位置关系。
- 现有 `16-toc-overlay.png` 和派生图可作为目录扩展覆盖层参考，但不再代表四主按钮点击后的唯一 UI 形态。
- 完整目录页仍从书籍详情目录预览或目录快捷模块的 `完整目录 >` 进入。

## 阅读中覆盖层

从阅读控制层打开。

布局：

- 阅读页保留在底层。
- 底部面板。
- 标题：`目录`
- 搜索与页面菜单。
- 书籍进度副标题。
- 标签：`目录 / 书签`
- 吸顶路径栏。
- 章节列表。
- 当前章节不在视口中时，显示浮动 `回到当前` 胶囊。

进入关系：

- 仅由阅读控制层底栏 `目录` 按钮进入。
- 覆盖层内部不提供跳转到完整目录页的按钮。
- 点击章节行后关闭覆盖层并回到阅读页对应章节。

## 完整目录页

从书籍信息页或目录预览打开。

布局：

- 顶部应用栏
- 书籍摘要卡片
- 标签
- 工具行
- 吸顶路径栏
- 章节列表
- 浮动回到当前胶囊

进入关系：

- 从书籍信息页的 `目录预览 / 全部目录` 进入。
- 不从阅读中目录覆盖层直接进入。
- 点击章节行后进入阅读页对应章节。

## 多级路径栏

展示当前可见章节的上级目录上下文，例如：

- `第一部 风起`
- `第一卷 雨城`

如果书籍是扁平目录，则隐藏路径栏。

目录层级按书籍真实结构自适应，不强制所有小说都有 `部 / 卷 / 章`：

| 目录结构 | 覆盖层显示 | 完整目录页显示 | 章节行 |
| --- | --- | --- | --- |
| 只有章节 | 不显示路径栏 | 不显示路径栏 | 直接显示章节列表，例如 `第 55 章 灯下人` |
| 卷 + 章 | 路径栏显示当前卷，例如 `第一卷 雨城` | 当前卷作为吸顶路径或分组标题 | 章节行不额外缩进，必要时在卷切换处显示分组标题 |
| 部 + 卷 + 章 | 路径栏显示当前部与当前卷 | 部/卷作为上下文路径，章节列表保持干净 | 章节行仍只显示章名和右侧状态 |
| 自定义分组 + 章 | 路径栏显示原始分组名 | 分组标题使用原文 | 不改写章节名 |

显示原则：

- 不为扁平章节强行补 `第一卷`。
- 不把卷名重复拼进每一行章节标题。
- 路径栏只表达当前位置上下文，不作为章节行的左侧层级树。
- 如果分组名过长，路径栏单行省略，完整名称可在层级选择或搜索结果中展示。
- 列表行密度保持一致，不因目录层级变化而改变右侧缓存/书签槽位位置。

## 回到当前

当当前章节不可见时：

- 显示 `回到当前 · 第 32 章 雨夜`
- 点击滚动到当前阅读章节
- 如果当前在书签标签，点击后切换到目录标签并定位当前章节

## 章节行

内容：

- 章节标题
- 可选副标题，例如 `当前阅读 · 38%`
- 右侧状态区有两个固定槽位：
  - 缓存
  - 书签 + 数量

当前章节用柔和行背景和 `当前阅读` 标识，不使用左侧竖线。

## 按钮

| 界面元素 | 功能 |
| --- | --- |
| 搜索图标 | 搜索当前目录中的章节或书签。 |
| 三点菜单 | 目录选项：刷新目录、缓存目录、折叠/展开层级。 |
| `跳到当前` | 滚动到当前阅读章节。 |
| `倒序` | 反转章节顺序。 |
| `仅缓存` | 过滤已缓存章节。 |
| 章节行 | 打开章节。 |
| 书签图标/数量 | 打开书签标签并过滤到该章节。 |
| `回到当前` | 回到当前阅读章节。 |

## 书签标签

书签标签用于查看当前书籍内已保存的书签记录，不显示缓存状态。

书签记录内容：

- 所属章节标题
- 书签片段
- 段落位置或创建时间
- 可选笔记状态
- 右侧固定书签图标槽位

书签标签仍保留 `回到当前` 胶囊。当前阅读章节对应的书签记录只使用柔和行背景和 `当前阅读` 文案，不使用左侧标记。

## 搜索状态

搜索入口来自覆盖层或完整页顶部搜索图标。

- 搜索范围包含章节标题和书签内容。
- 搜索结果可以按 `全部 / 章节 / 书签` 过滤。
- 命中文字可使用浅色背景高亮。
- 点击章节结果进入对应章节。
- 点击书签结果进入对应书签位置。

## 三点菜单

目录三点菜单只处理目录自身动作：

- `刷新目录`
- `缓存当前卷`
- `折叠层级`
- `目录排序`

菜单不得混入书源检测、换源、WebDAV、RSS 或应用全局设置。

## 交互矩阵

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回/关闭 | 关闭目录。 | 覆盖层返回 `阅读页 - 沉浸模式`，完整页返回上一页 |
| 搜索图标 | 搜索章节或书签。 | 目录搜索输入框或底表 |
| 右上三点菜单 | 打开目录选项。 | 菜单：刷新目录、缓存目录、折叠/展开层级 |
| `目录` 标签 | 显示章节列表。 | 当前页/底表，目录标签 |
| `书签` 标签 | 显示书签列表。 | 当前页/底表，书签标签 |
| `跳到当前` | 滚动到当前阅读章节。 | 当前页/底表，当前章节行可见 |
| `倒序` | 反转章节顺序。 | 当前页/底表，列表顺序更新 |
| `仅缓存` | 过滤已缓存章节。 | 当前页/底表，列表过滤 |
| 吸顶路径栏 | 显示当前上级目录上下文。 | 默认不跳转，后续可打开层级选择器 |
| 章节行 | 打开选中章节。 | `阅读页 - 沉浸模式`，定位到选中章节 |
| 缓存图标 | 表示章节已缓存。 | 默认不跳转 |
| 书签图标/数量 | 查看该章节书签。 | `书签` 标签，过滤到选中章节 |
| `回到当前` 胶囊 | 回到当前阅读章节。 | 当前页/底表滚动到当前；必要时从书签标签切回目录标签 |

## 与最开始设计稿的对齐校正

最开始的目录设计稿对应 `16-toc-overlay.png` 和 `17-toc-full.png`。当前规划需要以这两张为准，修正点如下：

| 项目 | 最开始设计稿 | 后续规划偏差 | 当前校正 |
| --- | --- | --- | --- |
| 覆盖层来源 | 阅读控制层底栏 `目录` 按钮 | 曾把覆盖层扩展成可进入完整页的中转层 | 覆盖层只负责阅读中快速跳章 |
| 完整目录来源 | 书籍信息页或目录预览 | 曾写成可从覆盖层进入 | 完整目录只从书籍详情相关入口进入 |
| 章节行 | 无左竖线、无圆点、无行级箭头、无行级菜单 | 后续部分描述混入额外状态列和入口 | 保留干净章节行，只显示右侧缓存/书签状态 |
| 当前章节 | 行背景和 `当前阅读` 文案 | 曾有左侧标记或复杂进度表达倾向 | 不使用左侧标记 |
| 书源关系 | 目录页不处理书源 | 后续规划容易和换源/检测流程混在一起 | 目录只做章节和书签导航 |
| 已废弃图片 | 不包含 `_rejected/41-reader-directory-full-page-flow.png` | 41 被误生成，视觉和跳转逻辑偏离 | 不作为设计依据 |
