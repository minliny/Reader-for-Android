## 设置覆盖层

在自动翻页开始前，从阅读控制层打开。

布局：

- 阅读页保留在底层。
- 顶部覆盖层标题：`自动翻页`。
- 底部面板包含速度、模式、选项和开始操作。

控件：

| 界面元素 | 功能 |
| --- | --- |
| 速度滑条 | 设置翻页间隔。 |
| `滚动 / 翻页 / 仿真翻页` | 选择翻页方式。 |
| `屏幕常亮` | 阅读时保持屏幕常亮。 |
| `到章末停止` | 到当前章节末尾时停止。 |
| `音量键调速` | 允许用音量键调整速度。 |
| `取消` | 关闭设置覆盖层。 |
| `开始自动翻页` | 开始自动翻页。 |

## 运行胶囊

自动翻页运行状态只使用一个小型胶囊。

胶囊内容：

- `暂停`
- `关闭`

按钮：

| 界面元素 | 功能 |
| --- | --- |
| `暂停` | 暂停自动翻页。 |
| `关闭` | 停止并退出自动翻页。 |

运行状态不显示速度按钮、倒计时文本、底部停靠栏、底表或额外浮动控制。

## 完整设置页

用于持久化自动翻页配置。

分区：

- 默认速度
- 翻页方式
- 运行行为
- 触发区域
- 预览

按钮：

| 界面元素 | 功能 |
| --- | --- |
| 重置/菜单 | 重置或打开低频设置操作。 |
| `取消` | 放弃修改。 |
| `保存设置` | 保存设置。 |

## 交互矩阵

### 设置覆盖层

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 关闭/返回 | 关闭设置。 | `阅读页 - 沉浸模式` |
| 右上三点菜单 | 打开本次会话操作。 | 菜单：重置会话设置、打开完整设置 |
| 速度滑条 | 设置翻页间隔。 | 当前覆盖层，数值更新 |
| `滚动/翻页/仿真翻页` | 选择翻页方式。 | 当前覆盖层，模式更新 |
| `屏幕常亮` | 切换屏幕常亮。 | 当前覆盖层 |
| `到章末停止` | 切换章末停止。 | 当前覆盖层 |
| `音量键调速` | 切换音量键调速。 | 当前覆盖层 |
| `取消` | 放弃设置。 | `阅读页 - 沉浸模式` |
| `开始自动翻页` | 开始自动翻页。 | 阅读页上的 `自动翻页 - 运行中极简胶囊` |

### 运行胶囊

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| `暂停` | 暂停自动翻页。 | 当前阅读页，胶囊可切换为暂停/继续状态 |
| `关闭` | 停止自动翻页。 | `阅读页 - 沉浸模式`，移除胶囊 |

### 设置页

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 返回 | 离开设置页。 | 上一页或自动翻页设置覆盖层 |
| 重置/菜单 | 重置或打开低频设置操作。 | 菜单或重置确认 |
| 默认速度控件 | 调整已保存的默认速度。 | 当前页，预览更新 |
| 翻页方式选择器 | 选择默认翻页方式。 | 当前页，选中项更新 |
| 开关行 | 启用或停用行为选项。 | 当前页 |
| 触发区域选项 | 修改手势行为。 | 当前页 |
| `取消` | 放弃修改。 | 上一页或覆盖层 |
| `保存设置` | 保存设置。 | 上一页或覆盖层，默认值更新 |
