# 自动翻页视觉规格

状态：正式视觉规格
继承：`../08-阅读派生模块继承视觉规格.md`

| 状态 | 类型 | 图片基准 | 等级 |

## 2. Overlay 约束

- 覆盖层不改变正文排版，不承载朗读功能。
- 模式选择 chip 高度 `34dp`，圆角 `17dp`。
- 速度滑杆轨道高 `4dp`，填充 `#4B817A`，轨道 `#E7DED0`。
- 主按钮高度 `42dp`，圆角 `21dp`，背景 `#4B817A`。
- 次要说明文字 `12sp / #8C8173`。

## 3. RunningCapsule 约束

- 胶囊尺寸遵循共享规格：高度 `44dp`，圆角 `18dp`，背景 `#FFFDF8`。
- 只显示当前速度、暂停/继续、关闭。
- 不显示朗读图标，不显示章节列表，不扩展为底表。

## 4. FullPage 约束

- 页面背景 `#F8F5EF`。
- 设置卡片背景 `#FFFDF8`，边框 `#E7DED0 / 1dp`，圆角 `16dp`。
- 设置行最小高度 `52dp`，标题 `15sp / #3D352C`。
- 模式说明卡片使用低对比暖色，不使用蓝紫或高饱和提示色。
