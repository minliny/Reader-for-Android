# 自动翻页正式 UI 设计稿

状态：正式稿
母版：`../08-阅读派生模块继承视觉规格.md`

## 定义范围

本页面包定义自动翻页设置覆盖层、运行胶囊和完整设置页。自动翻页不负责朗读。

## 结构

- `Overlay`：速度、翻页方式、屏幕常亮、章末停止、音量键调速、取消、开始自动翻页。
- `RunningCapsule`：暂停/继续、关闭。
- `FullPage`：默认速度、翻页方式、运行行为、触发区域、预览。

## 继承规则

- 设置覆盖层不改变阅读正文排版。
- 运行态只使用极简胶囊。
- 完整设置页不显示阅读控制层底部模块导航。

## 禁止项

- 不显示复杂速度工具条或底部停靠栏。
- 不把朗读控制混入自动翻页。
- 不在运行态显示底表。
