## 约束

- 阅读控制层固定结构以 `阅读控制层/图片/03-当前视觉基线.png` 为当前手机基线。
- 平板控制层不得反向修改手机控制层结构。
- 历史候选和不采用图片保留在对应页面包内，但必须在说明或图片总册中标记用途。
- 书架入口只引用阅读入口，不能把阅读控制层完整链路混入书架链路。
- 换源只作为 P1 阅读异常修复链路；P0 只保留入口语义和跳转关系，不冻结完整换源页面。
