# 阅读控制层图片使用规则

本文定义图片在阅读控制层文档中的角色。图片未被删除不代表仍可作为正式依据。

## 1. 使用等级

| 等级 | 含义 |
| --- | --- |
| `STRUCTURE_BASELINE` | 定义结构关系 |
| `FUNCTION_BASELINE` | 定义功能组成 |
| `VISUAL_BASELINE` | 定义当前视觉、尺寸和位置 |
| `MODULE_CONTENT_REFERENCE` | 只参考模块内容，不覆盖固定结构 |
| `OVERLAY_REFERENCE` | 阅读内专项浮层或菜单参考 |
| `FULL_PAGE_REFERENCE` | 完整功能页或管理页参考 |
| `FLOW_REFERENCE` | 流程、多状态或辅助说明 |
| `RUNNING_STATE_REFERENCE` | 运行态胶囊 |
| `HISTORY_ONLY` | 历史记录，不采用 |
| `EXCLUDED` | 不得作为正式设计依据 |

## 7. 外观拆解图

`28–33` 仅用于理解：

- 信息结构；
- 字段内容；
- 页面流程拆解。

`28–33` 不得用于反推：

- 控制层尺寸；
- 底部 sheet 高度；
- 字体大小；
- 按钮布局；
- 最终视觉基准。
