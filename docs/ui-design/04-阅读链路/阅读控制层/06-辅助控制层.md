## 概念

本文档记录阅读控制层中不属于四个底栏主入口的辅助交互，包括夜间模式、换源、顶部更多菜单、亮度系统模式和章节进度控制。

## 夜间模式

从阅读控制层圆形快捷按钮进入。

状态：

- 日间控制层：按钮显示月亮图标，点击后开启夜间模式。
- 夜间控制层：阅读背景变为柔和深色，正文为低刺激浅色；按钮显示太阳图标或 `日间模式` 语义。
- 夜间沉浸页：不显示控制按钮，不显示底部进度条，只保留必要的弱信息。

交互：

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 夜间模式按钮 | 开启夜间模式。 | `夜间控制层` |
| 日间模式按钮 | 关闭夜间模式。 | `日间控制层` |
| 控制层外区域 | 收起控制层。 | `夜间沉浸页` 或日间沉浸页 |

## 阅读页换源

从阅读页顶部换源图标进入。

视觉规则：

- `24-1-reader-source-switch-list.png` 是换源组的视觉基准。
- `24-1/24-2/24-3` 只保留流程和内容参考；三张图都必须按 04-2 的颜色、字体、图标、悬浮底距和四角圆角重做。
- `24-2` 和 `24-3` 必须保持与标准化后的换源列表一致的外框位置和尺寸。
- 换源是顶部按钮触发的底表流程，不属于四主按钮快捷模块，不显示右侧亮度条和底部四主按钮。

流程：

- 阅读控制层顶部 `换源` 图标。
- `换源` 底表展示当前书籍可用来源。
- 可读来源可以直接切换。
- 需要确认的来源通过行级 `检查` 按钮进入检查流程。
- 检查流程用于确认目录匹配、章节定位、正文读取和内容校验。
- 切换成功后返回阅读页，保持阅读位置，并显示轻量提示。

交互：

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 顶部换源图标 | 打开当前书籍换源。 | `24-1 换源列表` |
| 来源行 `切换` | 直接切换可读来源。 | `阅读页`，来源更新 |
| 来源行 `检查` | 检查该来源是否适合当前阅读位置。 | `24-2 来源检查中` |
| `刷新来源` | 重新拉取可用来源。 | 当前换源列表刷新 |
| `取消` | 关闭换源底表。 | 返回阅读控制层 |
| `取消检查` | 中断检查。 | 返回 `24-1 换源列表` |
| 检查通过 | 展示可切换结果。 | `24-3 来源检查结果` |
| `切换到此来源` | 应用新来源。 | `阅读页`，来源更新并继续当前章节 |
| `返回列表` | 放弃当前结果。 | `24-1 换源列表` |

## 顶部更多菜单

从阅读页顶部三点菜单进入。

视觉规则：

- `25-4` 是顶部右上弹出菜单当前采用版，不是底表。
- `25-4` 的锚点和菜单内容可参考，但图标、字体、颜色、圆角和阴影以 04-2 为准。
- `25-1` 是顶部右上弹出菜单高度对齐前版本。
- `25-2` 是阅读页轻量反馈，不是新页面。
- `25-3` 是调试信息底表，底表尺寸和风格参考 `24` 换源组。
- 顶部更多菜单属于顶部栏交互，不属于四主按钮快捷模块，不显示右侧亮度条和底部四主按钮。

菜单项：

- `刷新本章`
- `刷新目录`
- `打开来源页`
- `复制本章链接`
- `书籍缓存`
- `调试信息`

交互：

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 顶部三点菜单 | 打开阅读维护菜单。 | 右上弹出菜单 |
| `刷新本章` | 重新加载当前章节。 | `25-2 刷新本章轻反馈` |
| `刷新目录` | 重新拉取目录。 | 当前阅读页，显示目录刷新提示 |
| `打开来源页` | 打开当前来源页。 | 外部页面或内置网页页 |
| `复制本章链接` | 复制当前章节链接。 | 当前页，显示复制成功提示 |
| `书籍缓存` | 打开缓存相关操作。 | 缓存底表或缓存页 |
| `调试信息` | 查看当前阅读调试信息。 | `25-3 调试信息底表` |
| `复制调试信息` | 复制当前阅读调试信息。 | 当前底表，显示复制成功 |
| `关闭` | 关闭调试信息底表。 | 返回阅读页 |

## 亮度系统模式

从阅读控制层亮度条的 `系统` 按钮切换。

状态：

- 手动亮度：滑条可拖动，`系统` 未选中。
- 跟随系统：滑条弱化或禁用，`系统` 高亮，显示当前系统亮度。
- 返回手动：拖动滑条或再次切换，恢复手动亮度。

亮度系统模式不得生成新的控制层外壳，只改变 04-2 右侧亮度栏内部状态。

交互：

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 亮度滑条 | 调整手动亮度。 | 当前控制层，亮度立即生效 |
| `系统` | 启用系统亮度。 | 当前控制层，滑条弱化 |
| 弱化滑条/手动切换 | 返回手动亮度。 | 当前控制层，滑条恢复可拖动 |

## 章节进度控制

从阅读控制层章节进度区操作。

状态：

- 章节进度：显示当前章节、上一章、下一章和本章进度。
- 拖动定位：显示浮动位置提示，例如 `本章 62% · 第 18 段`。
- 跳转完成：正文更新到新位置，显示轻量提示。
- 切换章节：点击上一章或下一章后显示章节切换状态。

章节进度拖动状态不得改变控制层位置、高度、圆角、亮度栏或底部工具栏。

交互：

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| `上一章` | 加载上一章。 | 当前阅读页，章节更新 |
| `下一章` | 加载下一章。 | 当前阅读页，章节更新 |
| 章节进度滑条拖动 | 预览当前章节内位置。 | 当前控制层，显示浮动位置提示 |
| 松开章节进度滑条 | 跳转到选中位置。 | 当前阅读页，正文和进度更新 |
