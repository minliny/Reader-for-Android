## 核心原则

阅读控制层是固定模块容器，不是多个互相替换的独立底表。

控制层打开后，底部控制层由三个稳定区域组成：

| 区域 | 位置 | 规则 |
| --- | --- | --- |
| 左侧主内容区 | 底部控制层左侧上方 | 根据当前功能切换内容 |
| 底部四主按钮 | 左侧主内容区下方 | 始终保持 `目录 / 朗读 / 界面 / 设置` 原位置 |
| 右侧亮度条 | 底部控制层右侧 | 常驻，不被四主按钮内容覆盖 |

点击四个主按钮时，只替换左侧主内容区。不得移动四主按钮，不得隐藏或复制右侧亮度条。

## 四主按钮快捷模块内容

### 目录快捷模块

用于阅读中快速跳章和查看书签，不承担完整目录管理。

建议内容：

- 当前章节附近目录
- `目录 / 书签` 小型切换
- 搜索入口
- `完整目录 >` 或 `更多目录 >`
- `回到当前`

完整目录页仍属于独立管理页或书籍详情链路，不应直接把整个控制层替换成完整目录页。

### 朗读快捷模块

用于阅读中快速开始或调整当前朗读会话。

建议内容：

- 当前句或当前段预览
- 播放 / 暂停
- 大尺寸参数模块：`语速 / 音色 / 范围 / 定时`
- `朗读设置 >`

不显示段落数量或段落进度。

朗读设置页是完整管理页，不属于控制层本体。

### 界面快捷模块

用于即时调整阅读外观。

建议内容：

- 字号
- 行距
- 常驻主题：`米色 / 白色 / 护眼 / 夜间`
- 字体入口
- `主题更多 >`
- `更多外观 >`

字体管理、主题选择、主题编辑、版式高级和翻页动画可以进入完整管理页。

### 设置快捷模块

用于阅读行为的高频设置。

建议内容：

- 屏幕方向
- 屏幕超时
- 隐藏状态栏
- 音量键翻页
- 单手翻页
- 进度信息
- `完整设置 >`

设置快捷模块不包含字体、主题、朗读、自动翻页速度、书源、WebDAV、RSS 或同步。

## 亮度条

亮度条是控制层右侧常驻模块。

- 四主按钮切换时亮度条不被覆盖。
- `系统` 亮度仍属于右侧亮度条。
- 完整设置页不重复提供同一套亮度滑条。

## 完整管理页

完整管理页不属于阅读控制层本体，可以离开控制层结构。

| 来源 | 完整 UI |
| --- | --- |
| 目录快捷模块 | 完整目录页 |
| 朗读快捷模块 | 朗读设置页 |
| 界面快捷模块 | 字体管理、主题选择、主题编辑、版式高级、翻页动画 |
| 设置快捷模块 | 完整阅读设置页、屏幕与显示行为、翻页与手势、进度与信息显示、阅读行为高级、设置预设 |
| 搜索覆盖层 | 完整内容搜索页 |
| 自动翻页覆盖层 | 自动翻页完整设置页 |
| 替换覆盖层 | 替换规则页、替换规则编辑页 |

## 被覆盖的旧规划

以下旧口径不再使用：

- 点击四主按钮后直接打开一张脱离 `04-2-reader-controls-opaque-icon-aligned.png` 结构的大底表。
- 把底部四主按钮重新放到某个功能面板内部。
- 点击 `设置` 后隐藏或复制右侧亮度条。
- 把 `设置快捷模块` 和 `完整阅读设置页` 混为同一个页面。
