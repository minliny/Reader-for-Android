# 阅读控制层设计规格

本文是阅读控制层结构的唯一权威定义。动作与跳转见
`READER_CONTROL_ACTION_FLOWS.md`，手机竖屏适配见
`READER_CONTROL_RESPONSIVE_RULES.md`，统一坐标、尺寸和视觉参数见
`READER_CONTROL_GEOMETRY_SPEC.md`，图片角色见
`READER_CONTROL_IMAGE_USAGE.md`。

## 1. 设计范围

阅读场景分为三层：

1. 沉浸式阅读页；
2. 阅读控制层；
3. 阅读内专项浮层、菜单、运行态与完整功能页。

阅读控制层是覆盖在阅读正文上的固定模块容器，不是多个独立底表的集合。

## 3. 沉浸式阅读页

基线图片：`../04-阅读链路/沉浸阅读/图片/01-沉浸阅读-默认态.png`。

显性 UI：

- 阅读正文；
- 左上：书名与章节；
- 右上：时间；
- 左下：阅读百分比；
- 右下：当前章节信息；只显示当前章节名或当前章节序号，不显示总章节数。

透明点击热区属于交互规格，不是图片中的显性 UI。沉浸页不显示阅读控制按钮、
App 主底栏或贴屏幕底部的额外细进度条。

## 4. 阅读控制层组成

阅读控制层由两个主要部分组成：

### 4.1 顶部控制条

固定内容：

- 返回；
- 书名；
- 来源与当前章节；
- 换源；
- 更多菜单。

模块切换时，顶部控制条的位置、尺寸、内容层级和图标风格保持不变。

### 4.2 底部模块化控制面板

固定区域：

| 区域 | 默认内容 | 模块态规则 |
| --- | --- | --- |
| 左侧主内容区域 | 快捷功能区 + 章节进度区 | 由当前模块内容替换 |
| 右侧亮度栏 | 手动亮度 + 系统亮度 | 常驻，位置和比例不变 |
| 底部模块导航 | 目录 / 朗读 / 界面 / 设置 | 常驻，只改变选中态 |
| sheet 外壳 | 不透明悬浮面板 | 位置、圆角、边界和底部留白不变 |

默认快捷功能：

- 搜索；
- 自动翻页；
- 替换。

默认章节进度：

- 当前章节；
- 上一章；
- 下一章；
- 章节内进度滑条；
- 当前章节阅读百分比。

## 5. 四个模块态

`目录 / 朗读 / 界面 / 设置` 是底部控制层模块切换入口，不是普通跳页按钮。

点击后只替换底部控制面板中的左侧主内容区域：

| 模块 | 左侧主内容 |
| --- | --- |
| 目录 | 目录/书签切换、附近章节、回到当前、完整目录入口 |
| 朗读 | 当前句、播放控制、语速、音色、范围、定时、朗读设置入口 |
| 界面 | 字体、字号、行距、常用主题、版式与翻页动画入口 |
| 设置 | 屏幕方向、屏幕超时、状态栏、音量键翻页、单手翻页、进度信息 |

模块态必须保持：

- 顶部控制条不变；
- 底部 sheet 外壳不变；
- 右侧亮度栏不变；
- 底部模块导航基本位置不变；
- 阅读正文不重排；
- 同一时间只激活一个模块。

内容超出左侧主内容区域时，只允许模块内部滚动或精简可见条目，不得扩大、
下移或重画固定控制层结构。

## 6. 阅读内专项 UI

以下功能不属于四个模块态：

| 类型 | 功能 |
| --- | --- |
| 专项覆盖层 | 内容搜索、自动翻页配置、内容替换、目录扩展、朗读扩展、换源 |
| 锚定菜单 | 顶部更多菜单 |
| 临时辅助状态 | 系统亮度、章节进度拖动、刷新反馈、检查中/检查结果 |
| 运行态胶囊 | 自动翻页运行、朗读运行 |
| 完整功能页 | 内容搜索页、替换规则、朗读设置、自动翻页设置、字体、主题、版式等 |

专项覆盖层打开时，应覆盖或收起阅读控制层，不能与底部模块导航和亮度栏混乱叠加。

## 7. 图层与正文规则

从低到高：

1. 阅读正文与四角信息层；
2. 透明点击热区；
3. 控制层遮罩；
4. 顶部控制条；
5. 底部控制面板；
6. 模块内容与辅助提示；
7. 专项浮层、菜单或运行态。

控制层及所有阅读内 UI 都叠加在阅读页上，不得改变正文宽度、行高、分页结果或滚动位置。

## 8. 禁止事项

- 不得用模块派生图修改 `04-2` 的固定控制层结构；
- 不得把四个模块入口改成直接跳页的普通按钮；
- 不得复制、隐藏或移动右侧亮度栏；
- 不得移动、复制或重新排列底部模块导航；
- 不得把完整管理页画进控制层左侧区域；
- 不得引用 `34-reader-appearance-main-panel-standard.png`；
- 不得使用 `_rejected/`、`_contact_sheets/` 或 `_progress_restore_backup/` 作为正式依据。
