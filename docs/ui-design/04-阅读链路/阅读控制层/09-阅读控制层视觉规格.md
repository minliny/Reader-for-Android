# 阅读控制层视觉规格

状态：正式视觉 token 汇总
适用范围：阅读控制层默认态、模块态、手机底部模式、侧置标准模式
正式页面入口：`10-正式UI设计稿.md`

## 1. 权威来源

本文件不重新发明控制层参数，而是把已有专项规格合并为 Stitch 和实现可执行入口：

- `00-阅读控制层规格.md`：结构和模块边界。
- `02-响应式规则.md`：手机、平板、侧置模式进入条件。
- `03-几何规格.md`：坐标、尺寸、颜色、层级。
- `04-图片使用规则.md`：图片角色和采用状态。
- `05-平板规格.md`：平板竖横屏约束。
- `06-辅助控制层.md`：辅助态。
- `08-控制层模块.md`：模块内容。

视觉测量母版为 `图片/03-当前视觉基线.png`。

## 2. 图层顺序

| 层级 | 图层 | 规则 |
| --- | --- | --- |
| 0 | 阅读背景与正文 | 来自沉浸阅读母版 |
| 1 | 阅读信息层 | 控制态隐藏 |
| 2 | 透明点击热区 | 不可见 |
| 3 | 控制态遮罩 | 暖黑低透明度 |
| 4 | 顶部控制条 | 控制态显示 |
| 5 | 底部控制面板 | 手机默认模式 |
| 6 | 模块内容和辅助提示 | 只替换允许区域 |
| 7 | 专项浮层/菜单/运行胶囊 | 高于控制层 |

控制层不得改变正文排版、宽度、分页或滚动位置。

## 7. 模块态规则

点击 `目录 / 朗读 / 界面 / 设置` 时，只替换左侧主内容区：

- 顶部控制条不变。
- 面板外壳不变。
- 右侧亮度栏不变。
- 底部模块导航不变。
- 阅读正文不重排。
- 同一时间只激活一个模块。

内容超出时模块内部滚动或精简，不扩大面板。

## 8. 侧置标准模式

| --- | --- |

| `ReaderControl.SidePanel.Height` | `343dp` |
| `ReaderControl.SidePanel.Radius` | `18dp` |
| `ReaderControl.SidePanel.Padding` | `top 16dp / right 12dp / bottom 16dp / left 12dp` |

竖屏锚点：`x=386 y=913 width=390 height=343`。
横屏锚点：`x=24 y=433 width=390 height=343`。

横竖屏不得分别生成控制层；只改变母版在阅读底图上的锚点。

## 9. 禁止项

- 不用模块派生图修改固定控制层结构。
- 不移动、复制或隐藏右侧亮度栏。
- 不移动、复制或重排底部模块导航。
- 不把完整管理页画进左侧主内容区。
- 不引用 `_rejected/`、`_contact_sheets/`、`_progress_restore_backup/` 或历史废弃图。
