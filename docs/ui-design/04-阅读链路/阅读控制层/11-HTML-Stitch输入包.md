## 4. 页面生成约束

阅读控制层是覆盖在沉浸阅读页上的控制浮层，不显示主 Tab 底部导航。页面由阅读背景、顶部控制条、底部控制面板、右侧亮度/字号侧栏组成。

顶部控制条承载返回、书名/来源、换源和更多入口。底部控制面板承载快捷操作区、章节进度区、模块导航区和安全区。右侧栏承载亮度、字号和系统模式相关控件。

快捷操作区是一个整体容器，内部排列搜索、自动翻页、替换三项操作。章节进度区由标题行、上一章/下一章小胶囊、进度条组成，上一章/下一章位于标题行右侧。模块导航区是一个整体容器，内部排列目录、朗读、界面、设置四个模块入口。

响应式以 `834 x 1886 px` 为默认竖屏基准。短屏优先保持顶部控制条、底部控制面板和右侧栏的锚点关系；内容不足时底部面板内部滚动或压缩间距，不改变整体容器关系。横屏/平板使用侧置控制层母版和定稿图片作为专项参考。
