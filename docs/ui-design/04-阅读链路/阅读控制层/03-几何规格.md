# 阅读控制层统一几何与视觉参数规格

本文是阅读正文层、阅读控制层固定外壳、模块态、辅助态、专项浮层和运行胶囊的唯一
几何参数源。结构语义见 `READER_CONTROL_LAYER_SPEC.md`，动作与返回规则见
`READER_CONTROL_ACTION_FLOWS.md`，模式选择见 `READER_CONTROL_RESPONSIVE_RULES.md`，
图片角色见 `READER_CONTROL_IMAGE_USAGE.md`。

`04-2-reader-controls-opaque-icon-aligned.png` 是手机默认控制态的视觉测量母版。
`04-2-complete-relative-parameter-spec.md`、`04-2-visual-style-definition.md` 和
`04-reader-controls-layer-definition.md` 只保留为历史测量记录，不再单独定义当前参数。

## 1. 参数优先级

| 优先级 | 来源 | 负责内容 |
| --- | --- | --- |
| 1 | 本文件 | 坐标、尺寸、层级、颜色、字体、圆角、描边和允许误差 |
| 2 | `04-2` | 手机默认控制态的视觉校验 |
| 3 | `04` | 默认总控态包含的功能 |
| 4 | `04-1` | 模块关系和左侧主内容区替换规则 |
| 5 | 模块图、流程图 | 字段、内容和动作参考，不得修改固定结构 |

如文字与图片冲突，先按上述职责判断；涉及固定位置和视觉重量时以本文件与 `04-2`
为准。`34-reader-appearance-main-panel-standard.png` 是废弃历史方案，不得引用。

## 2. 坐标系与模式

### 2.1 手机底部悬浮模式

```text
逻辑画布：390 × 884dp
坐标原点：完整屏幕左上角
截图范围：包含状态栏、自绘阅读区和底部手势安全区的完整屏幕
```

高分辨率导出图按相对比例映射，不使用导出像素反推全部 dp。

### 2.2 侧置悬浮标准模式

```text
竖向逻辑画布：800 × 1280dp
横向逻辑画布：1280 × 800dp
建议导出倍率：2x
固定控制面板：390 × 343dp
```

两张图必须使用同一 dp 到 px 比例。模式进入、紧凑密度和退出条件以
`READER_CONTROL_RESPONSIVE_RULES.md` 为准。

侧置控制层图片实现必须复用：

```text
母版文件：../04-阅读链路/阅读控制层/图片/30-侧置控制层母版390x343.png
逻辑尺寸：390 × 343dp
当前导出尺寸：483 × 425px
导出宽高比：1.136471
目标逻辑宽高比：390 / 343 = 1.137026
```

横竖屏不得分别生成控制层，不得对母版进行非等比缩放。两种方向只改变母版在完整阅读
底图上的合成锚点。

## 3. 公共图层顺序

| 层级 | 图层 | 规则 |
| --- | --- | --- |
| 0 | 纸张背景 | 覆盖完整画布 |
| 1 | 纸张纹理 | 只影响背景 |
| 2 | 正文与章节标题 | 始终先完整排版 |
| 3 | 四角弱信息 | 仅沉浸态和运行胶囊态显示 |
| 4 | 控制态遮罩 | 暖黑 `#11100E`，透明度 `8%` |
| 5 | 顶部控制条 | 控制态显示 |
| 6 | 控制面板或专项浮层 | 不透明，覆盖正文 |
| 7 | 辅助提示或运行胶囊 | 位于当前交互层上方 |
| 8 | Toast、确认弹窗 | 当前页面最高层 |

控制层出现前必须完成整页正文排版。任何控制面板、浮层、菜单和胶囊都不得成为正文的
`margin`、`padding`、避让区、障碍区或分页边界。

## 4. 阅读正文与四角信息

### 4.1 手机正文

| 项目 | 参数 |
| --- | --- |
| 正文容器 | `x=34, y=126, width=322, bottom=884` |
| 正文字号 | `18sp` |
| 正文行高 | `36dp` |
| 字重 | `400` |
| 首行缩进 | `2em` |
| 段后间距 | `17dp` |
| 字间距 | `0` |

正文延伸至画布底部，控制面板仅覆盖，不触发重排。

### 4.2 侧置竖向正文

基于 `800×1280dp`：

```text
正文列：x=120, y=112, width=560, bottom=1216
章节标题：x=120, y=120, fontSize=28sp, lineHeight=42dp
正文首行：y=188
正文：22sp / 41dp，首行缩进 44dp，段后间距 20dp
```

### 4.3 侧置横向正文

基于 `1280×800dp`：

```text
正文组：x=148, y=108, width=984, bottom=752
左栏：x=148, width=460
栏间距：64
右栏：x=672, width=460
章节标题：只在左栏出现一次
左栏正文首行：y=180
右栏正文：从 y=108 承接左栏后续内容
```

左右栏是一个连续文本流，不是两份重复正文。

### 4.4 四角弱信息

| 模式 | 水平边距 | 垂直边距 | 字体 |
| --- | --- | --- | --- |
| 手机 | `22dp` | 顶部 `18dp`，底部 `18dp` | `14sp / 20dp` |
| 侧置标准 | `32dp` | 顶部 `24dp`，底部 `24dp` | `14sp / 20dp` |

内容固定为：左上书名章节、右上时间、左下阅读百分比、右下当前章节信息。右下只显示当前章节名或当前章节序号，不显示总章节数，不使用 `第 N / M 章`。控制态隐藏四角
信息，避免与顶部控制条重复。

## 5. 顶部控制条

### 5.1 手机悬浮条

```text
rect = x13 y43 width364 height59
paddingHorizontal = 17dp
radius = 14dp
border = 1dp
```

扣除内边距后的列比例：

```text
返回 12.73% / 书籍信息 65.76% / 换源 13.64% / 更多 7.87%
```

### 5.2 侧置通栏

```text
rect = x0 y0 width=W height60
paddingHorizontal = 32dp
bottomBorder = 1dp
```

返回触控区 `48×48dp`；返回图标 `28dp`。右侧换源触控区宽 `56dp`，更多触控区
`48dp`。书籍信息区占用二者之间剩余宽度，标题与副标题左对齐，单行省略。

## 6. 固定控制面板外壳

### 6.1 手机底部面板

```text
rect = x5 y528 width380 height346
bottomGap = 10dp
radius = 18dp
padding = top34 right10 bottom15 left10
dragHandle = 38×4dp，top=11dp，水平居中
```

### 6.2 侧置标准面板

竖向画布：

```text
rect = x386 y913 width390 height343
```

横向画布：

```text
rect = x24 y433 width390 height343
```

共同参数：

```text
radius = 18dp
padding = top16 right12 bottom16 left12
无拖动柄
```

侧置面板固定在规定侧边底部，不支持自由拖动。竖向左手模式只镜像外部锚点，不镜像
面板内部结构。

## 7. 控制面板内部网格

### 7.1 手机网格

控制面板内容区：

```text
leftColumn = x10 y34 width288 height297
columnGap = 10dp
brightness = x308 y34 width62 height297
quickArea = x10 y34 width288 height94
chapterArea = x10 y137 width288 height97
moduleNav = x10 y243 width288 height77
rowGap = 9dp
bottomOpticalSpace = 11dp
```

横向闭合关系为 `10 + 288 + 10 + 62 + 10 = 380`；纵向闭合关系为
`34 + 94 + 9 + 97 + 9 + 77 + 11 + 15 = 346`。不得再使用历史 demo 中无法在外壳内
闭合的 `293 + 10 + 67` 或 `100 + 10 + 104 + 10 + 82` 固定组合。

### 7.2 侧置标准网格

控制面板局部坐标：

```text
leftColumn = x12 y16 width293 height311
brightness = x315 y16 width63 height311
columnGap = 10dp
quickArea = x12 y16 width293 height99
chapterArea = x12 y125 width293 height102
moduleNav = x12 y237 width293 height78
rowGap = 10dp
```

余下 `12dp` 为底部光学留白。所有模块态必须复用这些矩形。

### 7.3 不可变区域

在默认态、目录、朗读、界面和设置五种状态中，下列区域不得移动或改尺寸：

- 顶部控制条；
- 控制面板外壳；
- 右侧亮度栏；
- 底部模块导航；
- 面板外部锚点；
- 圆角、描边和阴影。

模块态只替换：

```text
手机：面板局部 x10 y34 width288 height200（快捷区 + 间距 + 章节区）
侧置：局部 x12 y16 width293 height211
```

模块内容溢出时只在该矩形内部滚动。

## 8. 默认态组件

### 8.1 快捷功能区

- 三等分：搜索、自动翻页、替换；
- 图标槽 `42×42dp`，圆角 `10dp`；
- 图标视觉尺寸 `27dp`；
- 标签 `14sp / 20dp`；
- 手机图标槽中心约在区域高度 `37%`，标签基线约在 `75%`；
- 侧置沿用相同物理尺寸和三等分中心线。

### 8.2 章节进度区

| 项目 | 参数 |
| --- | --- |
| 左右内边距 | 区域宽度 `4.2%` |
| 顶部内边距 | 区域高度 `13.5%` |
| 章节标题 | `18sp / 24dp / 600`，单行省略 |
| 上一章、下一章 | 单个宽度为区域宽度 `19%`，高度为区域高度 `28%` |
| 轨道高度 | 区域高度 `3.85%` |
| 滑块直径 | 区域高度 `19.23%` |
| 默认进度 | `38%`，仅作为示例值 |
| 进度标签 | `13sp / 18dp` |

本章进度只存在于章节卡片内部。屏幕底边不得出现额外细进度条。

### 8.3 底部模块导航

- 四等分：目录、朗读、界面、设置；
- 图标槽 `42×42dp`，图标 `27dp`；
- 标签 `14sp / 20dp`；
- 当前项只改变浅蓝底、图标和文字颜色，不改变几何。

### 8.4 亮度栏

| 分区 | 占亮度栏高度 |
| --- | --- |
| 顶部留白 | `3.15%` |
| 标题 | `8.52%` |
| 太阳图标 | `13.88%` |
| 竖向滑块 | `43.53%` |
| 系统 A 按钮 | `13.88%` |
| 系统标签 | `9.15%` |
| 底部留白 | `7.09%` |

轨道宽度为亮度栏宽度 `5.97%`，滑块直径为亮度栏宽度 `29.85%`，系统 A 按钮直径
为亮度栏宽度 `58.21%`。亮度栏不得改成水平滑杆或普通入口按钮。

## 9. 四个模块态内容网格

四个模块使用第 7.3 节的同一可替换矩形。模块标题行固定高 `34dp`，左右内边距
`12dp`；内容区位于标题行下方，底部保留 `8dp`，行间距 `8dp`。

| 模块 | 内容区规则 |
| --- | --- |
| 目录 | 分段控件 `32dp`；路径行 `28dp`；章节行 `34dp`，可见 3–4 行并内部滚动 |
| 朗读 | 当前句最多 2 行；主播放区 `52dp`；语速/音色/范围/定时使用 `2×2` 网格 |
| 界面 | 字号、行距、字体、主题、动画按紧凑设置行排列；主题色块横排；内容内部滚动 |
| 设置 | 屏幕方向、超时使用分段控件；二值项使用 `2×2` 网格；内容内部滚动 |

“完整目录”“朗读设置”“更多外观”“完整设置”位于模块标题行右侧，使用文本入口。
模块图只提供字段顺序和状态语义，不得改变上述矩形。

## 10. 辅助状态

### 10.1 亮度辅助态

只改变右侧亮度栏：

- 拖动时滑块直径增大 `10%`；
- 跟随系统时轨道和滑块弱化，系统 A 使用浅蓝选中态；
- 其他区域位置、尺寸和内容不变；
- 不新增屏幕级亮度面板。

### 10.2 章节进度辅助态

只改变章节进度区：

```text
提示气泡：最小 72×32dp，圆角 16dp
锚点：水平中心对齐当前滑块，底边距滑块顶端 8dp
文字：13sp / 18dp
```

提示内容为“本章 62%”或章节内位置。提示不得贴屏幕底部，不得形成第二条进度线。

## 11. 专项浮层模板

搜索、自动翻页、替换、换源和来源检测打开时，默认控制面板收起。专项浮层保留完整
正文作为底层，并使用本文件第 3 节的遮罩和层级。

### 11.1 手机

```text
顶部操作栏：x13 y43 width364 height59
底部浮层：x5 width380 bottom10
底部浮层最大高度：520dp
底部浮层圆角：18dp
底部浮层内边距：16dp
```

内容少时按内容高度收缩；内容多时浮层内部滚动，不贴屏幕底边。

### 11.2 侧置标准

```text
建议浮层宽度：390dp
最大高度：min(560dp, H - 96dp)
边缘安全距：24dp
圆角：18dp
内边距：16dp
```

竖向默认锚定右下侧，横向默认锚定左下侧。专项浮层与完整控制面板互斥，不拼接成大型
控制台。

### 11.3 顶部更多菜单

| 模式 | 参数 |
| --- | --- |
| 手机 | 宽 `156dp`，右边缘对齐顶部栏内容区，顶部距画布 `96dp` |
| 侧置 | 宽 `176dp`，右边缘距安全区 `32dp`，顶部 `56dp` |

菜单圆角 `12dp`，行高 `48dp`，水平内边距 `14dp`。菜单外点击关闭，不触发翻页。

## 14. 动效

```text
底部面板进入：220ms，从自身高度下方进入
侧置面板进入：220ms，从锚定侧平移 24dp 并淡入
面板退出：180ms
模块内容切换：160ms 淡入淡出，不移动外壳
按钮反馈：90ms
滑块反馈：120ms
```

动效不得触发正文重排、控制面板尺寸变化或顶部栏位移。

## 15. 验收误差与禁止项

| 项目 | 最大误差 |
| --- | --- |
| 主要区域位置 | `±2dp` |
| 主要区域尺寸 | `±2dp` |
| 内部间距 | `±2dp` |
| 字号 | `±1sp` |
| 图标尺寸 | `±1dp` |
| 图标描边 | `±0.15dp` |
| 圆角 | `±1dp` |
| 颜色 | RGB 每通道 `±4` |

禁止：

1. 控制层出现后移动、缩窄、截断或重新分页正文。
2. 在屏幕底边添加细进度条、通栏进度或状态条。
3. 模块态修改顶部控制条、面板外壳、亮度栏或底部模块导航。
4. 使用半透明实体面板。
5. 侧置面板自由漂移、居中放大或横跨屏幕底部。
6. 横竖屏使用不同导出倍率造成 DPI 和控件物理尺寸不一致。
7. 用 `28–33`、`34` 或历史失败示意图反推固定几何。
8. 用默认 Material、Lucide 或 Emoji 替代 `04-2` 图标体系。
