# 阅读控制层正式 UI 设计稿

状态：正式稿
等级：P0 / READING_CONTROL_MASTER
适用页面：阅读页控制层默认态和模块态
视觉基准图片：`图片/03-当前视觉基线.png`
功能对照图片：`图片/01-阅读控制层-默认总控态.png`

## 1. 设计目标

阅读控制层是在阅读正文上方叠加的固定模块容器，用于完成阅读内高频操作：

- 退出阅读或切换来源。
- 搜索、自动翻页、内容替换。
- 跳转章节和调整章节内进度。
- 切换目录、朗读、界面、设置四个快捷模块。
- 调整亮度。

控制层不是 App 主页面，不显示主 Tab 底部导航，也不是多个独立底表的集合。

## 3. 页面分层

从低到高：

1. 阅读正文与阅读背景。
2. 透明点击热区。
3. 控制态遮罩。
4. 顶部控制条。
5. 底部模块化控制面板。
6. 模块内容、辅助状态、菜单或专项浮层。

控制层叠加显示，不改变正文排版。

## 5. 默认态正式设计

默认态左侧主内容区包含：

1. 快捷操作：`搜索 / 自动翻页 / 替换`。
2. 章节进度：章节标题、`上一章`、`下一章`、章节内进度滑条、`本章 38%`。
3. 模块导航：`目录 / 朗读 / 界面 / 设置`。

右侧亮度栏常驻，显示亮度、亮度滑条、字号入口和系统亮度状态。

## 6. 模块态正式设计

四个模块只替换左侧主内容区：

| 模块 | 替换内容 |
| --- | --- |
| 目录 | 目录/书签切换、附近章节、回到当前、完整目录入口 |
| 朗读 | 当前句、播放控制、语速、音色、范围、定时、朗读设置入口 |
| 界面 | 字体、字号、行距、主题、版式和翻页动画入口 |
| 设置 | 屏幕方向、屏幕超时、状态栏、音量键翻页、单手翻页、进度信息 |

顶部控制条、面板外壳、右侧亮度栏、底部模块导航不随模块变化。

## 7. 交互定义

| 入口 | 行为 |
| --- | --- |
| 返回 | 离开阅读页，回到来源页 |
| 换源 | 进入换源入口，P1 当前不进入主闭环 |
| 更多 | 打开顶部更多菜单 |
| 搜索 | 打开内容搜索覆盖层 |
| 自动翻页 | 打开自动翻页设置覆盖层 |
| 替换 | 打开内容替换快捷覆盖层 |
| 上一章/下一章 | 切换章节 |
| 章节进度滑条 | 修改当前章节阅读位置 |
| 目录/朗读/界面/设置 | 切换左侧主内容区模块 |
| 亮度滑条 | 调整阅读亮度 |
| 系统 | 切换系统亮度 |

## 10. 明确排除

- 不显示主 Tab 底部导航。
- 不把四个模块入口改成跳页按钮。
- 不移动、隐藏、复制右侧亮度栏。
- 不把完整设置页、完整目录页、完整朗读页画进控制层。
- 不引用历史废弃图作为当前实现。

## 11. 验收标准

- 控制层出现时正文不重排。
- 顶部控制条、底部面板、亮度栏和模块导航位置稳定。
- 默认态能识别搜索、自动翻页、替换、章节进度、四模块和亮度。
- 模块切换只替换左侧主内容区。
