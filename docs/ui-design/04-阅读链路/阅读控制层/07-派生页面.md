## 一、纳入范围

只有满足以下任一条件的页面纳入本文：

1. 点击阅读控制层中的按钮直接出现；
2. 仍以阅读正文作为不可变背景；
3. 保留顶部阅读栏、底部控制层或其运行态胶囊；
4. 是亮度、章节进度等控制层内部状态。

完整目录页、朗读设置页、字体管理、主题管理、完整阅读设置页等独立全页面不属于本文，它们只继承视觉语言，不继承控制层结构。

## 三、页面类型

### A. 固定控制层派生状态

保留：

- 阅读正文；
- 顶部弱状态；
- 顶部控制栏；
- 悬浮控制层外壳；
- 右侧亮度栏；
- 底部四主按钮。

只允许替换左侧功能内容。

包含：

- 目录快捷区；
- 朗读快捷区；
- 界面快捷区；
- 设置快捷区；
- 亮度系统状态；
- 章节进度拖动状态。

### B. 阅读覆盖层

保留阅读正文作为背景，但收起默认底部控制层。覆盖层拥有自己的顶部操作栏或底部悬浮面板。

包含：

- 内容搜索；
- 自动翻页设置；
- 内容替换；
- 换源与来源检测；
- 调试信息。

### C. 轻量浮层与运行态

不显示完整底部控制层，只叠加一个菜单、胶囊或提示。

包含：

- 顶部更多菜单；
- 自动翻页运行胶囊；
- 朗读运行胶囊；
- 刷新、复制、切换成功等轻提示。

## 四、固定控制层派生状态统一骨架

### 1. 固定区域

| 区域 | 继承规则 |
| --- | --- |
| 顶部弱状态 | 与 `04-2` 相同 |
| 顶部控制栏 | 相对矩形、圆角、阴影、字体、图标全部不变 |
| 统一控制层外壳 | 默认态及四主按钮快捷态均为左右 `1.28%`、底部留白 `1.13%`、高度 `39.14%` |
| 右侧亮度栏 | 宽度占控制层内容区 `17.28%`，五种状态的位置和尺寸完全一致 |
| 底部工具栏 | 占左侧功能列可用高度 `25.79%`，五种状态的位置、按钮尺寸完全一致 |
| 左侧可替换区 | 左列中工具栏上方的全部高度 |

### 2. 左侧可替换区

```text
controlHeight = 页面高度 × 39.14%
toolbarHeight = 左侧功能列可用高度 × 25.79%
moduleGap = 左侧功能列可用高度 × 3.14%
moduleAreaHeight = leftColumnHeight - toolbarHeight - moduleGap
```

快捷模块必须完整落入 `moduleAreaHeight`，不允许：

- 改变控制层左右位置、底部留白和底部圆角；
- 向下扩展或把控制层贴到屏幕底部；
- 压缩工具栏；
- 把工具栏移动到模块内部；
- 添加屏幕底部页脚；
- 覆盖右侧亮度栏。

默认态和四主按钮快捷态必须使用同一外壳，不允许切换时改变控制层边界。默认态通过快捷入口、章节切换和章节进度三个有效区域填满内容区，不使用装饰性空容器。当前来源和换源入口只保留在控制层上方的顶部栏，不在默认控制层中重复展示。

### 3. 画布规则

本版本只设计 `390 × 884` 竖屏手机画布，不包含横屏手机、折叠屏和平板方案。

- 默认态和四主按钮快捷态使用同一控制层外壳；
- 控制层内部网格使用扣除内边距后的可用高度；
- 内容超出时只允许对应模块内部滚动；
- 覆盖层、菜单和运行胶囊必须保持在手机安全区内；
- 其他设备形态留待后续独立规划，不得从本版手机稿推导。

### 4. 选中状态

| 元素 | 规则 |
| --- | --- |
| 当前主按钮图标槽 | 使用强调蓝的 `8%–12%` 浅混合底色 |
| 当前主按钮图标 | 可使用 `--accent-blue`，其余图标保持 `--icon-primary` |
| 当前主按钮文字 | 使用 `--accent-blue`，字重保持 `400` |
| 未选中按钮 | 与默认控制层一致 |

## 十七、夜间状态

夜间模式保持全部几何参数不变，只替换颜色 Token：

| 日间语义 | 夜间建议 |
| --- | --- |
| 页面底色 | 柔和深灰黑，不使用纯黑 |
| 一级面板 | 比页面底色略亮 |
| 二级面板 | 与一级面板保持可见层次 |
| 主文字 | 低刺激浅灰白 |
| 次文字 | 中灰 |
