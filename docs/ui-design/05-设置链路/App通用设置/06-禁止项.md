# App通用设置禁止项

## 功能禁止

- 不新增当前页面包未声明的入口、按钮或跳转。
- 不引入账号、社区、会员、推荐流、广告或云端商业化入口。
- 不把历史图、废弃图、备份图作为当前最终视觉依据。

## 视觉禁止

- 不使用与当前页面包图片冲突的卡片圆角、边框粗细、阴影和字体层级。
- 不让文字覆盖图标、按钮或列表内容。
- 不为了适配小屏压缩触控区；应换行、滚动或降低信息密度。

## 交互禁止

- 不在提交中允许重复点击。
- 不在关闭弹层后丢失来源页筛选、分组、排序和滚动位置。
- 不用隐藏手势承载唯一关键操作。

## 专项约束补充

### 页面专属禁止项

- 不得显示主底部导航
- 不得新增账号/会员/社区入口
- 不得使用大于 8dp 的设置卡片圆角
- 不得使用玻璃拟态或大阴影

### 禁止项执行规则

- 禁止项用于约束生成器边界，不替代正向结构和组件规格。
- 与 UI 图冲突时，先按唯一约束参考判定当前图是否为当前源图；历史图和候选图不能覆盖本禁止项。
