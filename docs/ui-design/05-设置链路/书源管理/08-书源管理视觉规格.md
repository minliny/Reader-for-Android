## 3. 页面 token

| Token | 约束 |
| --- | --- |
| `Page.Background` | 以当前 UI 设计图取色；缺图时继承 设置二级页 背景，不新增渐变或大色块 |
| `Page.PaddingX` | 常规 `16dp`，小屏 `<360dp` 使用 `14dp` |
| `Page.MaxContentWidth` | 大屏居中，最大 `720dp`；阅读派生页按阅读控制层母版限制 |
| `Page.SafeAreaTop` | 跟随系统 inset，顶部内容不得贴边 |
| `Page.SafeAreaBottom` | 避让系统手势区和底部固定操作 |

## 4. 组件 token

| Token | 约束 |
| --- | --- |
| `TopBar.Height` | 继承全局/设置/阅读链路对应顶部栏高度 |
| `TopBar.IconSlot` | 图标触控槽不小于 `44dp` |
| `Card.Background` | 以 UI 图为准；缺图继承 `#FFFDF8` |
| `Card.BorderWidth` | 常规 `1dp`，不得混用粗边框 |
| `Card.Radius` | 以 UI 图为准；缺图继承同链路卡片圆角 |
| `Card.Padding` | 常规 `16dp`，紧凑屏可降到 `14dp` |
| `ListItem.MinHeight` | 不小于 `56dp`；设置项和管理项不小于 `64dp` |
| `Control.TouchTarget` | 不小于 `44dp` |

## 5. 文本 token

| 层级 | 约束 |
| --- | --- |
| 页面标题 | 继承对应链路标题字号和字重 |
| 分组标题 | `16sp` 左右，字重 `600` |
| 正文 | `14-16sp`，行高不小于字号 `1.35` 倍 |
| 辅助信息 | `12-13sp`，颜色低于正文一级 |
| 错误/警告 | 使用语义色，不用大面积强警示背景 |

## 6. 状态 token

- `loading`：局部加载优先，不清空整页结构。
- `empty`：保留返回入口和一个主行动作。
- `error`：说明原因，提供重试或返回。
- `offline`：保留本地可见内容，只阻断联网动作。
- `permission`：先说明用途，再提供系统授权入口。

## 7. 验收要求

- HTML/Stitch 输出必须与当前 UI 图同尺寸截图对比。
- 主模块位置误差目标 `±2px`；圆角、边框、字号误差目标 `±1px`。
- 缺少独立 UI 图或完整组件局部时，只能标记 `HTML_DRAFT`，不得标记 `VISUAL_VERIFIED`。
