- `../05-设置链路/设置首页历史图/图片/01-设置首页管理入口.png`

## 概念

设置首页是底部主导航 `设置` 对应的主页面，也是 App 级能力和管理入口的聚合页。

主页面定义：仅指底部主导航一级入口对应页面。当前底栏固定为：

- `书架`
- `发现`
- `RSS`
- `设置`

本项目不规划账户功能，不设置 `我的` 底栏入口。阅读统计、缓存、备份、关于、反馈等能力统一放入设置首页。

设置首页不是阅读控制层里的 `设置快捷模块`，也不是阅读内完整设置页。它不展示阅读页底部四按钮、右侧亮度条或章节进度条。

## 页面结构

### 顶部区域

- 状态栏。
- 标题：`设置`。
- 顶部右侧：搜索图标、更多图标。
- 底部主导航：`书架 / 发现 / RSS / 设置`，当前高亮 `设置`。

### 概览卡片

用于展示本地状态，不出现账户头像、昵称、登录入口。

建议内容：

- `本地书籍`：数量。
- `订阅源`：数量。
- `书源可用`：可用数量或健康状态。
- `最近备份`：时间或未设置。

### 常用管理入口

2 x 2 紧凑入口卡：

- `书源管理`
- `RSS/订阅管理`
- `阅读偏好`
- `缓存管理`

每个入口包含图标、标题、一行状态说明。点击后进入对应功能页，当前阶段不展开。

### 分组列表

设置首页主体使用分组列表，保持管理型页面的扫描效率。

| 分组 | 入口 | 内容说明 |
| --- | --- | --- |
| `通用` | 语言、启动页、主题跟随系统 | App 级通用偏好。 |
| `阅读` | 阅读默认行为、阅读内完整设置 | 进入阅读行为相关设置；不包含字体主题完整外观页。 |
| `书架与搜索` | 书架展示、搜索范围、历史记录 | 管理书架展示与搜索偏好。 |
| `书源与订阅` | 书源管理、RSS/订阅管理、来源检测 | 管理内容来源。 |
| `同步与备份` | 本地备份、WebDAV、导入导出 | 管理数据备份与恢复。 |
| `隐私与权限` | 存储权限、通知权限、隐私清理 | 管理系统权限和隐私数据。 |
| `关于与反馈` | 版本信息、更新日志、反馈 | 放置低频信息入口。 |

## 视觉规则

- 视觉参数统一使用 `SETTINGS_PAGE_DESIGN_SPEC.md`。
- 画布基准沿用现有主页面：移动端竖屏，暖色纸张背景。
- 保持与书架、发现、RSS 一致的标题、底栏、卡片和列表密度。
- 不使用账户头像或个人主页式布局。
- 不做营销式大卡片；设置页应是安静、紧凑、可扫描的管理页面。
- 不出现阅读控制层元素。
