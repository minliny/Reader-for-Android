# 缺失页面文字定稿 V1

本文定义当前已有入口、但尚无正式 UI 示意图的页面内容与布局。后续生成图片时必须按
本文逐页生成，不得临时增加账户体系、营销模块或新的主导航。

## 1. 当前范围

主底栏固定为：

```text
书架 / 发现 / RSS / 设置
```

本项目不设置“我的”页面，不规划账户、头像、昵称或登录模块。

本轮暂不补：

- 发现页的信息源选择、更多分类、榜单和来源检测次级页；
- RSS 搜索、订阅详情、订阅源管理和分组管理；
- 横屏或平板版 App 主页面；
- 启动页和品牌宣传页。

这些内容继续保留为后续功能，不参与当前缺页补图。

## 2. 统一页面基准

视觉 token、顶部栏、字体、卡片、设置行、图标和控件统一使用
`SETTINGS_PAGE_DESIGN_SPEC.md`。本节只补充页面结构。

| 项目 | 规则 |
| --- | --- |
| 逻辑画布 | `390×884` 手机竖屏 |
| 截图范围 | 完整屏幕，包含状态栏和底部手势安全区 |
| 页面背景 | `#FAF7F2` 暖白纸张色 |
| 顶部栏 | 使用设置页独立顶部栏，不继承阅读控制层 pill 顶栏 |
| 返回按钮 | 左上独立线性箭头，返回上一级页面 |
| 页面标题 | 左对齐；管理页可在标题下显示一行摘要 |
| 页面主体 | 紧凑列表、分组行和表单，不使用营销式大卡片 |
| 底部主导航 | 只在四个主页面显示；二级和三级页面不显示 |
| 卡片圆角 | 固定 `8dp`，确认弹窗可为 `8-12dp` |
| 列表滚动 | 顶部栏固定，内容区域纵向滚动 |
| 保存行为 | 设置项默认即时生效；表单页使用明确的保存按钮 |

## 3. 第一批：阅读内完整设置

阅读内完整设置从阅读控制层设置模块的“完整设置”进入。进入完整页后，阅读控制层、
亮度栏和四个模块按钮全部收起。

### 3.1 阅读设置首页

页面 ID：`reader_settings_home`

入口：

```text
阅读控制层 → 设置模块 → 完整设置
```

布局：

1. 顶部栏：返回、标题“阅读设置”、右侧预设菜单。
2. 当前状态摘要：当前方向、翻页方式、屏幕常亮状态。
3. 四个设置分组入口：
   - 屏幕与显示；
   - 翻页与手势；
   - 阅读辅助；
   - 进度与信息。
4. 高级区域：
   - 自动加载下一章；
   - 章节失败自动重试；
   - 缓存优先；
   - 误触保护。
5. 底部危险度较低的文本操作：“恢复默认阅读设置”。

每个分组入口由图标、标题、一行当前值摘要和右箭头组成。页面不出现字体、主题、字号、
行距或翻页动画，这些属于“界面”模块。

返回：

- 返回阅读控制层的设置模块；
- 如果阅读页已被系统回收，则返回阅读页沉浸态。

### 3.2 屏幕与显示

页面 ID：`reader_settings_display`

布局：

1. 顶部栏：返回、标题“屏幕与显示”。
2. 屏幕方向：`跟随系统 / 竖屏 / 横屏` 分段控件。
3. 屏幕常亮：开关；开启时显示超时时间已忽略。
4. 屏幕超时：`跟随系统 / 5 分钟 / 10 分钟 / 常亮`。
5. 系统栏：
   - 隐藏状态栏；
   - 隐藏导航栏；
   - 沉浸式系统栏。
6. 阅读信息：
   - 显示四角信息；
   - 显示时间；
   - 显示阅读百分比；
   - 显示章节序号。
7. 预览行：使用简化阅读页缩略预览，不显示完整控制层。

规则：

- 方向改变需显示确认提示，避免误触；
- “显示四角信息”关闭时，其三个子项禁用；
- 不在本页调整正文颜色、字体或主题。

### 3.3 翻页与手势

页面 ID：`reader_settings_behavior`

布局：

1. 顶部栏：返回、标题“翻页与手势”。
2. 点击区域预览：显示 `30% / 40% / 30%` 三个透明热区示意。
3. 点击翻页：
   - 左侧上一页；
   - 右侧下一页；
   - 中间打开控制层。
4. 物理按键：
   - 音量键翻页；
   - 反转音量键方向；
   - 耳机按键控制。
5. 手势：
   - 左右滑动翻页；
   - 上下滑动调节亮度；
   - 双击切换控制层；
   - 长按选择文字。
6. 单手模式：`关闭 / 左手 / 右手`。
7. 防误触灵敏度：三段选择 `低 / 标准 / 高`。

规则：

- 存在冲突的手势必须显示说明；
- 关闭点击翻页后，中间区域仍可打开控制层；
- 当前热区配置在预览中即时更新。

### 3.4 阅读辅助

页面 ID：`reader_settings_assist`

布局：

1. 顶部栏：返回、标题“阅读辅助”。
2. 章节加载：
   - 自动预加载下一章；
   - Wi-Fi 下预加载更多；
   - 加载失败自动重试；
   - 失败重试次数。
3. 缓存策略：
   - 优先读取缓存；
   - 自动缓存后续章节；
   - 缓存范围选择。
4. 阅读保护：
   - 防止意外切章；
   - 到章末确认；
   - 来源失效时提示换源。
5. 辅助能力：
   - TTS 不可用时显示安装入口；
   - 允许后台朗读；
   - 耳机断开自动暂停。
6. 权限状态行：通知权限、电池后台权限；缺失时显示“去设置”。

规则：

- 权限行是系统能力入口，不使用普通开关伪装；
- 加载和缓存选项只影响阅读行为，不改变书源内容。

### 3.5 进度与信息

页面 ID：`reader_settings_progress_info`

布局：

1. 顶部栏：返回、标题“进度与信息”。
2. 四角信息总开关。
3. 左上内容：`书名与章节 / 仅章节 / 隐藏`。
4. 右上内容：`时间 / 电量 / 隐藏`。
5. 左下内容：`全书百分比 / 本章百分比 / 隐藏`。
6. 右下内容：`章节序号 / 页码 / 隐藏`。
7. 章节进度显示：
   - 仅控制层内部；
   - 拖动时显示百分比提示；
   - 是否显示段落位置。
8. 信息预览：简化阅读页四角预览。

规则：

- 屏幕底部不出现常驻细进度条；
- 本章进度仍只存在于阅读控制层章节卡片；
- 预览只表达信息位置，不改变正文排版。

### 3.6 阅读预设与恢复默认

页面 ID：`reader_settings_preset`

形态：从阅读设置首页右上角进入的完整页面，不使用小菜单承载全部内容。

布局：

1. 顶部栏：返回、标题“阅读预设”、新增按钮。
2. 系统预设：
   - 默认；
   - 单手阅读；
   - 沉浸阅读；
   - 朗读优先。
3. 自定义预设：名称、包含项目摘要、更新时间、应用按钮、更多菜单。
4. 底部操作：“恢复全部阅读设置为默认”。

交互：

- 应用预设前显示将被修改的设置摘要；
- 恢复默认使用确认弹窗；
- 成功显示 Toast；
- 失败保留原设置并显示错误原因。

## 4. 第二批：书源管理完整流程

书源管理属于 App 全局设置，不等同于阅读页换源浮层。书源管理二级页面不显示主底栏。

### 4.1 书源详情

页面 ID：`source_detail`

布局：

1. 顶部栏：返回、标题“书源详情”、右侧编辑和更多。
2. 状态摘要：
   - 书源名称；
   - 启用状态；
   - 可用/异常状态；
   - 最近检测时间；
   - 平均延迟。
3. 基本信息：地址、分组、类型、登录要求、更新时间。
4. 能力摘要：搜索、发现、详情、目录、正文支持状态。
5. 规则摘要：搜索规则、目录规则、正文规则，仅显示摘要和“查看”。
6. 操作区：
   - 检测书源；
   - 查看调试信息；
   - 查看错误日志；
   - 编辑书源。
7. 底部危险操作：删除书源。

规则：

- 启用开关可直接操作；
- 地址和规则长文本支持复制；
- 删除必须经过确认。

### 4.2 新增书源

页面 ID：`source_create`

布局：

1. 顶部栏：取消、标题“新增书源”、保存。
2. 导入方式：`手动填写 / 粘贴 JSON / 从文件导入`。
3. 基本信息：名称、地址、分组、类型、备注。
4. 功能规则折叠组：
   - 搜索；
   - 发现；
   - 详情；
   - 目录；
   - 正文。
5. 高级选项：请求头、登录、编码、并发限制。
6. 固定底部操作：测试、保存。

校验：

- 名称和地址必填；
- JSON 导入失败时保留原文本；
- 测试成功不自动保存；
- 离开未保存页面时显示放弃修改确认。

### 4.3 编辑书源

页面 ID：`source_edit`

与新增页共用表单结构，差异为：

- 标题“编辑书源”；
- 显示当前版本与最近修改时间；
- 提供“恢复上次保存”；
- 保存前检测规则变化；
- 允许另存为新书源；
- 不允许静默覆盖导入内容。

### 4.4 书源检测中

页面 ID：`source_check_running`

布局：

1. 顶部栏：返回、标题“检测书源”。
2. 检测对象摘要：名称、地址、当前启用状态。
3. 总进度：已完成数量、总项目、耗时。
4. 检测步骤列表：
   - 网络连接；
   - 搜索请求；
   - 发现分类；
   - 详情解析；
   - 目录解析；
   - 正文解析。
5. 每行状态：等待、检测中、通过、警告、失败。
6. 固定底部操作：取消检测。

规则：

- 取消后保留已完成结果；
- 检测过程中离开页面时转为后台任务，并显示通知或管理页状态。

### 4.5 书源检测结果

页面 ID：`source_check_result`

布局：

1. 顶部栏：返回、标题“检测结果”、分享/复制。
2. 结果摘要：`全部通过 / 部分异常 / 检测失败`。
3. 指标：总耗时、平均延迟、成功项、警告项、失败项。
4. 步骤结果列表：状态、耗时、一行说明。
5. 失败详情展开区：错误阶段、错误码、错误摘要。
6. 底部操作：
   - 重新检测；
   - 查看调试信息；
   - 编辑书源。

部分失败时不使用全屏错误页，保留成功结果供判断。

### 4.6 调试信息

页面 ID：`source_debug_info`

布局：

1. 顶部栏：返回、标题“调试信息”、复制全部。
2. 顶部分段：`请求 / 响应 / 解析 / 日志`。
3. 请求：URL、方法、请求头、请求体。
4. 响应：状态码、耗时、响应头、响应内容摘要。
5. 解析：命中规则、提取结果、空字段和警告。
6. 日志：时间、级别、阶段、消息。
7. 敏感字段提示和脱敏开关。

规则：

- 长文本使用等宽字体；
- 支持行内复制和全文复制；
- 默认隐藏 Cookie、Token 等敏感信息。

### 4.7 错误日志

页面 ID：`source_error_logs`

布局：

1. 顶部栏：返回、标题“错误日志”、清空。
2. 筛选行：书源、时间、级别、阶段。
3. 日志列表：时间、书源、阶段、错误摘要、重复次数。
4. 点击行展开：完整堆栈、请求信息、复制、进入对应书源。
5. 空状态：“暂无错误日志”。

清空日志必须经过确认；清空不删除书源。

### 4.8 删除书源确认

状态 ID：`source_delete_confirm`

使用全局确认弹窗：

- 标题“删除书源？”
- 显示书源名称；
- 说明不会删除已加入书架的书籍，但该来源将不可继续搜索或更新；
- 可选项“同时清除该书源缓存与日志”；
- 按钮“取消 / 删除”；
- 删除按钮使用危险色，不能预选附加清理项。

## 5. 第三批：App 设置二级页面

### 5.1 通用设置

页面 ID：`app_settings_general`

内容：

- App 主题：跟随系统、浅色、深色；
- 语言；
- 启动页面：书架、发现、RSS；
- 自动检查更新；
- 点击主底栏当前项时回到顶部；
- 动画与减少动态效果；
- 恢复通用设置。

### 5.2 书架与搜索设置

页面 ID：`app_settings_bookshelf_search`

内容：

- 默认书架布局：封面/列表；
- 封面列数；
- 默认分组；
- 更新标记显示；
- 搜索默认范围；
- 合并同名同作者结果；
- 保存搜索历史；
- 清空搜索历史。

### 5.3 缓存管理

页面 ID：`app_cache_management`

布局：

1. 存储概览：总缓存、书籍缓存、封面、搜索、RSS。
2. 分类清理列表：占用空间、进入详情、清理按钮。
3. 自动清理：关闭、30 天、60 天、空间不足时。
4. 下载与缓存位置。
5. 危险操作：“清除全部缓存”。

清理缓存不删除书架记录、阅读进度或用户规则。

### 5.4 同步与备份

页面 ID：`app_sync_backup`

内容：

- 本地备份：立即备份、选择位置、最近备份；
- 恢复备份；
- WebDAV：未配置/已连接状态；
- 自动备份频率；
- 包含内容：书架、进度、设置、书源、订阅；
- 导入/导出；
- 冲突处理：询问、以本地为准、以远端为准。

本项目无账户，页面不得出现账号云同步。

### 5.5 隐私与权限

页面 ID：`app_privacy_permissions`

内容：

- 文件访问权限；
- 通知权限；
- 后台运行权限；
- 电池优化状态；
- 剪贴板读取说明；
- 清除搜索历史；
- 清除调试日志；
- 隐私说明。

权限缺失使用状态行和“去设置”，不使用不可操作的假开关。

### 5.6 关于与反馈

页面 ID：`app_about_feedback`

内容：

- App 名称、版本号、构建号；
- 检查更新；
- 更新日志；
- 开源许可；
- 项目主页；
- 问题反馈；
- 导出诊断信息；
- 隐私说明。

页面不放账户信息、勋章、等级或阅读统计大卡片。

## 6. 第四批：书架、搜索与详情补充状态

### 6.1 书架空状态

状态 ID：`bookshelf_empty`

- 保留完整书架标题栏、分组栏和主底栏；
- 主体居中显示简洁书本图标；
- 标题“书架还是空的”；
- 说明“可以搜索网络书籍，或导入本地文件”；
- 主按钮“搜索书籍”；
- 次按钮“导入本地书”。

### 6.2 书籍操作底表

页面 ID：`bookshelf_book_actions`

- 顶部显示封面、书名、作者；
- 操作：继续阅读、查看信息、刷新、缓存、移动分组、导出；
- 危险操作：移出书架；
- 不展示书源切换，换源在书籍信息页处理。

### 6.3 书架排序与筛选

页面 ID：`bookshelf_sort_filter`

- 排序：最近阅读、最近更新、加入时间、书名；
- 顺序：升序/降序；
- 筛选：全部、本地书、网络书、有更新、已缓存；
- 操作：重置、应用。

### 6.4 书架分组管理

页面 ID：`bookshelf_group_management`

- 分组列表：名称、书籍数、拖动排序、更多；
- 新建分组；
- 重命名；
- 删除确认；
- 默认分组不可删除；
- 删除分组时书籍移动到默认分组，不删除书籍。

### 6.5 搜索中、无结果和失败

三个状态复用搜索结果页骨架：

| 状态 | 主体内容 | 操作 |
| --- | --- | --- |
| `book_search_loading` | 结果卡片骨架、当前搜索范围 | 停止搜索 |
| `book_search_empty` | “没有找到相关书籍”、查询词和范围摘要 | 修改关键词、扩大范围 |
| `book_search_error` | 错误摘要、失败书源数量 | 重试、查看搜索日志 |

搜索失败时不得把已成功返回的结果清空；部分失败显示顶部行内警告。

### 6.6 书籍详情加载和失效

| 状态 | 布局 |
| --- | --- |
| `book_info_loading` | 保留顶部栏，封面、元信息、目录和简介使用稳定骨架 |
| `book_info_source_invalid` | 保留已有书籍信息，来源行显示失效；主操作“更换来源”，次操作“重试” |

来源失效不等于书籍不存在，不能直接显示全屏空页面。

## 7. 第五批：全局状态模板

### 7.1 页面级 Loading

- 保留页面顶部栏、底栏和固定筛选结构；
- 内容区使用与真实布局相同的骨架；
- 不使用覆盖全屏的无限转圈；
- 超过约 8 秒显示“加载时间较长”和取消/重试入口。

### 7.2 Empty

- 简洁单色图标；
- 一行标题；
- 一行原因说明；
- 最多两个操作；
- 空状态不隐藏页面导航和筛选。

### 7.3 Error

- 显示用户可理解的错误摘要；
- 主按钮“重试”；
- 次操作按场景提供“查看日志”“更换来源”或“返回”；
- 不直接显示完整技术堆栈。

### 7.4 Network Error

- 页面已有缓存时继续展示缓存内容，并显示行内网络提示；
- 无缓存时使用页面级网络错误；
- 操作：重试、检查网络设置。

### 7.5 Permission Required

- 说明权限用途；
- 显示当前状态；
- 主操作“去设置”；
- 次操作“暂不”；
- 不把系统权限做成普通 App 开关。

### 7.6 Operation Success / Failed

- 成功：优先使用 Toast 或 Snackbar，不阻断当前任务；
- 失败：可恢复的小操作使用 Snackbar + 重试；
- 保存、删除、导入等关键失败使用对话框或页面内错误区；
- 所有反馈文案说明操作对象。
