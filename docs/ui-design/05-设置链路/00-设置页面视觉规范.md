# 设置与管理页面统一设计规范

本文是 App 设置首页、阅读完整设置页、书源管理页、App 设置二级页及其状态页面的唯一
视觉参数源。页面字段与流程仍由各专项 spec 定义；发生视觉冲突时，以本文为准。

`READER_CONTROL_GEOMETRY_SPEC.md` 只负责阅读正文与阅读控制层。完整设置页离开控制层后，
不得继承控制层的 pill 顶栏、18dp 面板圆角、25-28dp 图标或粗描边体系。

## 1. 适用范围与优先级

| 优先级 | 来源 | 负责内容 |
| --- | --- | --- |
| 1 | 本文件 | 设置与管理页的颜色、字体、间距、圆角、图标、列表和控件 |
| 2 | `MISSING_PAGE_TEXT_SPEC_V1.md` | 页面字段、顺序、状态和业务规则 |
| 3 | `settings.md`、`reader-settings.md` 等专项 spec | 页面语义、入口和交互 |
| 4 | `43`、`46`、`47`、`48`、`51` 系列图片 | 内容与构图参考，不得反向覆盖 token |

## 2. 页面基准

| 项目 | 参数 |
| --- | --- |
| 手机逻辑画布 | `390×884dp` |
| 页面背景 | `#FAF7F2` |
| 卡片背景 | `#FFFCF7` |
| 主文字 | `#11100E` |
| 次级文字 | `#6F6A64` |
| 弱文字 | `#8B857E` |
| 边框与分隔线 | `#DED6CC` |
| 开启与主要强调 | `#087EAD` |
| 选中浅背景 | `#E7F2F5` |
| 危险文字 | `#B2473E` |
| 页面水平安全边距 | `20dp`；窄屏不得小于 `18dp` |

背景必须是暖白纸张色，不使用纯白冷色背景。卡片、弹窗和控件主体必须不透明；禁止玻璃
拟态、渐变卡片、大面积高饱和色和明显投影。

## 3. 顶部栏

| 项目 | 参数 |
| --- | --- |
| 顶部内容区 | 状态栏下方，高 `64-72dp` |
| 水平边距 | `20-24dp` |
| 返回触控区 | 最小 `44×44dp` |
| 返回图标 | `24dp`，`2dp` 圆角线性描边 |
| 页面标题 | `22-24sp / 600`，左对齐 |
| 右侧图标 | `22-24dp`，与返回图标统一描边 |

返回箭头独立放置，不进入有色按钮、pill 或描边容器。顶部区域直接延续页面背景，不使用
实色悬浮条、外框、圆角底板、阴影或底部分隔线。

## 4. 字体层级

| 层级 | 参数 | 用途 |
| --- | --- | --- |
| 页面标题 | `22-24sp / 600` | 顶部栏标题 |
| 分组标题 | `15-16sp / 500`，次级文字色 | 卡片上方分组名 |
| 设置项标题 | `16-17sp / 500` | 列表行主标题 |
| 设置项说明 | `12-13sp / 400`，次级文字色 | 标题下说明 |
| 当前值 | `14-16sp / 400`，次级文字色 | 右侧值或摘要 |
| 按钮文字 | `14-16sp / 500` | 文本按钮与主要操作 |

分组标题与卡片左边缘对齐。标题与说明垂直间距 `3-5dp`，不得通过缩小图标或放大控件
制造层级差异。

## 5. 分组卡片与设置行

| 项目 | 参数 |
| --- | --- |
| 卡片圆角 | `8dp` |
| 卡片描边 | `1dp #DED6CC` |
| 阴影 | 无；仅弹窗允许极弱环境阴影 |
| 卡片间距 | `16-20dp` |
| 分组标题到卡片 | `8dp` |
| 普通行高 | `56-64dp` |
| 带说明行高 | `68-76dp` |
| 行水平内边距 | `14-16dp` |
| 左侧图标区域 | 固定 `24dp`，图标后留 `20-24dp` |
| 右侧控件边界 | 同一卡片内统一右对齐 |
| 行间分隔线 | `1dp #DED6CC`，从文字起始线延伸至右边距 |

同行元素垂直居中。一个设置分组只使用一层卡片，不嵌套卡片；最后一行不绘制分隔线。
复杂说明必须换行时增加行高，但不得超过 `76dp`，更长内容应移到详情页。

## 6. 图标

| 项目 | 参数 |
| --- | --- |
| 视觉尺寸 | `22-24dp` |
| 描边 | `1.8-2.2dp`，同页统一，默认 `2dp` |
| 颜色 | `#11100E`；状态图标可使用语义色 |
| 端点与连接 | round |
| 填充 | 不使用实心功能图标 |

同一页面不得混用实心图标与线性图标，不得混用不同线宽。更多菜单也使用线性轮廓或统一
描边表达；仅状态圆点、单选中心点等控件内部标记允许实心。

## 7. 控件

### 7.1 Switch

| 项目 | 参数 |
| --- | --- |
| 尺寸 | `48×28dp` |
| 滑块 | `24dp`，白色 |
| 开启轨道 | `#087EAD` |
| 关闭轨道 | `#CFC8C0` |
| 禁用态 | 整体透明度 `38%`，保留状态可辨识性 |

### 7.2 分段控件

- 高 `36-40dp`，外框 `1dp #DED6CC`，圆角 `8dp`。
- 各段等宽，选中项使用 `#E7F2F5`，文字仍以 `#11100E` 为主。
- 不使用强蓝色整块填充，不让控件比左侧标题更醒目。

### 7.3 步进器与选择项

- 步进器高 `36-40dp`，减号、数值、加号三段等宽，段间使用细分隔线。
- 选择项当前值右对齐；右箭头视觉尺寸 `18-20dp`、描边 `1.8-2dp`。
- 权限状态必须使用可操作状态行和 `去设置`，不得伪装为不可操作的 switch。

## 8. 预览与状态

预览区是说明性组件，使用 `#FFFCF7` 或 `#F3EEE8` 不透明浅暖背景、`8dp` 圆角和
`1dp` 暖灰边框。预览比例应接近真实页面，只表达设置结果，不改变正文排版规则。

loading、empty、error、权限和确认状态沿用本页 token。状态插图使用同一套
`22-24dp / 2dp` 线性语言，不使用冷白大卡片或默认 Material 蓝色状态模板。

## 9. 危险操作

- 放在所属分组底部或页面内容底部，保持普通行 `56-64dp` 的点击区域。
- 使用居中的 `14-16sp / 500 / #B2473E` 文本，不使用大面积红底。
- 删除、清空、恢复默认等不可逆操作必须进入确认弹窗。
- 确认弹窗继续使用暖白背景、`8-12dp` 圆角和克制红色确认按钮。

## 10. 页面结构边界

- 主底栏只出现在 `书架 / 发现 / RSS / 设置` 四个一级页面。
- 设置二级页、书源管理页和阅读完整设置页不显示主底栏。
- 完整页面不显示阅读控制层四主按钮、亮度条、章节进度或控制层 pill 顶栏。
- 不增加账户、头像、昵称、登录或“我的”入口。

## 11. 禁止项

- 不使用大于 `8dp` 的设置卡片圆角。
- 不使用大阴影、玻璃拟态、渐变卡片或纯白冷色背景。
- 不混用实心与线性功能图标，不随意改变图标和文字比例。
- 不使用不可操作的假开关表达权限或系统能力。
- 不以大 hero、营销卡片或大面积插图降低设置页扫描效率。
