# 更多按钮审计（More Action Audit）

## 总原则（Overall Rule）

- `more` 不是默认装饰按钮，必须有页面级规划来源。
- `more` 分为三类：顶栏更多（Top Bar More）、行内更多（Row More）、模块更多（Module More）。
- 没有事件或文字稿支撑的页面不得因为 Shell 默认值自动出现三点按钮。
- 仅保留入口的 `more` 只能显示轻量反馈，不生成临时菜单、底表或完整页面。
- 已规划菜单的 `more` 必须只包含当前页面声明的操作项。

## 主标签页（Main Tab Pages）

| 页面（Page） | 类型（Type） | 文档依据（Source） | 当前结论（Decision） |
|---|---|---|---|
| 书架（Bookshelf） | 顶栏更多 | `书架/文字稿.md`：打开书架级更多菜单，后续定义。 | 可显示并打开书架级菜单；只放书架链路入口，不放单本书操作。 |
| 发现（Discover） | 顶栏更多 | `发现/文字稿.md`：保留入口，菜单暂缓。 | 只保留入口和轻量反馈，不展开菜单。 |
| RSS | 顶栏更多 | `RSS/文字稿.md`：保留入口，菜单暂缓；行内三点也只保留入口。 | 顶栏和行内更多都不展开菜单。 |
| 设置（Settings） | 顶栏更多 | `设置/文字稿.md`：导入导出/恢复默认后续设计；当前不补弹层状态。 | 只保留入口和轻量反馈，不展开菜单，不执行破坏性操作。 |

## 书架链路（Library Flow）

| 页面（Page） | 类型（Type） | 文档依据（Source） | 当前结论（Decision） |
|---|---|---|---|
| 书架空状态（Bookshelf Empty） | 顶栏更多 | `COMPONENT_SPEC.md` 有 `more`，但页面作为 MainTab 空状态承载。 | 复用书架主 TAB 顶栏更多规则。 |
| 书籍搜索（Book Search） | 顶栏更多 | `COMPONENT_SPEC.md` 有 `more`，文字稿未定义菜单内容。 | 当前 demo 不显示顶栏更多；后续如保留入口，必须补清楚菜单范围。 |
| 书籍详情（Book Detail） | 页面更多 / 内容操作 | `COMPONENT_SPEC.md` 有 `more`；文字稿声明修改配置、换源底表、阅读/加入书架。 | 当前 demo 用页面内“更多操作”按钮打开书籍操作底表，不在返回顶栏自动生成三点。 |
| 书籍目录（Book Directory） | 顶栏更多 | 文字稿明确不生成搜索、更多、筛选、排序等未定义入口。 | 禁止显示顶栏更多。 |
| 排序与筛选（Sort and Filter） | 顶栏更多 | 文字稿只声明返回或关闭入口。 | 禁止显示顶栏更多。 |
| 书籍操作底表（Book Action Sheet） | 触发来源更多 | 文字稿说明从书架书籍更多按钮或长按书籍条目进入。 | 页面自身不再额外显示顶栏更多；底表内只显示书籍操作。 |
| 分组管理（Group Management） | 行内更多 | 文字稿/规格声明分组行可有更多操作。 | 顶栏不显示更多；行内更多只用于可重命名/可删除分组。 |
| 本地书导入（Local Import） | 顶栏更多 | 文字稿和事件表只声明返回、选择文件、重新选择、完成、返回书架等事件。 | 禁止显示顶栏更多；这是当前 demo 必须固定的规则。 |

## 阅读链路（Reader Flow）

| 页面（Page） | 类型（Type） | 文档依据（Source） | 当前结论（Decision） |
|---|---|---|---|
| 阅读控制层（Reader Control Layer） | 顶部控制条更多 | `阅读控制层/文字稿.md` 明确从顶部控制条右侧打开锚定菜单。 | 可显示并打开锚定菜单，菜单项限定为刷新本章、刷新目录、打开来源页、复制本章链接、书籍缓存、调试信息。 |
| 沉浸阅读（Immersive Reading） | 无更多 | 沉浸页只保留正文和弱信息层。 | 不显示更多；点击中心进入阅读控制层后再使用控制层更多。 |
| 目录与书签（TOC and Bookmarks） | 模块更多 | 文字稿和规格声明更多菜单只包含当前图中声明操作。 | 可显示模块更多；菜单项不得扩展到阅读控制层顶部更多之外的未声明操作。 |
| 阅读外观（Reading Appearance） | 更多外观入口 | 文字稿声明更多主题/外观入口。 | 可作为外观模块内部入口，不等同于全局顶栏更多。 |
| 朗读（Read Aloud） | 顶部三点待收敛 | 当前局部输入件有三点，但文字稿未定义菜单内容。 | demo 进入 ReaderShell 后不新增朗读顶栏更多菜单；完整页后续需补菜单范围或移除。 |
| 阅读设置（Reading Settings） | 无顶栏更多 | 文字稿只声明返回或关闭入口。 | 不显示顶栏更多。 |
| 自动翻页（Auto Page） | 更多选项 | 规格中 `optionsTitle` 为“更多选项”。 | 是内容区选项组，不是顶栏三点菜单。 |
| 内容搜索（Content Search） | 入口来源 | 文字稿说明可从阅读控制层顶部或更多菜单进入。 | 完整页自身不新增顶栏更多。 |
| 内容替换（Content Replacement） | 入口来源 | 文字稿说明可从阅读控制层更多菜单或快捷入口进入。 | 完整页自身不新增顶栏更多。 |
| 换源（Source Switch） | 控制层延续 | FlowShell 延续阅读控制层，窗口自身不声明新的三点。 | 换源窗口不新增自己的更多；若看到更多，只能是延续的 ReaderTopBar。 |
