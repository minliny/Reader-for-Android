# 交互规则草案

## 控制层显示和隐藏

- 点击阅读页中间区域显示或隐藏阅读控制层：`NEEDS_DECISION`，现有文档未完整冻结点击热区。
- 点击控制层外正文区域关闭控制层：`NEEDS_DECISION`，需确认是否所有快捷模块都适用。
- 返回键优先关闭当前浮层或快捷面板，再关闭控制层，再退出阅读页。
- 自动隐藏规则：`NEEDS_DECISION`，需定义控制层无操作后自动隐藏时间。

## 左右热区

- 左右区域用于翻页或跳转：`NEEDS_DECISION`。
- 需确认左/右热区与章节跳转、手势设置、单手模式之间的优先级。

## 底部按钮和快捷区

- 底部四主按钮顺序固定：`目录 / 朗读 / 界面 / 设置`。
- 点击底部按钮只替换左侧主内容区。
- 快捷区互斥：打开一个快捷区时，其他快捷区关闭。
- 右侧亮度条保持原位置。
- 底部四主按钮保持原位置。

## 章节进度

- 拖动章节进度条时显示章节进度提示。
- 松开后跳转到选中位置。
- 章节进度提示是浮层，不影响正文布局。
- 进度拖动 final 图：`NEEDS_DECISION`。

## 亮度条

- 亮度条位于阅读控制层右侧。
- 手动拖动立即调整亮度。
- 点击 `系统` 启用系统亮度。
- 亮度条出现/消失规则：`NEEDS_DECISION`。
- 亮度条不影响正文布局。

## 目录、内容搜索、内容替换

- 目录快捷模块属于阅读控制层左侧主内容区。
- 目录覆盖层和完整目录页必须与目录快捷模块区分。
- 内容搜索覆盖层打开时，控制层应收起或被覆盖：`NEEDS_DECISION`。
- 内容替换覆盖层打开时，控制层应收起或被覆盖：`NEEDS_DECISION`。
- 全页或底表弹层不得与控制层底部工具栏混乱叠加。

## 朗读

- 朗读快捷区从底栏 `朗读` 进入。
- 开始朗读后进入朗读运行胶囊。
- 朗读设置页是完整管理页，不属于控制层本体。
- 是否将朗读运行态和朗读设置页纳入 V1：`NEEDS_DECISION`。

## 界面

- 界面快捷区从底栏 `界面` 进入。
- 字体、主题、主题编辑、版式高级、翻页动画从界面快捷区进入完整管理页。
- 已确认 `28-34` 不使用。
- 已确认 `35-39` 作为完整界面全页面候选。
- `21-1` 表达阅读控制层内的界面快捷模块；`35-39` 表达完整界面全页面。正式冻结时必须分状态命名，不得混为同一个状态。

## 自动翻页

- 自动翻页从默认快捷操作进入设置覆盖层。
- 开始后进入自动翻页运行胶囊。
- 是否纳入 V1：`NEEDS_DECISION`。
