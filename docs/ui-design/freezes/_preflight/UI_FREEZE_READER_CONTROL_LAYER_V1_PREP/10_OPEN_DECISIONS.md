# 开放决策清单

## 必须人工确认

1. 当前 PNG 大尺寸是否视为 `390 x 884` 的等比例导出。
2. 朗读运行态和朗读设置页是否纳入 V1。
3. 顶部更多菜单是否纳入 V1。
4. 内容搜索/内容替换是否作为 V1 全页弹层纳入，还是只保留入口关系。
5. 自动翻页是否纳入 V1，还是只保留入口关系。
6. 控制层快捷模块是否允许替换默认章节进度区，还是必须保留弱化章节进度信息。
7. 亮度状态和章节进度拖动状态是否需要标准手机 final 图。
8. 目录快捷模块与目录覆盖层是否都纳入 V1，还是只纳入快捷模块。
9. 完整管理页是否纳入阅读控制层 V1，尤其是朗读设置页、字体、主题、版式、翻页动画。
10. 顶部换源流程是否排除在 V1 之外，或作为顶部栏辅助状态纳入。
