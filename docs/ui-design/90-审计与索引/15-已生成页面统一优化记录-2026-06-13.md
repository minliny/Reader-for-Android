## 2. 统一优化内容

对 Stitch 生成的 HTML 页面追加统一移动预览约束：

- 固定预览宽度为 `390px`。
- 固定最小高度为 `884px`。
- 禁止横向滚动。
- 限制 `header/main/footer/nav` 和常见 `max-width` 容器不得超过 `390px`。
- 限制固定底栏/底表锚定在 `390px` 宽度内。
- 图片、SVG、canvas、video 不得超出容器宽度。

这次优化解决的是“本地 PNG 和 HTML 作为设计参考图时尺寸混乱、横向裁切、缩略图不可比”的问题，不等同于逐页完成视觉母版精修。
