## 4. 质量边界

本轮补齐的是“生成输入包”和“组件局部参考”，不是 HTML 视觉复刻完成。所有生成结果仍必须停留在 `HTML_DRAFT`，直到同尺寸截图叠放通过。

多图页面已经在输入包中指定默认母版和状态参考；生成时必须遵守输入包，不得把旧版、候选、流程图合并成一个页面。
