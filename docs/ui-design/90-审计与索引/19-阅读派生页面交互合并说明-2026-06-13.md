# 阅读派生页面交互合并说明

日期：2026-06-13
交互原型：`.stitch/designs/阅读派生页面交互初稿.html`

## 结论

这些 HTML 可以合并成一个交互 HTML。合并方式不是把 26 个页面平铺进一个长页面，而是把它们整理为阅读页状态机：

| 状态类型 | 进入方式 | 退出方式 | 说明 |
| --- | --- | --- | --- |
| `ReaderBase` | 默认进入 | 无 | 沉浸阅读正文作为底层母版 |
| `ControlQuickModule` | 点击底部控制层四个模块入口 | 切换模块或关闭控制层 | 目录、朗读、界面、设置只替换左侧内容区 |
| `Overlay` | 搜索、替换、自动翻页、目录覆盖层入口 | 点击遮罩或进入完整页 | 覆盖层不重排正文 |
| `RunningCapsule` | 启动朗读或自动翻页 | 点击关闭 | 运行态极简浮层 |
| `FullPage` | 从快捷模块或覆盖层进入 | 顶部返回 | 完整管理页，不保留控制层外壳 |

## 当前已串联路径

- 底部模块：目录、朗读、界面、设置。
- 目录：目录快捷模块 -> 目录覆盖层 -> 完整目录页 -> 返回阅读页。
- 朗读：朗读快捷模块 -> 朗读运行胶囊 -> 关闭运行态；朗读设置页可从快捷模块进入。
- 阅读外观：界面快捷模块 -> 字体选择、主题选择、主题编辑、版式高级、翻页动画。
- 阅读设置：设置快捷模块 -> 阅读设置首页、屏幕与显示、翻页与手势、进度与信息。
- 内容搜索：阅读页入口 -> 内容搜索覆盖层 -> 完整搜索页。
- 内容替换：阅读页入口 -> 内容替换覆盖层 -> 替换规则管理页 -> 规则编辑页。
- 自动翻页：阅读页入口 -> 自动翻页设置覆盖层 -> 运行胶囊或完整设置页。

## 仍需精修

- 当前交互 HTML 是初稿级状态机验证，不是视觉母版。
- 阅读页顶部的搜索/替换/自动翻页入口目前用于串联测试，后续应收敛到真实触发入口，例如顶部更多菜单或长按菜单。
- 完整页之间的二级跳转已可表达，但需要逐页精修信息架构和返回层级。
