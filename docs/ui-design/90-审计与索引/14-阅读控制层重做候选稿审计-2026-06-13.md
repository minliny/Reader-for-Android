## 5. 修复方向

下一轮不应再只调 CSS token，而要按基线图重建视觉层：

1. 先移除控制态顶部弱信息，除非重新定义为信息层透出状态。
2. 背景正文直接继承沉浸阅读母版的字号、行距、边距、段落密度和纸张纹理。
3. 放大并加重关键图标：返回、换源、更多、搜索、自动翻页、替换、目录、朗读、界面、设置。
4. 按基线图重新校准快捷区、章节区、导航区的内部 padding、图标框尺寸、标题字号、按钮高度。
5. 亮度栏按基线图重画滑条、旋钮、A 字号入口和系统文字垂直节奏。
6. 补一条不同屏幕比例下的锚点规则：控制面板固定贴底，顶部条固定安全区下方，正文层不随控制层重排。
