## 2. 判定标准

页面达到统一页面视觉最低要求，需要同时满足：

```text
VISUAL_MASTER + FORMAL_DRAFT + VISUAL_SPEC + COMPONENT_REFERENCE + TOKEN_READY
```

本次审计按以下文件判断：

| 条件 | 判断 |
| --- | --- |
| `VISUAL_MASTER` | `图片/*.png` 存在 |
| `FORMAL_DRAFT` | `10-正式UI设计稿.md` 存在 |
| `VISUAL_SPEC` | `08-*视觉规格.md` 或 `09-*视觉规格.md` 存在 |
| `COMPONENT_REFERENCE` | 页面包内存在完整组件局部图目录 |
| `TOKEN_READY` | 视觉规格中有页面/组件/内容/状态 token 表并可执行 |

本次没有页面包具备页面级完整组件局部，因此没有页面进入 `STITCH_READY`。
