## 3. 合格母版必须满足

合格阅读控制层 demo 至少需要满足：

1. 以 `图片/03-当前视觉基线.png` 为视觉母版，而不是只按文字近似。
2. 严格复核 `09-阅读控制层视觉规格.md` 中的顶部条、底部面板、左区、亮度栏、三块左区内容坐标。
3. 图标必须换成可执行的线性图标资产，不能用字符替代。
4. 背景正文必须来自沉浸阅读母版，控制层出现时只覆盖，不改变正文排版。
5. 输出图片要能作为 Stitch 强参考图；如果 Stitch 复原偏离，必须标记失败，而不是接受。
6. metadata 状态只能在人工审计通过后改为 `visual_master_candidate` 或 `refined_demo`。
