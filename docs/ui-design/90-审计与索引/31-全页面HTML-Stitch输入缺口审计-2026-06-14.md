## 5. 生成边界

1. 可生成页面不等于 `STITCH_READY`。
2. 生成 HTML 后必须按输入包基准尺寸截图。
3. 截图必须与当前母版叠放验收，主模块位置目标误差 `±2px`，圆角、边框、字号目标误差 `±1px`。
4. 多状态页面必须以输入包指定的 `VISUAL_MASTER` 为默认母版，其他图只作为 `STATE_OR_VARIANT_REFERENCE`。
5. 历史图、旧版图、候选图、废弃图不得反向覆盖当前母版。
