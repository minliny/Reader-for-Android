## 4. 使用边界

可以使用：

- 作为后续阅读链路派生模块的固定背景和控制层基线。
- 作为 Stitch 强参考图重新生成时的图片输入。
- 作为实现侧阅读控制层坐标和组件层级参考。

不能直接使用：

- 不能把 `visual_master_candidate` 当成最终视觉母版。
- 不能用阅读控制层覆盖沉浸阅读正文排版。
- 不能在派生模块中移动右侧亮度栏、底部模块导航或顶部控制条。
