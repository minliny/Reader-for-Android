## 页面包完成标准

每个主标签页根页面必须满足：

1. 有唯一默认态 source 图片。
2. 有必要的模式态或状态态图片示例。
3. 有 `页面结构稿`、`组件规格稿`、`交互规则稿`、`状态规则稿`、`文案稿`、`禁止项`。
4. 复用 `../01-全局规范/02-应用外壳组件视觉规格.md` 的顶部栏、底部导航、页面背景、卡片、列表、chip、状态组件 token。
5. 与 `../01-全局规范/00-唯一约束参考.md` 的交付要求一致。
6. 明确哪些入口只保留入口、哪些次级页面不在主标签页根页面范围内。

## 使用方式

Stitch 或人工设计主标签页根页面时，必须同时读取：

1. 本目录的范围、视觉规格和交付矩阵。
2. 对应页面包下的全部 Markdown。
3. 对应页面包 `图片/` 目录中的图片示例。

图片只作为结构、密度和视觉方向参考；页面结构、组件复用、状态边界、响应式和禁止项以文字稿为准。
