## 权威约束

- 正式入口：`10-正式UI设计稿.md`
- 页面视觉 token：`08-设置首页视觉规格.md`
- 主标签页共用 token：`../../01-全局规范/02-应用外壳组件视觉规格.md`
- 设置视觉参考：`../../05-设置链路/00-设置页面视觉规范.md`
- 原始文字稿参考：`../../05-设置链路/01-设置链路说明.md`

设置首页只约束根页面。App 通用设置、缓存、同步、隐私、关于、书源管理、阅读设置均为次级页面。

## 约束继承

- 唯一全局约束：`../../01-全局规范/00-唯一约束参考.md`
- 文档权威索引：`../../01-全局规范/07-文档权威索引.md`
- 当前 UI 设计图优先定义视觉比例、密度、颜色、圆角、阴影、边距和字号。
- 本目录文字稿定义页面结构、入口、交互、状态、文案和禁止项。
- 未具备完整组件局部和截图叠放验收前，不得标记为 `STITCH_READY` 或 `IMPLEMENTATION_READY`。
