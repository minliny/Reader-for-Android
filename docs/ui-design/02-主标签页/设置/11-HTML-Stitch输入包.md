## 4. 页面生成约束

设置是主 Tab 根页面，必须继承书架 App Shell：页面背景、顶部栏、底部四项导航、导航高度、图标尺寸、文字字号、卡片圆角、边框、阴影和激活态颜色保持一致。

页面结构从上到下为顶部栏、本地概览卡、常用管理网格、全部设置列表、底部导航。当前激活底部 Tab 为设置。

本地概览卡保持四项统计的 2x2 结构。常用管理入口默认两列网格。全部设置列表保持单列扫描结构，设置行包含左侧图标、标题、说明、右侧箭头和分隔线。设置首页只承载入口，不直接执行清空、删除、恢复默认等破坏性操作。

响应式以 `852 x 1846 px` 为基准。小屏优先保留顶部栏、本地概览、至少两个常用管理入口、首屏设置行和底部导航；短屏允许内容区滚动，底部导航固定。
