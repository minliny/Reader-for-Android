# 设置首页 Page Spec

## 页面定位

设置首页是底部主导航 `设置` 对应的根页面，用于聚合 App 级管理入口。它不是账户页，也不是阅读器内设置面板。

## 页面结构

1. 系统状态栏安全区。
2. 顶部栏：标题 `设置`，右侧搜索、更多。
3. 本地概览卡：本地书籍、订阅源、书源可用、最近备份。
4. 常用管理入口：`书源管理`、`RSS/订阅管理`、`阅读偏好`、`缓存管理`。
5. 分组列表：通用、阅读、书架与搜索、书源与订阅、同步与备份、隐私与权限、关于与反馈。
6. 底部主导航：当前高亮 `设置`。

## 范围边界

- 设置首页只作为入口聚合页。
- 设置二级页不在本页面包内展开。
- 不展示账户头像、昵称、登录入口。
