# 设置首页 Component Spec

| 组件 | 说明 | Token 来源 |
| --- | --- | --- |
| `AppTopBar` | 标题、搜索、更多 | `AppShell.TopBar.*` |
| `LocalOverviewCard` | 本地状态概览 | `AppShell.Card.*` |
| `QuickEntryGrid` | 常用管理入口 2×2 | `AppShell.Card.*` |
| `SettingsSection` | 设置分组 | `AppShell.Card.*` |
| `SettingsListItem` | 设置行 | `AppShell.ListItem.*` |
| `BottomNav` | 底部主导航 | `AppShell.BottomNav.*` |

设置首页应保持管理型页面的扫描效率，避免营销页式 hero 和超大装饰卡。
