## 4. 页面生成约束

发现是主 Tab 根页面，必须继承书架 App Shell：页面背景、顶部栏、底部四项导航、导航高度、图标尺寸、文字字号、卡片圆角、边框、阴影和激活态颜色保持一致。

页面结构从上到下为顶部栏、搜索入口、信息源卡片、当前分类 chip、推荐书籍、来源状态条、来源榜单更新、底部导航。当前激活底部 Tab 为发现。

搜索入口是页面内入口条，不展开为完整搜索页。来源类型和分类均保持横向单行。推荐书籍卡片保留封面、标题、作者/来源、标签、简介和阅读/加入书架按钮；来源更新列表保留排名、封面、标题、来源标签和状态。

响应式以 `831 x 1893 px` 为基准。小屏优先保留顶部栏、搜索入口、信息源卡片、分类 chip、首个推荐内容和底部导航；短屏允许内容区滚动，底部导航固定。
