# 发现首页 Page Spec

## 页面定位

发现首页是底部主导航 `发现` 对应的根页面，用于从当前选择的信息源浏览可发现内容。当前阶段只保留首页结构，不补完整发现链路。

## 页面结构

1. 系统状态栏安全区。
2. 顶部栏：标题 `发现`，右侧搜索、更多。
3. 搜索入口栏。
4. 信息源类型：`全部来源 / 书源 / 订阅源`。
5. 当前来源行：来源名称、状态、切换入口。
6. 来源内分类 chip：分类名称来自当前来源。
7. 内容区：推荐、榜单、最近更新或栏目内容。
8. 来源状态条：来源数量、更新时间、刷新入口。
9. 底部主导航：当前高亮 `发现`。

## 范围边界

- 当前只生成发现首页。
- 来源选择底表、更多分类、发现设置、榜单详情、来源检测暂缓。
- 搜索入口跳转到书籍搜索链路，不在发现首页内展开。

## 自适应

- 信息源类型和分类 chip 横向滚动。
- 内容卡片在小屏保持单列。
- 大屏内容宽度不超过 `AppShell.Page.MaxContentWidth`。
