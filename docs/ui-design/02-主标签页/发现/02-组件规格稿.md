# 发现首页 Component Spec

| 组件 | 说明 | Token 来源 |
| --- | --- | --- |
| `AppTopBar` | 标题、搜索、更多 | `AppShell.TopBar.*` |
| `SearchEntry` | 进入搜索的只读输入栏 | `AppShell.Card.*` |
| `SourceTypeSegment` | 全部来源/书源/订阅源 | `AppShell.Chip.*` |
| `CurrentSourceCard` | 当前来源和切换入口 | `AppShell.Card.*` |
| `SourceCategoryChips` | 来源内分类 | `AppShell.Chip.*` |
| `DiscoveryContentCard` | 推荐/榜单/栏目卡 | `AppShell.Card.*` |
| `SourceStatusBar` | 更新时间、刷新 | `AppShell.ListItem.*` |
| `BottomNav` | 底部主导航 | `AppShell.BottomNav.*` |

发现首页不得使用独立于书架/RSS/设置的顶栏、卡片和 chip 体系。
