# 发现首页 Interaction Spec

| UI 元素 | 当前阶段交互 |
| --- | --- |
| 搜索图标/搜索入口 | 进入书籍搜索页 |
| 更多图标 | 保留入口，菜单暂缓 |
| `全部来源 / 书源 / 订阅源` | 当前页内切换内容类型 |
| 当前来源/切换来源 | 保留入口，来源选择底表暂缓 |
| 分类 chip | 当前页内刷新内容区 |
| 刷新 | 刷新当前来源和分类 |
| 推荐书籍/榜单行 | 进入书籍详情 |
| 加入书架 | 加入后显示轻提示 |
| 底部导航 | 切换主标签页 |

所有未展开的次级交互只保留入口语义，不生成完整次级页面。
