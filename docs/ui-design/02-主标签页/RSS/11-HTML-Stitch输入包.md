## 4. 页面生成约束

RSS 是主 Tab 根页面，必须继承书架 App Shell：页面背景、顶部栏、底部四项导航、导航高度、图标尺寸、文字字号、卡片圆角、边框、阴影和激活态颜色保持一致。

页面结构从上到下为顶部栏、订阅概览卡片、状态分组 chip、订阅源分类、最新订阅列表、底部导航。当前激活底部 Tab 为 RSS。

两组 chip 均为横向单行，不改成多行标签墙。订阅条目列表为单列扫描结构，保留封面、标题、分类标签、摘要、时间、未读点和行内更多入口。

响应式以 `866 x 1815 px` 为基准。小屏优先保留顶部栏、概览卡、第一组 chip、至少一条订阅条目和底部导航；短屏允许内容区滚动，底部导航固定。
