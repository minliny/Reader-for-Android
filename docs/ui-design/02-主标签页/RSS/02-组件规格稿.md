# RSS 首页 Component Spec

| 组件 | 说明 | Token 来源 |
| --- | --- | --- |
| `AppTopBar` | 标题、搜索、更多 | `AppShell.TopBar.*` |
| `SubscriptionSummaryCard` | 订阅概览 | `AppShell.Card.*` |
| `FeedStatusChips` | 全部/未读/收藏等 | `AppShell.Chip.*` |
| `FeedSourceChips` | 订阅源分组 | `AppShell.Chip.*` |
| `RssEntryItem` | 订阅条目行 | `AppShell.ListItem.*` |
| `UnreadIndicator` | 未读状态 | 复用强调色 `#2F6F93` |
| `BottomNav` | 底部主导航 | `AppShell.BottomNav.*` |

RSS 列表可以比设置列表更高，但图标、字号和触控区必须遵循 AppShell。
