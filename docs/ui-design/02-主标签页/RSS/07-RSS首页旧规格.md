## 概念

RSS/订阅首页是底部主导航 `RSS` 对应的主页面，用于查看用户已添加订阅源的最新内容流。

当前阶段只规划一个主页面，不设计添加订阅、订阅源管理、条目详情、分组管理、loading/empty/error 等次级页面或状态。后续实际功能再独立设计开发。

## 底栏规则

底部主导航固定为：

- `书架`
- `发现`
- `RSS`
- `设置`

本项目不规划账户功能，不设置 `我的` 底栏入口。原本可能属于 `我的` 的阅读统计、缓存、备份、关于、反馈等能力统一归入 `设置` 页面。

## 页面结构

- 顶部状态栏。
- 页面标题 `RSS`。
- 顶部右侧 `搜索` 和 `更多`。
- `订阅概览` 卡片：订阅源数量、未读数量、更新时间、刷新入口。
- 内容筛选：`全部 / 未读 / 收藏 / 稍后读 / 书单`。
- 订阅源筛选：`全部来源 / 小说更新 / 技术文章 / 书单推送`。
- `最新订阅` 列表：来源图标、标题、来源标签、摘要、时间、未读状态。
- 底部主导航，当前高亮 `RSS`。
