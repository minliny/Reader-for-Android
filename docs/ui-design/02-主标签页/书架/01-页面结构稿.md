# 书架首页页面结构稿

状态：正式结构摘要
权威入口：`10-正式UI设计稿.md`

## 页面定位

书架首页是 App P0 根页面，用于查看本地书架、继续阅读、切换分组、切换封面/列表模式，并进入搜索、书籍详情和阅读。

## 页面结构

1. 系统状态栏安全区。
2. 顶部栏：标题 `书架`，右侧搜索、更多。
3. 分组 chip：`全部 / 在读 / 已完结 / 本地 / 自定义分组`。
4. 顶部信息模块：`继续阅读`、`最近更新`。
5. 标题行：`我的书架`，右侧排序/筛选、封面/列表切换。
6. 书籍内容区：封面网格或列表。
7. 底部主导航：`书架 / 发现 / RSS / 设置`，当前高亮 `书架`。

## 滚动与固定

- 顶部栏可随页面滚动保持常规位置，不做沉浸吸顶要求。
- 底部主导航固定。
- 内容底部必须避让底部主导航和系统手势区。
- 封面模式和列表模式只改变书籍内容区布局，不改变 App Shell。

## 自适应

- 小于 360dp 宽度：使用紧凑边距，封面网格 2 列。
- 360-430dp：封面网格根据最小卡片宽度自动 2 或 3 列。
- 高度不足时优先压缩顶部信息模块，保证首屏能看到书架内容。

完整响应式规则以 `10-正式UI设计稿.md` 和 `08-书架首页组件视觉规格.md` 为准。
