# 书架首页 State Spec

| 状态 | 要求 | 图片示例 |
| --- | --- | --- |
| 默认封面模式 | 展示顶部模块和封面网格 | `01-书架首页-封面模式.png` |
| 默认列表模式 | 同一数据切换为列表行 | `02-书架首页-列表模式.png` |
| 空书架 | 保留顶部栏和底部导航，内容区显示空态 | 见书架主标签页 图片 `06-书架空状态.png` |
| loading | 保留页面框架，书籍区使用骨架 | 后续补正式主标签页 状态图 |
| error | 保留页面框架，提供重试 | 后续补正式主标签页 状态图 |
| 无封面 | 使用封面占位 token | 由视觉规格约束 |
| 长书名 | 标题最多两行，尾部省略 | 由视觉规格约束 |

状态页不得改变底部导航结构。
