# 书架首页 Interaction Spec

| UI 元素 | 交互 |
| --- | --- |
| 搜索图标 | 进入书籍搜索首页 |
| 更多图标 | 打开书架级更多菜单，后续定义 |
| 分组 chip | 当前页内切换书籍分组，保留模式 |
| 继续阅读卡 | 进入最近阅读位置 |
| 最近更新卡 | 进入对应书籍详情或章节 |
| 排序/筛选 | 打开排序与筛选底表 |
| 封面/列表切换 | 当前页内切换展示模式 |
| 书籍卡/行 | 进入书籍详情 |
| 书籍更多按钮 | 打开书籍操作底表 |
| 底部发现/RSS/设置 | 切换到对应主标签页 根页面 |

## 书籍操作底表

书架页书籍操作底表只允许：

- `修改`：进入书籍详情页，在详情页修改配置。
- `删除`：对应移除书架。

不得在底表中放置书源、缓存、分组、导出等配置型操作。
