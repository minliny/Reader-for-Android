## 4. 页面生成约束

页面结构为主 Tab 页面：顶部栏、分组 chip、继续阅读、最近更新、书籍集合、底部四项导航。底部导航固定在页面底部，当前激活项为书架。

封面模式和列表模式必须共用以下 token：页面背景、顶部栏高度、chip 高度与圆角、卡片背景、卡片边框、圆角、阴影、书名/作者/章节文字样式、进度条高度与颜色、底部导航高度、图标尺寸、激活态颜色。

书籍集合在封面模式使用 BookCard 网格；列表模式使用 BookListItem。两种模式的信息字段一致，只改变排列方式和封面尺寸。点击书籍进入阅读入口；点击书籍更多操作打开 `03-书架链路/书籍操作底表`。

响应式以 `853 x 1844 px` 为基准。常规手机宽度按比例缩放水平间距、封面宽度和卡片列数；小屏优先保留顶部栏、分组 chip、继续阅读入口、至少一行书籍和底部导航；短屏允许内容区滚动，底部导航不参与内容滚动。
