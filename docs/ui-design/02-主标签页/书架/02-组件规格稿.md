# 书架首页 Component Spec

## 组件清单

| 组件 | 说明 | Token 来源 |
| --- | --- | --- |
| `AppTopBar` | 标题、搜索、更多 | `AppShell.TopBar.*` |
| `ShelfChipGroup` | 书架分组 | `Shelf.Chip.*` |
| `ContinueReadingCard` | 继续阅读模块 | `Shelf.PanelCard.*` |
| `RecentUpdateCard` | 最近更新模块 | `Shelf.PanelCard.*` |
| `ShelfSectionHeader` | `我的书架` 和模式切换 | `Shelf.SectionTitle.*` |
| `BookCard` | 封面模式书籍卡 | `Shelf.BookCard.*` |
| `BookListItem` | 列表模式书籍行 | `Shelf.BookListItem.*` |
| `ProgressBar` | 阅读进度 | `Shelf.Progress.*` |
| `BottomNav` | 底部主导航 | `AppShell.BottomNav.*` |

## 复用要求

- 封面模式和列表模式必须复用页面背景、顶部栏、chip、卡片、进度条、底部导航 token。
- `BookCard` 和 `BookListItem` 可以使用不同布局，但标题/作者/章节层级必须一致。
- 书籍封面统一 `2:3`，不得拉伸。
