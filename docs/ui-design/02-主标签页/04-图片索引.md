- 图片只作为结构、密度和视觉方向参考。
- 组件 token 以 `../01-全局规范/02-应用外壳组件视觉规格.md` 为准。
- 书架首页额外以 `08-书架首页组件视觉规格.md` 为准。

- 历史图、废弃图、拼图和审图 contact sheet 不进入本目录。
