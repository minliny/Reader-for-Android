## 全局决策

- 主视觉方向：柔和卡片 + 分区式信息块。
- 主页面仅指底部主导航一级入口页面，当前固定为 `书架 / 发现 / RSS / 设置`。
- 阅读基础页：适合长时间阅读的极简阅读样式。
- 阅读覆盖层必须叠加在阅读页上，不重排、不重设正文底层。
- 书架卡片不展示书源。
- 书籍搜索与书内内容搜索是两条独立流程。
- 目录章节行只展示缓存与书签状态，不展示未读标记。
- 每个页面规格都应包含交互矩阵，说明按钮行为与目标页面。
- 跨页面关系统一记录在 `page-flows.md`，避免静态 UI 示意图丢失导航上下文。
- 新增 UI 示意图不得覆盖既有主图。补充状态图统一使用 `原编号-序号-页面说明.png`，并在 `drafts/README.md` 与对应规格文档中登记。
- 阅读控制层四主按钮以 `reader-control-layer-modules.md` 为准：点击 `目录 / 朗读 / 界面 / 设置` 时只切换左侧主内容区，右侧亮度条和底部四主按钮保持原控制层位置。
