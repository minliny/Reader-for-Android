## 页面定位

书架是用户的阅读资产中心，展示已经加入、导入、扫描或缓存的书籍。它不是书城，也不直接展示网络搜索结果，除非用户已将结果加入书架。

## 共享布局

封面模式和列表模式必须保持相同锚点：

- 顶部标题：`书架`
- 右上操作：搜索图标、页面级更多菜单
- 分组筛选：`全部 / 默认 / 本地书 / 追更`
- 顶部分区信息块：`继续阅读` 与 `最近更新`
- 分区标题：`我的书架`
- 分区右侧控制：模式标签、排序图标、书架设置齿轮
- 底部导航：`书架 / 发现 / RSS / 设置`

封面模式和列表模式只改变书籍内容的呈现方式。

## 顶部操作

| 界面元素 | 功能 |
| --- | --- |
| 搜索图标 | 打开书籍搜索页。 |
| 右上更多菜单 | 打开书架级低频操作：刷新、布局与排序、分组管理、本地书导入、网址添加、书架导入导出、日志。 |
| 分组筛选 | 按分组过滤书架。 |
| 模式标签 `封面` / `列表` | 表示当前布局模式。 |
| 排序图标 | 打开排序选项。 |
| 齿轮图标 | 打开书架布局或设置。 |

## 继续阅读卡片

内容：

- 封面
- 书名
- 作者
- 当前章节
- 阅读进度
- `阅读` 按钮

按钮：

- `阅读`：从当前阅读位置打开阅读页。

## 最近更新卡片

内容：

- 最近更新的书籍
- 最新章节
- 小型更新标记

点击条目打开书籍信息页或该书的详情上下文。

## 书籍条目内容

书架条目展示：

- 书名
- 作者
- 最新章节或阅读进度
- 封面
- 需要时显示进度条

书架条目不展示书源。书源属于书籍信息页、换源页或长按详情。

## 封面模式

手机竖屏使用三列自适应网格。

每个条目包含：

- 封面
- 书名
- 作者
- 最新章节或阅读进度

## 列表模式

每行包含：

- 封面缩略图
- 书名
- 作者
- 最新章节
- 阅读进度百分比或进度条
- 必要时显示行级更多操作

列表行保持适中密度，支持快速扫读。

## 交互矩阵

| 界面元素 | 动作 | 目标 |
| --- | --- | --- |
| 搜索图标 | 开始搜索书籍。 | `书籍搜索页` |
| 右上更多菜单 | 打开书架级低频操作。 | 菜单：刷新、分组管理、导入本地书、通过网址添加、书架导入导出、日志 |
| 分组筛选 | 按选中分组过滤书籍。 | 当前页，封面或列表内容刷新 |
| 继续阅读卡片中的 `阅读` | 继续阅读当前书籍。 | `阅读页 - 沉浸模式`，定位到已保存位置 |
| 最近更新条目 | 打开更新书籍上下文。 | `书籍信息页`，聚焦该书 |
| `我的书架` 模式标签 `封面/列表` | 切换展示模式。 | 当前页，在封面模式和列表模式间切换 |
| 排序图标 | 选择书架排序方式。 | 排序底表或弹窗 |
| 齿轮图标 | 配置书架布局和可见字段。 | 书架布局/设置页或底表 |
| 书籍封面/卡片主体 | 打开书籍阅读。 | `阅读页 - 沉浸模式`，定位到已保存位置 |
| 书名/元信息区域 | 查看书籍详情。 | `书籍信息页` |
| 书籍条目更多操作 | 打开条目操作。 | 书籍操作底表：信息、移除、刷新、缓存、导出、移动分组 |
| 长按书籍条目 | 进入选择或条目操作模式。 | 多选模式或书籍操作底表 |
