# 阅读内设置完整页任务

阅读内设置完整页从 `22-2-reader-settings-quick-panel-height-aligned.png` 的 `完整设置 >` 进入，不属于控制层本体。

| 页面 ID | 页面名称 | 入口 | 退出 | 需要设计图 | 必要状态 | 备注 |
| ----- | ---- | -- | -- | ----- | ---- | -- |
| `reader_settings_home` | 阅读设置首页 | 设置快捷模块 `完整设置 >` | 返回阅读控制层或阅读页 | 是 | 默认、加载、错误 | 汇总屏幕、行为、辅助、预设入口。 |
| `reader_settings_display` | 阅读显示设置 | 阅读设置首页 | 返回设置首页 | 是 | 默认、预览更新、错误 | 屏幕方向、状态栏、导航栏、信息显示；不重复界面外观设置。 |
| `reader_settings_behavior` | 阅读行为设置 | 阅读设置首页 | 返回设置首页 | 是 | 默认、开关变更、错误 | 音量键翻页、点击区域、手势、单手模式。 |
| `reader_settings_assist` | 阅读辅助设置 | 阅读设置首页 | 返回设置首页 | 是 | 默认、权限缺失、错误 | 误触保护、章节加载、缓存优先、失败重试。 |
| `reader_settings_preset` | 阅读预设/恢复默认 | 阅读设置首页右上菜单 | 取消/应用后返回 | 是 | 默认、确认、成功、失败 | 包含保存预设、应用预设、恢复默认。 |

## 边界规则

- 阅读完整设置页不保留底部四主按钮。
- 阅读完整设置页不重复右侧亮度滑条。
- 字体、主题、版式、翻页动画归 `界面`，不归阅读设置。
