## 3. 统一组件规格说明

本轮目标规格为：

- Quick Action Row：统一 `grid-cols-4`，四等分，搜索 / 自动翻页 / 内容替换 / 夜间模式，统一图标尺寸、文字字号、间距、padding。
- Chapter Jump Row：统一 `skip_previous / 25% 章节进度条 / skip_next`，统一行高、圆点尺寸和进度状态。
- Bottom Function Row：统一 `menu_book / record_voice_over / tune / settings`，目录 / 朗读 / 界面 / 设置，四项普通状态无高亮。
- Bottom Control Stack 容器：固定在屏幕底部，不嵌入任何展开面板。
- 展开区域容器：位于 Bottom Control Stack 上方，统一宽度、圆角、padding、背景和轻量虚线边界。
- 边界标注风格：以轻量虚线边界为主，减少文字 label，不混用英文 handoff 标签。

实际自检结果：目录、朗读、界面、设置四个候选 screen 未完全达到上述统一规格，因此本轮不建议进入只读复审。
